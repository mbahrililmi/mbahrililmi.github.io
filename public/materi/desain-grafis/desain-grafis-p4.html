2 Raster vs Vector Graphics

🟧 Raster/Bitmap Graphics

  • Terdiri dari pixel (picture element)
  • Resolution-dependent — pixelated jika diperbesar
  • File size besar untuk high resolution
  • Format: JPG, PNG, GIF, TIFF, PSD, XCF
  • Best untuk: Photography, complex images
  • Software: Photoshop, GIMP

🔷 Vector Graphics (Review)

  • Terdiri dari mathematical paths
  • Resolution-independent — scalable tanpa quality loss
  • File size relatif kecil
  • Format: AI, SVG, EPS, PDF
  • Best untuk: Logos, icons, print design
  • Software: Illustrator, Inkscape

6 Layer System (Konsep Terpenting!)

Layer adalah seperti lembaran transparan yang ditumpuk — edit satu layer tanpa mempengaruhi layer lain. Ini adalah fondasi dari non-destructive editing.

Background Layer

Default layer, locked, no transparency. Convert ke normal: double-click

Normal Layer

Standard image layer, supports transparency, dapat direorder

Adjustment Layer

Non-destructive color/tone adjustments, dapat dimodifikasi kapan saja

Text Layer

Editable text, vector-based dan scalable

Shape Layer

Vector shapes yang dapat di-edit

Smart Object

Non-destructive transforms, dapat embed file lain

Layer Operations & Shortcuts

OperasiShortcut
New layerCtrl + Shift + N
Duplicate layerCtrl + J
Merge selected layersCtrl + E
Merge visibleCtrl + Shift + E
Group layersCtrl + G
Rename layerDouble-click nama layer
🔑 Blend Modes Penting: Normal (default), Multiply (darken), Screen (lighten), Overlay (contrast), Soft Light (subtle overlay). Blend modes mengubah cara layer berinteraksi dengan layer di bawahnya.
7 Selection Tools (Fundamental!)
M

Rectangular Marquee

Select rectangular area. Hold Shift: perfect square

M

Elliptical Marquee

Select circular/oval area. Hold Shift: perfect circle

L

Lasso Tool

Freehand selection. Quick tapi tidak precise

L

Polygonal Lasso

Click untuk titik-titik. Untuk edge lurus

L

Magnetic Lasso

Snap to edges automatically. Untuk high contrast subjects

W

Magic Wand

Select by color similarity. Tolerance setting mengatur range warna

W

Quick Selection

Brush-like, auto detect edges. Sangat intuitif untuk beginners

W

Object Selection

AI-powered object detection. Drag sekitar objek

💡
Selection Modifiers:
  • Shift + drag = Add to selection
  • Alt + drag = Subtract from selection
  • Ctrl + A = Select All | Ctrl + D = Deselect
  • Ctrl + Shift + I = Inverse selection
