Motion Graphics Concepts

Apa itu Motion Graphics?

Definisi
Animated graphic design – typography, shapes, logos bergerak. Blend antara desain dan animasi. Bukan character animation.
Aplikasi
Logo animations, title sequences, explainer videos, social media content, TV graphics, advertising, UI animations.
MoGraph vs Character Animation

Motion Graphics

  • Abstract, design-focused
  • Typography, shapes, logos
  • Geometric precision
  • Design principles apply
  • Fast production

Character Animation

  • Organic, storytelling
  • Characters, creatures
  • Rigging diperlukan
  • 12 Principles heavy
  • Lebih time-intensive

Text Animation & 3D Typography

Membuat 3D Text

  1. Shift+A → Text
  2. Tab untuk masuk edit mode – ketik teks Anda
  3. Tab lagi ke Object Mode
  4. Properties → Object Data → Geometry:
    • Extrude: Ketebalan 3D (misal: 0.1)
    • Bevel Depth: Rounded edges (misal: 0.01)
    • Bevel Resolution: Kehalusan bevel
  5. Object Data → Font → Open Font untuk ganti typeface
  6. Apply material untuk metallic/emissive text

Text Animation Techniques

Scale Reveal
Frame 1: Scale = 0. Frame 20: Scale = 1. Add ease in/out via Graph Editor untuk natural feel.
Position Entry
Start dari off-screen, animate masuk ke posisi final. Overshoot + settle back untuk bounce feel.
Glow Animation
Keyframe Emission Strength dari 0 ke 5+. Material "turns on". Dramatic dan eye-catching.
Per-Letter Animation
Object → Convert to Mesh → P → Separate by Loose Parts. Animate tiap huruf individual dengan stagger timing.

Simple Motion Graphics Projects

Logo Animation Sequence ▼
  1. Import atau buat logo 3D (text + shape)
  2. Stage 1 (0-1s): Elements masuk dari berbagai arah
  3. Stage 2 (1-3s): Assemble menjadi logo lengkap
  4. Stage 3 (3-5s): Hold dengan glow/pulse effect
  5. Animate camera subtle untuk dynamism
  6. Final: static hold dengan emission gentle
Typography Reveal ▼
  1. Convert text ke mesh → Separate letters
  2. Set origin ke geometry tiap huruf
  3. Animate tiap huruf dengan offset 2-3 frame antar huruf
  4. Scale dari 0 ke 1, atau position dari bawah
  5. Stagger timing untuk cascade effect
  6. Final: semua huruf di posisi, gentle settle
Product Showcase Clip ▼
  1. 3D produk dengan materials proper
  2. 0-5s: Turntable 360° untuk perkenalkan produk
  3. 5-8s: Camera cut ke close-up detail
  4. 8-12s: Text overlay animate masuk (nama produk, fitur)
  5. 12-15s: Wide shot dengan logo brand
  6. Professional lighting throughout

Final Project – Tugas Besar 3D

Deskripsi Final Project

Buat sebuah animated 3D scene yang menshowcase semua skills dari pertemuan 10-15. Project ini akan menjadi portfolio piece dan dinilai sebagai culmination dari pembelajaran 3D Anda.

Pilih Salah Satu Project Type

A

Product Showcase Animation

  • Model 3D product (nyata atau imajiner)
  • Realistic materials dan textures
  • Professional lighting
  • Turntable atau animated reveal
  • Text overlay animated
  • Durasi: 15-30 detik
B

Logo Animation Sequence

  • 3D logo design (boleh dari tugas sebelumnya)
  • Multiple animation stages (build, reveal, finalize)
  • Dynamic camera atau static composition
  • Effects (glow, depth of field)
  • Durasi: 5-15 detik
C

Motion Graphics Piece

  • Abstract atau thematic piece
  • Animated shapes, text, objects
  • Synchronized animation dengan timing bagus
  • Durasi: 10-20 detik
D

Animated Scene

  • Simple environment/scene
  • Multiple animated objects
  • Camera animation
  • Elemen storytelling sederhana
  • Durasi: 20-30 detik

Minimum Requirements (Semua Project Type)

Modeling
  • Minimal 3 objek distinct
  • Edit mode techniques digunakan
  • Minimal 2 modifier efektif
  • Clean topology
Materials
  • Minimal 3 material berbeda
  • Minimal 1 procedural material
  • Penggunaan Metallic, Roughness, Emission tepat
Animation
  • Minimal 10 detik durasi
  • Minimal 3 property animated
  • Smooth dengan proper easing
  • No jerky movements
Rendering
  • Min 1920×1080 Full HD
  • 24 fps atau 30 fps
  • Minimal noise
  • Output: MP4 video

Rendering Optimization

Tips Render Lebih Cepat ▼
  • Gunakan Eevee untuk previews, Cycles untuk final
  • Aktifkan Denoising untuk reduce noise dengan sample lebih rendah
  • Test render di 50% resolution sebelum final
  • GPU rendering untuk Cycles jauh lebih cepat dari CPU
  • Frame skip: render setiap 5 frame untuk timing check
  • Render region (Ctrl+B) untuk test area tertentu
Compositing Dasar (Optional) ▼
  • Compositing workspace di Blender
  • Node-based post-processing
  • Color correction, bloom/glare, vignette
  • Lebih cepat dari re-render untuk adjustments warna
  • Render image sequence → composite → export video
Common Mistakes dalam Final Project ▼
  • Scope terlalu ambisius – simplify dan polish lebih baik
  • Manajemen waktu buruk – mulai jauh sebelum deadline
  • Tidak test render berkala – cek progress sering
  • Lighting buruk merusak model yang bagus
  • Animasi jerky karena tidak pakai Graph Editor
  • Lupa dokumentasi dan presentation
  • File tidak ter-organize
