3 Advanced Masking Techniques

Quick Mask Mode (Q)

Visualize selection sebagai red overlay. Paint selection dengan brush.

Shortcut: Tekan Q untuk toggle. Exit = active selection.

Color Range Masking

Select > Color Range. Select by color similarity dengan fuzziness control.

Great untuk skies, solid colors, uniform backgrounds.

Luminosity Masking

Select by brightness. Ctrl + Click RGB channel di Channels panel.

Professional technique untuk landscape photography dan sky replacement.

Channel Masking

Duplikat channel dengan contrast terbaik, increase contrast, load as selection.

Perfect untuk hair, trees, smoke — edge yang sangat complex.
⭐ Select and Mask Workspace (Paling Recommended!) ▼

Cara masuk: Select > Select and Mask (atau Ctrl+Alt+R)

  • Refine Edge Brush untuk area rambut, bulu, dan detail halus
  • Adjust Edge Detection — radius sesuai complexity subject
  • Smooth, Feather, Contrast adjustments
  • Decontaminate Colors untuk remove fringe/halo
  • Output ke Layer with Mask untuk non-destructive result
✅ Select and Mask adalah cara terbaik untuk membuat selection yang complex, terutama hair masking. Hasilnya jauh lebih natural dibanding manual masking.
4 Image Manipulation & Compositing

📋 Planning Composite — Checklist Sebelum Mulai

  • Buat concept sketch atau mood board terlebih dahulu
  • Kumpulkan semua source images yang diperlukan
  • Match perspective antar semua elemen
  • Match lighting direction (dari mana datangnya cahaya?)
  • Match color temperature semua elemen

Matching Elements — Kunci Compositing Realistis

ElemenTeknik
Match ColorsImage > Adjustments > Match Color, atau manual dengan Curves per channel
Match LightingMultiply layer dengan soft black brush untuk shadows; Screen layer untuk highlights
Match PerspectiveEdit > Transform > Perspective atau Distort untuk sesuaikan sudut
Match SharpnessBlur elemen yang terlalu tajam, atau sharpen yang kurang tajam. Consistency is key.

Creating Shadows & Reflections

🌑
Shadow:
  1. Duplicate object layer
  2. Fill black
  3. Transform: skew/distort ke arah ground
  4. Gaussian Blur (radius sesuai distance)
  5. Opacity 20-40%
  6. Layer mask untuk fade edges
🪞
Reflection:
  1. Duplicate object layer
  2. Flip Vertical
  3. Position tepat di bawah object
  4. Layer mask dengan gradient (fade to transparent)
  5. Lower opacity 30-60%
  6. Add blur untuk distance effect
🎭
Blend Modes untuk Compositing:
  • Multiply — Darken, baik untuk shadows
  • Screen — Lighten, baik untuk glow dan light effects
  • Overlay — Contrast blend, memperkuat light dan dark
  • Soft Light — Subtle, natural-looking overlay

⚠️ Gunakan artistic filters dengan hemat — efeknya sering terlihat obvious dan mengurangi professional look. Preview di Filter Gallery sebelum apply.

Smart Filters = Non-Destructive Filters!

  1. Convert layer ke Smart Object terlebih dahulu: Layer > Smart Objects > Convert to Smart Object
  2. Apply filter biasa: Filter > [Any Filter]
  3. Filter menjadi editable di bawah layer thumbnail
  4. Dapat adjust opacity dan blend mode filter
  5. Dapat mask filter untuk selective application
  6. Dapat double-click untuk edit parameter kapan saja
✅ Smart Filters adalah cara profesional menggunakan filters. Selalu convert ke Smart Object sebelum apply filter penting.
  • Skin Smoothing — AI-powered skin retouching otomatis
  • Style Transfer — Transfer artistic style dari painting ke foto
  • Colorize B&W — Automatic colorization foto hitam putih
  • Super Zoom — AI upscaling untuk enlarge image
  • Sky Replacement — Replace langit secara otomatis
🤖 Neural Filters tersedia di Photoshop 2021+. Membutuhkan koneksi internet untuk processing di Adobe servers. Output sebagai Smart Filter atau new layer.
6 Special Techniques
🎭

Double Exposure Effect

Stack dua foto, gunakan blend mode Screen atau Lighten. Layer mask untuk control area mana yang menampilkan masing-masing foto. Popular artistic effect untuk portrait + landscape combination.

💥

Dispersion Effect

Subject yang "hancur" menjadi partikel. Gunakan Liquify filter untuk distorsi, particle brushes untuk fragment, multiple layers untuk depth effect yang meyakinkan.

🪄

Levitation Effect

Shoot dua foto: dengan stool dan tanpa. Composite keduanya di Photoshop, hapus stool dari foto, tambahkan shadow di tanah untuk grounding yang natural.

🔭

Miniature / Tilt-Shift Effect

Blur top dan bottom foto dengan gradient mask, increase saturation dan contrast. Hasilnya terlihat seperti foto miniature model. Cocok untuk aerial/high-angle shots.

🎨

Duotone / Tritone

Convert ke grayscale, tambahkan Gradient Map adjustment layer dan map ke dua atau tiga warna. Classic vintage poster look yang timeless.

9 Ringkasan
📝
Poin-poin kunci Pertemuan 7:
  1. Color Grading — Creative color adjustments menggunakan Curves, LUTs, Gradient Maps, Split Toning untuk mood dan style
  2. Advanced Masking — Quick Mask, Color Range, Luminosity, Channel Masking, dan Select and Mask workspace
  3. Photo Manipulation — Planning, blending, matching colors/lighting/perspective/sharpness, shadows, reflections, depth
  4. Filters — Blur, sharpen, artistic, dan Smart Filters untuk non-destructive effects
  5. Special Techniques — Double exposure, dispersion, levitation, miniature, duotone effects

Key takeaway: Photo manipulation yang baik adalah tentang detail. Lighting, shadows, color matching, perspective — semua harus seamless. The difference between amateur and professional work is subtlety and realism.
