Pengantar Blender 3D

Mengapa Blender?

Blender adalah software 3D open-source gratis yang digunakan secara professional di industri film, game, dan advertising. Blender mencakup modeling, texturing, rigging, animation, simulation, rendering, compositing, dan video editing dalam satu paket.

Gratis & Open Source
Free forever, tidak ada biaya lisensi. Community besar, update rutin, ribuan tutorial online.
Industry Standard
Digunakan oleh studio profesional. Film Netflix, iklan brand besar, game indie menggunakan Blender.
All-in-One
Modeling, texturing, animation, rendering, compositing – semuanya dalam satu software.
Instalasi Blender
  1. Download dari blender.org (versi terbaru)
  2. Install (Windows: .msi, Mac: .dmg, Linux: .tar.xz)
  3. Buka Blender – akan muncul splash screen
  4. Klik area kosong untuk dismiss splash screen
  5. Siap bekerja!

Interface Blender

Area Utama Interface ▼
  • 3D Viewport: Area kerja utama – tempat model 3D berada
  • Timeline: Panel bawah untuk animasi dan frame
  • Outliner: Panel kanan atas – daftar semua objek
  • Properties Panel: Panel kanan bawah – settings objek/scene/render
  • Menu Bar: Atas – File, Edit, Render, Window, Help
  • Header: Atas viewport – mode selector, view menu
Workspaces ▼
  • Layout: General modeling workspace
  • Modeling: Fokus pada mesh editing
  • Sculpting: Mode sculpt untuk organic forms
  • UV Editing: Untuk UV maps dan textures
  • Texture Paint: Mengecat langsung pada model
  • Shading: Node-based material editor
  • Animation: Graph editor, timeline, dope sheet
  • Rendering: Render output preview
  • Compositing: Post-processing node editor
Properties Panel Tabs ▼
  • Tool (T): Active tool settings
  • Scene: Scene-wide settings
  • World: Background, lighting environment
  • Object: Transform, visibility, relations
  • Modifier: Modifier stack
  • Material: Material slots dan settings
  • Render: Render engine dan output settings
  • Output: File output settings

Navigasi 3D Viewport

Mouse Navigation
Middle Mouse + Drag: Orbit/putar view
Shift + Middle Mouse: Pan/geser view
Scroll / Ctrl + Middle: Zoom in/out
Numpad 0: Camera view
Numpad Shortcuts
Numpad 1: Front view
Numpad 3: Right view
Numpad 7: Top view
Numpad 5: Toggle ortho/perspective
Shortcut Navigasi Penting
  • G Move / Grab objek
  • R Rotate objek
  • S Scale objek
  • X/Y/Z Batasi ke axis tertentu
  • Enter Konfirmasi transformasi
  • Shift+A Add object menu
  • X / Del Hapus objek
  • Tab Toggle Edit/Object mode
  • Z Viewport shading pie menu
  • H / Alt+H Hide / Unhide

Objek & Mode Dasar

Tambah objek dengan Shift+A → Mesh:

Cube
Kotak dasar. Paling sering digunakan sebagai basis bangunan, furnitur, produk.
UV Sphere
Bola dengan poles dan meridian. Cocok untuk bola, kepala, planet.
Cylinder
Silinder. Basis untuk gelas, botol, pilar, roda.
Plane
Bidang datar. Ground plane, layar, dinding.
Cone
Kerucut. Topi, tanda, puncak bangunan.
Torus
Donat. Ring, gelang, pipa melingkar.
  • Object Mode: Default. Select, move, delete objek sebagai unit
  • Edit Mode (Tab): Edit vertices, edges, faces
  • Sculpt Mode: Sculpt model seperti clay
  • Vertex Paint: Cat warna langsung ke mesh
  • Texture Paint: Melukis texture pada model
  • Weight Paint: Untuk rigging dan skinning
  • Grab (G): Pindahkan objek. G → X/Y/Z untuk 1 axis
  • Rotate (R): Putar objek. R → X/Y/Z → nilai derajat
  • Scale (S): Ubah ukuran. S → X/Y/Z untuk axis tertentu
  • Precise input: Ketik angka setelah G/R/S (contoh: G Z 2 = pindah 2 unit di Z)
  • Reset: Alt+G (location), Alt+R (rotation), Alt+S (scale)
  • Origin: Titik pivot objek untuk transformasi

Mesh Editing Dasar

Edit Mode

Tekan Tab untuk masuk Edit Mode. Di sini Anda dapat mengedit individual vertices, edges, dan faces untuk membentuk model sesuai keinginan.

Selection dalam Edit Mode ▼
  • 1 Vertex select mode
  • 2 Edge select mode
  • 3 Face select mode
  • A Select/Deselect all
  • Alt+Click Loop select
  • Ctrl+Click Shortest path select
Operasi Mesh Penting ▼
  • E Extrude – tarik keluar face/edge/vertex
  • I Inset – buat face lebih kecil di dalam face
  • Ctrl+R Loop Cut – tambah edge loop
  • K Knife Tool – potong mesh bebas
  • F Fill – buat face dari edges/vertices
  • M Merge vertices
  • Ctrl+B Bevel edge/vertex
  • Ctrl+J Join objects (Object Mode)
Contoh: Membuat Gelas ▼
  1. Shift+A → Mesh → Cylinder
  2. Tab ke Edit Mode
  3. Select top face (klik face 3 di header)
  4. I (Inset) untuk buat ring dalam
  5. E (Extrude) ke bawah untuk isi dalam gelas
  6. S → scale untuk taper bagian bawah
  7. Apply Solidify modifier untuk ketebalan

Modifier Dasar

Subdivision Surface
Smoothing mesh dengan menambah geometry. Level 1-3 untuk preview, level 2 untuk render. Hasil organic dan halus.
Mirror
Cermin objek di X/Y/Z axis. Ideal untuk model simetris. Model setengah, mirror untuk sempurnakan.
Solidify
Tambah ketebalan pada mesh. Berguna untuk plane menjadi object padat. Gelas, dinding, kertas.
Bevel
Buat tepi yang rounded. Menghilangkan sharp edge. Memberi kesan nyata dan plastis.
Array
Duplikasi objek dengan offset. Pagar, tangga, grid, pattern berulang. Count dan offset kontrolabel.
Boolean
Combine, subtract, intersect dua objek. Union, Difference, Intersect. Berguna untuk lubang dan potongan kompleks.

Material & Rendering Pertama

Membuat Material Dasar

  1. Select objek → Properties panel → Material tab
  2. Klik "New" untuk buat material baru
  3. Ganti nama material (optional)
  4. Ubah Base Color untuk warna dasar
  5. Atur Metallic (0=plastik, 1=logam)
  6. Atur Roughness (0=mirror, 1=kasar)
  7. Gunakan Viewport Shading "Material Preview" (Z → Material Preview) untuk lihat hasil

Lighting Dasar

Point Light
Cahaya dari satu titik ke segala arah. Seperti bola lampu. Shift+A → Light → Point.
Sun Light
Cahaya directional sejajar (seperti matahari). Tidak terpengaruh posisi, hanya rotasi.
Spot Light
Cahaya seperti spotlight – cone-shaped. Kontrol sudut dan softness.
Area Light
Cahaya dari area datar. Bayangan lembut. Paling natural untuk studio setup.

Render Pertama

  1. Setup camera: Numpad 0 untuk camera view, Ctrl+Alt+Numpad 0 snap camera ke view
  2. Properties → Render → Pilih Engine: Eevee (cepat) atau Cycles (realistis)
  3. Tekan F12 untuk render
  4. Image Editor terbuka dengan hasil render
  5. Image → Save As untuk simpan

Praktikum Pertemuan 10

Latihan 1: Eksplorasi Interface
  • Buka Blender, eksplorasi semua workspaces
  • Latih navigasi: orbit, pan, zoom menggunakan mouse
  • Coba semua numpad view shortcuts
  • Toggle ortho/perspective dengan Numpad 5
  • Waktu: 20-30 menit
Latihan 2: 3 Objek Sederhana
  • Buat meja, kursi, atau objek rumah tangga sederhana
  • Gunakan primitives (cube, cylinder) sebagai basis
  • Gunakan G, R, S untuk transformasi
  • Apply minimal 2 modifier berbeda
  • Waktu: 60-90 menit
Latihan 3: Render Scene Pertama
  • Buat simple scene: objek + ground plane + lighting
  • Beri material dengan warna berbeda tiap objek
  • Setup camera dengan komposisi yang menarik
  • Render menggunakan Eevee
  • Save hasil render sebagai PNG
  • Waktu: 45-60 menit
