💡 Desain grafis berbasis vector adalah keterampilan fundamental yang harus dikuasai desainer. File vector seperti SVG, AI, dan EPS dapat diperbesar ke ukuran billboard tanpa kehilangan kualitas sedikit pun.
2 Vector vs Raster Graphics

🔷 Vector Graphics

  • Dibuat dengan persamaan matematis
  • Terdiri dari anchor point, path, dan fill
  • Scalable — tidak buram saat diperbesar
  • File kecil untuk desain sederhana
  • Format: AI, SVG, EPS, PDF
  • Cocok untuk: Logo, icon, ilustrasi

🟦 Raster/Bitmap Graphics

  • Dibuat dari kumpulan pixel
  • Resolution-dependent
  • Tidak scalable — buram saat diperbesar
  • File besar untuk resolusi tinggi
  • Format: JPG, PNG, PSD, GIF, TIFF
  • Cocok untuk: Foto, texture detail tinggi
AspekVectorRaster
TeknologiMatematika (paths, curves)Pixel (kotak-kotak kecil)
Scalability✅ Bebas ukuran, tetap tajam❌ Buram jika diperbesar
File SizeKecil (desain simple)Besar (high resolution)
Edit WarnaMudah, kapan sajaTerbatas pada piksel

✅ Analogi mudah: Vector seperti peta matematika — instruksi untuk menggambar ulang dengan presisi sempurna. Raster seperti mozaik foto — jutaan kotak warna yang dilihat dari jauh terlihat seperti gambar.
3 Pengenalan Software

💼 Adobe Illustrator (Berbayar)

  • Industry standard profesional
  • Integrasi Adobe Creative Cloud
  • Tools dan fitur sangat lengkap
  • Harga: ~Rp 350.000/bulan
  • Tersedia trial 7 hari

🆓 Inkscape (GRATIS — Direkomendasikan)

  • Open source dan gratis selamanya
  • Cross-platform: Windows, Mac, Linux
  • Fitur hampir setara Illustrator
  • Format file: SVG (standar web)
  • Komunitas besar, banyak tutorial
  • 📥 Download: inkscape.org
💡 Untuk perkuliahan ini, Inkscape sangat direkomendasikan karena gratis dan fiturnya memadai. Konsep yang dipelajari di Inkscape langsung dapat diaplikasikan di Illustrator jika nanti menggunakan software berbayar.

5 Tool-Tool Dasar
S / F1

Selection Tool

Pilih, pindah, dan ubah ukuran seluruh objek. Klik dua kali untuk masuk ke dalam grup.

N / F2

Node Tool

Edit anchor point individual. Ubah bentuk path dengan menggeser node.

Z

Zoom Tool

Zoom in klik, zoom out Alt+klik. Shortcut: Ctrl+= dan Ctrl+-

H

Hand/Pan Tool

Geser canvas. Shortcut cepat: tahan Spasi sambil drag.

💡 Klik objek sekali = mode scale/resize. Klik lagi = mode rotate/skew. Tekan Shift saat resize untuk menjaga proporsi.
R / F4

Rectangle Tool

Buat persegi panjang. Tekan Ctrl untuk persegi sempurna.

E / F5

Ellipse Tool

Buat oval. Tekan Ctrl untuk lingkaran sempurna.

*

Star Tool

Buat bintang atau polygon. Atur jumlah sudut di toolbar atas.

3D

3D Box Tool

Buat kotak perspektif 3D (Inkscape). Untuk efek dimensi cepat.

✅ Tips: Saat menggambar, tahan Ctrl untuk constrain proporsi, tahan Alt untuk menggambar dari titik tengah, tahan keduanya untuk lingkaran/persegi dari tengah.
P / F6

Pen Tool (Bezier)

Tool terpenting vector! Klik = garis lurus. Klik + Drag = kurva Bezier.

B

Pencil (Freehand)

Gambar bebas seperti pensil. Otomatis dikonversi ke path.

C

Calligraphy

Brush calligraphy. Menghasilkan path dengan variasi ketebalan.

N

Node Editor

Edit handles Bezier. Shift+drag node = tangent independent.

⚠️ Pen Tool memerlukan latihan! Konsepnya: setiap klik membuat anchor point, dan drag dari anchor point membuat Bezier handle yang menentukan arah kurva. Tutup path dengan klik di titik awal.
📚 Cara Kerja Bezier Curve ▼

Kurva Bezier dikendalikan oleh anchor points dan control handles:

  • Corner node: Klik biasa — membuat sudut tajam
  • Smooth node: Klik + drag — kurva halus, handles sejajar
  • Cusp node: Alt + drag handle — kurva tidak simetris
  • Menutup path: Klik di titik awal, atau tekan Enter
  • Cancel: Tekan Escape untuk batal
T / F8

Text Tool

Tambahkan teks. Point text: klik. Area text: klik drag untuk kotak.

F7

Color Picker

Ambil warna dari area mana pun di layar.

G / F9

Gradient Tool

Buat dan edit gradien pada objek terpilih.

Ctrl+F

Find/Replace

Cari dan ganti teks atau atribut SVG dalam dokumen.

🎨 Mengatur Fill dan Stroke ▼

Warna objek dikontrol oleh Fill (isi) dan Stroke (garis tepi):

  • Fill: Klik kotak warna kiri bawah — isi objek
  • Stroke: Klik kotak warna kanan bawah — garis tepi
  • No fill/stroke: Klik tanda X di color palette
  • Color dialog detail: Buka dengan Shift+Ctrl+F
  • Gradient fill: Gunakan Gradient Tool atau dialog Fill
💡 Di status bar bawah, tertera nilai warna fill/stroke dalam format HEX atau RGBA. Klik langsung pada indikator warna di status bar untuk membuka color picker cepat.
📐 Align & Distribute ▼

Buka panel dengan Shift+Ctrl+A. Pilih beberapa objek, lalu gunakan:

  • Align to page/drawing/selection/last selected: Tentukan referensi alignment
  • Align left/center/right edges: Perataan horizontal
  • Align top/center/bottom edges: Perataan vertikal
  • Make horizontal/vertical gaps equal: Distribusi jarak merata
7 Tips Workflow Profesional
  1. Gunakan keyboard shortcuts untuk efisiensi — kurangi penggunaan mouse
  2. Organize dengan Layers — beri nama layer sesuai kontennya
  3. Save berkala dengan Ctrl+S, dan buat backup dengan Save As
  4. Gunakan Guides dan Grid (View > Grid) untuk presisi alignment
  5. Zoom in untuk detail, zoom out untuk melihat keseluruhan
  6. Manfaatkan Snap to Grid / Snap to Objects untuk alignment akurat
  7. Gunakan Object > Align and Distribute daripada alignment manual

9 Praktikum
🔬 Latihan 1: Membuat Bentuk Dasar ▼
🎯 Tujuan: Mengenal dan menggunakan shape tools dasar, mengatur fill dan stroke.
1

File › New Document. Pilih ukuran A4, orientasi Portrait, units: mm.

2

Tekan R, drag sambil tahan Ctrl untuk persegi sempurna. Buka Fill & Stroke (Ctrl+Shift+F), set fill ke merah, stroke ke "no paint".

3

Tekan E, drag sambil tahan Ctrl. Set fill biru, stroke hitam dengan ketebalan 3px.

4

Gunakan Star Tool (*) untuk segitiga (3 sudut), bintang (5 sudut), dan hexagon (6 sudut). Atur jumlah sudut di Tool Controls Bar.

5

Susun kelima objek dengan rapi menggunakan Align & Distribute. Simpan: Latihan01_BentukDasar_[Nama].svg

🔬 Latihan 2: Manipulasi Objek ▼
🎯 Tujuan: Belajar memindah, resize, rotate, dan duplicate objek untuk membuat pola.
1

Tekan E, tahan Ctrl saat drag. Di Tool Controls Bar, set width dan height ke 40px.

2

Tekan Ctrl+D sebanyak 7 kali. Setiap duplikat menumpuk persis di atas aslinya.

3

Geser setiap lingkaran ke posisi melingkar (seperti angka jam). Gunakan Rotate untuk orientasi.

4

Klik setiap lingkaran, pilih warna berbeda. Buat persegi di tengah susunan lingkaran.

5

File › Save As: Latihan02_ManipulasiObjek_[Nama].svg

🔬 Latihan 3: Bermain dengan Pen Tool ▼
⚠️ Perhatian: Pen Tool adalah tool yang paling sulit untuk dipelajari tetapi paling powerful. Jangan menyerah jika hasilnya tidak sempurna pada percobaan pertama — ini normal!
1

Pastikan mode "Create regular bezier path" aktif di Tool Controls Bar.

2

Klik beberapa titik. Akhiri dengan double-klik atau Enter. Hasilnya: path dengan garis lurus.

3

Klik beberapa titik, lalu klik di titik awal untuk menutup path. Hasilnya: bentuk filled polygon.

4

Klik dan drag (jangan langsung lepas) untuk membuat anchor point dengan kurva. Arah drag menentukan arah kurva.

5

Buat setidaknya 3 bentuk unik dengan pen tool. Simpan: Latihan03_PenTool_[Nama].svg

🎨 Eksplorasi Tool dan Kreativitas Bentuk — "Geometris"

Buat sebuah komposisi desain dengan tema "Geometris" menggunakan hanya shape tools dasar (tanpa pen tool).


Ketentuan:
  • Ukuran canvas: A4 portrait
  • Minimal 15 objek dalam komposisi
  • Minimal 5 warna berbeda
  • Terapkan prinsip komposisi (balance, contrast, hierarchy)
  • Boleh membuat pattern, ilustrasi, atau abstract art
