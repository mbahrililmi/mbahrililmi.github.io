Edit Mode Lanjutan

Proportional Editing (O) ▼
  • O untuk toggle Proportional Editing
  • Transform mempengaruhi vertices di sekitarnya
  • Scroll wheel saat transform untuk adjust radius pengaruh
  • Fall-off types: Smooth, Sphere, Root, Linear, Constant, Random
  • Berguna untuk landscape, organic shapes, smooth deformation
  • Contoh: Angkat area dataran untuk buat gunung
Knife Tool (K) ▼
  • K untuk activate Knife Tool
  • Klik untuk mulai cut, klik lagi untuk corner
  • Enter untuk confirm cut
  • Esc untuk cancel
  • Ctrl+K: Knife Project – potong berdasarkan 2D object
  • Berguna untuk detail custom yang tidak bisa dengan loop cut
Bridge Edge Loops ▼
  • Select 2 edge loops yang terpisah
  • Edge menu (Ctrl+E) → Bridge Edge Loops
  • Menghubungkan 2 loops dengan geometry baru
  • Berguna untuk menghubungkan bagian-bagian model
  • Twist dan segments bisa diatur
  • Contoh: Lengan ke badan, handle ke cangkir
Snap Tool & Precision Modeling ▼
  • Shift+Tab toggle Snapping
  • Snap to: Vertices, Edges, Faces, Grid
  • Hold Ctrl untuk snap saat transform
  • Grid snap untuk precise alignment
  • Vertex snap: Drag vertex ke vertex lain persis
  • Berguna untuk architectural modeling yang harus akurat

Modifier Stack

Konsep Modifier Stack

Modifier diterapkan secara berurutan dari atas ke bawah. Urutan modifier sangat mempengaruhi hasil. Modifier bersifat non-destructive – model asli tidak berubah hingga modifier di-apply.

Array
Duplikasi objek berulang. Relative/Absolute/Object Offset. Pagar, tangga, pattern.
Boolean
Union (gabung), Difference (kurang), Intersect. Untuk lubang, detail kompleks.
Mirror
Cermin di X/Y/Z. Model setengah lalu mirror. Essential untuk karakter dan objek simetris.
Subdivision Surface
Smooth mesh. Level viewport vs render berbeda. Control dengan edge loops.
Solidify
Ketebalan pada plane/thin mesh. Dinding, kertas, panel.
Bevel
Rounded edges. Segments untuk kualitas. Limit method untuk kontrol area.
Curve
Deform objek mengikuti curve. Pipa melengkung, jalan berkelok, rel.
Lattice
Deform dengan cage. Reshape tanpa edit langsung mesh.
Displace
Deform berdasarkan texture. Rocky surfaces, bumpy terrain.
Smooth
Smooth/laplacian smooth untuk even out mesh irregularities.

Rekomendasi urutan modifier (dari atas ke bawah):

  1. Mirror – selalu pertama untuk simetri
  2. Array – duplikasi sebelum smooth
  3. Boolean – potong/gabung sebelum smooth
  4. Bevel – bevel edges sebelum subdivide
  5. Subdivision Surface – smooth terakhir
  6. Displace – setelah subdivide untuk detail

Catatan: Ini panduan umum – selalu cek hasil setelah mengubah urutan.

Boolean & Array Advanced

Boolean Operations

Union
Gabungkan 2 objek menjadi 1. Overlapping area dihapus. Hasil: 1 mesh baru.
Difference
Kurangi objek A dengan bentuk objek B. Membuat lubang atau potongan kompleks.
Intersect
Hanya bagian yang overlap tersisa. Membuat bentuk dari irisan dua objek.
Boolean Workflow
  1. Buat objek A (target – yang akan dipotong/digabung)
  2. Buat objek B (cutter/joiner)
  3. Select objek A
  4. Properties → Modifier → Add → Boolean
  5. Pilih operation: Union/Difference/Intersect
  6. Pilih Object: objek B sebagai operand
  7. Apply modifier
  8. Hapus objek B (optional)

Array + Curve untuk Pattern

  1. Buat objek yang akan diulang (misal: anak tangga)
  2. Tambah modifier Array → Relative Offset Z, count sesuai kebutuhan
  3. Buat Bezier Curve (Shift+A → Curve → Bezier)
  4. Edit curve sesuai jalur yang diinginkan
  5. Pada Array modifier: tambah Curve modifier di bawahnya
  6. Pilih curve sebagai objek deform
  7. Hasil: Array mengikuti jalur curve

Hard Surface Modeling

Apa itu Hard Surface?

Hard surface modeling adalah teknik membuat objek yang memiliki permukaan keras dan sudut tegas – mesin, kendaraan, elektronik, senjata, furniture, arsitektur. Berbeda dari organic modeling yang mengutamakan smooth curves.

Bevel untuk Sharp Edges ▼
  • Objek nyata tidak punya edge sempurna 90° – selalu ada sedikit bevel
  • Ctrl+B → Scroll untuk tambah segments
  • 2-3 segments = realistic bevel
  • Bevel Modifier: Lebih terkontrol, non-destructive
  • Limit method: "Angle" untuk bevel hanya edge tajam
  • Bevel sebelum Subdivision Surface untuk control
Edge Loops untuk Form Control ▼
  • Ctrl+R untuk tambah loop cut
  • Loop dekat edge tajam = "supporting loops"
  • Supporting loops menjaga bentuk saat subdivide
  • Semakin dekat loop ke edge, semakin tajam hasilnya setelah subdivide
  • Rule of thumb: 1 loop di setiap sisi edge yang ingin tetap tajam
Workflow Hard Surface ▼
  1. Mulai dari primitive yang paling dekat dengan bentuk akhir
  2. Block out bentuk besar terlebih dahulu
  3. Tambah detail secara bertahap (dari umum ke khusus)
  4. Gunakan Boolean untuk detail kompleks
  5. Tambah supporting loops di area penting
  6. Apply Bevel modifier untuk realistic edges
  7. Optional: Subdivision Surface untuk smooth keseluruhan
  8. Test render dari berbagai angle
Clean Topology ▼
  • Quads (4 vertices): Selalu utamakan quad faces
  • Tris (3 vertices): Hindari di area deformasi
  • N-gons (5+ vertices): Hindari, terutama untuk subdivision
  • Edge flow: Edges mengikuti form/kontur objek
  • Pole (5+ edges): Minimizekan, tempatkan di area flat
  • Supporting geometry: Edge loops di sekitar sudut tajam

Organic Modeling Introduction

Subdivision Surface Workflow
Mulai low-poly. Tambah edge loops strategis. Subdivide untuk smooth. Control form dengan edge placement. Lebih sedikit geometry = lebih mudah dikontrol.
Sculpting Basics
Sculpt Mode dengan Dynamic Topology. Brushes: Draw, Grab, Smooth, Crease. Untuk organic characters dan creatures. Workflow berbeda dari polygon modeling.
Retopology Concepts
Clean mesh di atas sculpt. Low-poly dari high-poly. Topology lebih baik untuk animasi. (Advanced topic, intro saja)
Hard Surface vs Organic
Hard surface: sharp edges, mechanical, precise. Organic: flowing curves, natural, life-like. Kedua teknik bisa dikombinasikan dalam satu project.

Reference Images

Menambahkan Reference Image
  1. Shift+A → Image → Reference
  2. Browse dan load image file
  3. Posisikan di scene (G untuk move)
  4. Align ke view (front view untuk front reference)
  5. Gunakan Lock Object Modes untuk tidak tertrigger
Setup 3-View Reference (Front, Side, Top)
  1. Siapkan 3 gambar referensi: front, side, top view objek
  2. Tambah image reference di front view (Numpad 1), load front view image
  3. Tambah lagi untuk side view (Numpad 3), load side image
  4. Tambah lagi untuk top view (Numpad 7), load top image
  5. Sesuaikan scale agar proporsional
  6. Model mengikuti referensi dari 3 sisi
Tips Modeling dari Reference

Sering berganti-ganti view (front, side, top) untuk check proporsi. Jangan terlalu terpaku pada satu view saja. Model tampak berbeda dari setiap angle – cek selalu dari perspektif 3D. Reference adalah panduan, bukan aturan mutlak.

Praktikum Pertemuan 12

Latihan 1: Hard Surface Object
  • Buat mechanical object: robot part, tool, atau tech item
  • Gunakan booleans dan bevels
  • Pastikan clean topology
  • Gunakan modifier untuk efisiensi
  • Waktu: 60-90 menit
  • Export sebagai .blend file
Latihan 2: Organic Shape Practice
  • Buat simple character atau creature base mesh
  • Gunakan Subdivision Surface workflow
  • Mirror modifier untuk simetri
  • Low-poly to high-poly workflow
  • Fokus pada smooth, flowing forms
  • Waktu: 60-90 menit
BONUS: Modeling dari Reference
  • Cari reference image objek (produk, kendaraan, dll)
  • Setup reference dalam Blender (minimal 2 view)
  • Model objek mengikuti reference
  • Fokus pada proporsi yang akurat
  • Waktu: 90-120 menit
