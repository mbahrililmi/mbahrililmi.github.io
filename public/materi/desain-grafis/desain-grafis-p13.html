Pengantar Materials & Lighting

Mengapa Materials & Lighting Penting?

Materials dan lighting adalah apa yang membedakan model 3D yang "flat" dengan hasil yang photorealistic. Bahkan model sederhana dapat terlihat amazing dengan materials dan lighting yang tepat, sementara model complex dapat terlihat dull tanpa perhatian pada aspek ini.

Tujuan Pembelajaran
Shader Editor, Principled BSDF, procedural textures, UV mapping, professional lighting, camera techniques.
Shader Editor
Node-based visual programming untuk material. Unlimited possibilities – buat material apapun yang bisa Anda bayangkan.
Render Engines
Eevee: real-time, cepat. Cycles: ray-tracing, photorealistic tapi lebih lambat.

Shader Editor

Membuka Shader Editor
  1. Buka workspace Shading (tab di atas)
  2. Shader Editor muncul di bagian bawah
  3. Select objek untuk lihat materialnya
  4. Tekan Z → Material Preview untuk lihat real-time

Cara Kerja Node Graph

Menambah Node
Shift+A untuk buka Add menu. Pilih kategori: Texture, Color, Vector, Converter, Shader. Drag untuk posisikan.
Menghubungkan Node
Drag dari socket output ke socket input. Warna socket: Kuning=warna, Hijau=shader, Ungu=vector. Ctrl+Shift+Click untuk preview node.

Principled BSDF (Deep Dive)

Satu Node untuk Semua Material

Principled BSDF adalah node material all-in-one yang dapat mensimulasikan hampir semua material di dunia nyata hanya dengan mengatur parameternya. Ini adalah standar industri untuk physically-based rendering (PBR).

Parameter Utama Principled BSDF ▼
  • Base Color: Warna dasar material
  • Metallic (0-1): 0 = dielectric (plastik, kayu), 1 = logam
  • Roughness (0-1): 0 = mirror sempurna, 1 = sangat kasar/matte
  • IOR: Index of Refraction – kaca=1.45, air=1.33, berlian=2.42
  • Transmission (0-1): Transparansi – 1 = full transparan (kaca)
  • Emission: Warna dan strength untuk self-illuminating material
  • Alpha (0-1): Transparansi umum – 0=invisible, 1=opaque
  • Specular: Refleksi specular untuk non-metallic

Contoh Material Umum

Logam (Metal)
Metallic: 1.0
Roughness: 0.1-0.3 (polished) atau 0.5-0.8 (brushed)
Base Color: warna logam
Kaca (Glass)
Metallic: 0
Roughness: 0
Transmission: 1.0
IOR: 1.45
Alpha: 0
Plastik
Metallic: 0
Roughness: 0.3-0.6
Base Color: warna plastik
Specular: 0.5
Kayu (Wood)
Metallic: 0
Roughness: 0.7-0.9
Base Color: coklat natural atau procedural texture
Emissive (Neon)
Emission Color: warna cahaya
Emission Strength: 1-10 (sesuai kebutuhan)
Material bersinar sendiri
Kulit (Skin)
Metallic: 0
Roughness: 0.4-0.6
Subsurface: 0.1-0.3
Base Color: flesh tone

Procedural Textures

Noise Texture ▼
  • Kegunaan: Detail variation, roughness maps, bump maps, cloud effects
  • Scale: Ukuran detail (kecil = detail halus, besar = kasar)
  • Detail: Octaves – semakin tinggi semakin kompleks
  • Distortion: Warping/twisting noise
  • Kombinasikan dengan ColorRamp untuk kontrol penuh
Voronoi Texture ▼
  • Kegunaan: Cell patterns, organic materials, floor tiles, stone
  • Distance metric: Euclidean (bulat), Manhattan (kotak), Chebyshev
  • Scale: Ukuran cell
  • Bisa buat pattern yang terlihat seperti kulit ular, batu akik, dll
Wave & Musgrave Texture ▼
  • Wave: Sine waves – kayu, air, zebra pattern. Scale dan distortion kontrolabel.
  • Musgrave: Fractal noise – terrain, batu, clouds. Natural-looking surfaces.
  • Magic: Psychedelic patterns untuk effect unik
ColorRamp – Node Esensial ▼
  • Remap nilai 0-1 ke warna atau nilai baru
  • Tambah/hapus color stops untuk gradasi
  • Interpolation: Linear, Ease, Cardinal, Constant, B-Spline
  • Gunakan untuk convert grayscale texture menjadi warna
  • Essential untuk kontrol hasil procedural texture

UV Mapping & Image Textures

Apa itu UV Mapping?

UV adalah koordinat 2D yang memetakan permukaan 3D ke bidang 2D. U = horizontal, V = vertical. Dengan UV map, image texture 2D dapat ditempelkan ke model 3D dengan tepat. Seperti membuka/meratakan kulit jeruk.

UV Unwrapping Workflow

  1. Masuk Edit Mode, select semua (A)
  2. Pilih edges yang akan jadi seam (garis potong)
  3. Ctrl+E → Mark Seam (tepi model biasanya bagus untuk seam)
  4. A untuk select all lagi
  5. U → Unwrap untuk proses unwrapping
  6. Buka UV Editing workspace untuk lihat dan edit UV
  7. Minimizekan stretching dan distorsi

Metode Unwrap

Smart UV Project
Otomatis, angle-based. Tidak perlu mark seam. Bagus untuk objek dengan banyak flat faces.
Cube/Cylinder Projection
Untuk bentuk kotak/silinder. Cepat tapi stretching di sudut. Bagus untuk furniture, botol.
Unwrap (Manual Seam)
Terbaik untuk kontrol penuh. Mark seam di tempat yang tidak terlihat, unwrap hasilnya bersih.
Project from View
Project dari sudut pandang current view. Flat, cocok untuk label atau decal di satu sisi.

Bump & Normal Maps

Bump Map
Grayscale image, simulate surface detail. Ringan, bersifat ilusi. Connect ke Normal input via Bump node.
Normal Map
RGB image (biru-ungu dominan). Lebih detail dari bump. Connect via Normal Map node ke Normal input.

Advanced Lighting

Key Light
Lampu utama. Posisi 45° dari subjek. Paling terang. Area light besar untuk bayangan lembut.
Fill Light
Mengisi bayangan dari key light. Posisi berlawanan dengan key. Intensitas 50% dari key. Ratio 2:1 atau 4:1.
Rim Light
Di belakang/sisi subjek. Memisahkan subjek dari background. Beri edge definition. Dramatic effect.

HDRI (High Dynamic Range Image) memberikan environment lighting yang natural dan realistis.

  1. World Properties → Color → klik tombol dot → Environment Texture
  2. Open Image → pilih file .hdr atau .exr
  3. Tambah Mapping node untuk kontrol rotasi
  4. Download HDRI gratis dari Poly Haven (polyhaven.com)

HDRI memberikan refleksi dan bayangan yang terlihat natural karena berasal dari foto lingkungan nyata.

  • Volumetric Lighting: Light shafts, kabut, atmosfer
  • World Properties → Volume → Add Principled Volume shader
  • Atur Density untuk tebal kabut
  • Kombinasikan dengan spot light untuk light shaft effect
  • Emissive Materials: Objek yang memancarkan cahaya sendiri – screen, LED, api

Render Settings & Optimisasi

Aspek Eevee Cycles
Teknologi Rasterization (real-time) Ray-tracing / Path tracing
Kecepatan Sangat cepat Lambat (tergantung scene)
Kualitas Bagus, stylized Photorealistic
Glass/Caustics Approximasi Akurat penuh
Sampling 64–256 samples 128–4096 samples
Cocok untuk Motion graphics, stylized Product viz, photoreal

Tips Optimisasi Render

  • Aktifkan Denoising untuk reduce noise dengan sample lebih rendah
  • Test render di 50% resolution sebelum final render
  • Viewport samples vs Render samples – set berbeda
  • Light Path: Kurangi bounces untuk Cycles yang lebih cepat
  • Render region (Ctrl+B dalam Camera view) untuk test area tertentu

Camera Techniques

Focal Length
24-35mm: Wide angle – environment, arsitektur
50mm: Normal – natural, documentary
85mm: Portrait – flattering, less distortion
100-200mm: Telephoto – product, compression
Depth of Field
Camera Properties → Depth of Field → pilih Focus Object atau atur jarak. F-Stop: lebih rendah = blur lebih banyak. Memfokuskan perhatian pada subjek utama.
Komposisi Kamera
  • Rule of Thirds: Posisikan subjek di 1/3 frame, bukan di tengah persis
  • Leading Lines: Gunakan garis dalam scene untuk memandu mata ke subjek
  • Framing: Gunakan elemen scene untuk membingkai subjek
  • Negative Space: Ruang kosong untuk emphasis
  • Camera roll: Slight tilt untuk dynamic feeling

Praktikum Pertemuan 13

Latihan 1: Material Library (5 Materials)
  • Buat 5 material berbeda: Plastik berwarna, Metal (brushed/polished), Kaca, Emissive/glowing, Kayu atau batu (procedural)
  • Apply ke objek simple dalam satu scene showcase
  • Render hasil dengan lighting yang baik
  • Waktu: 90 menit
Latihan 2: Lighting Scenarios
  • Setup 3 lighting berbeda untuk objek yang sama: Studio (product), Outdoor (HDRI), Dramatic (low-key)
  • Compare 3 render – dokumen settings yang digunakan
  • Waktu: 60-90 menit
Latihan 3: UV Unwrap Practice
  • Buat objek dengan sharp edges (mug, kotak bertekstur, dll)
  • Mark seams di tempat yang tepat
  • Unwrap dan minimizekan stretching
  • Apply image texture dari sebuah foto atau gambar
  • Waktu: 45-60 menit
Latihan 4: Complete Scene
  • Gunakan model dari pertemuan 12
  • Apply material yang sesuai
  • Setup professional lighting
  • Test render Eevee vs Cycles – bandingkan
  • Waktu: 60-90 menit
