# MODUL PERKULIAHAN DESAIN GRAFIS - PERTEMUAN 1

**JUDUL: PENGENALAN ADOBE ILLUSTRATOR / INKSCAPE**

## SUB-CPMK

Mahasiswa mampu menjalankan dan mengetahui tool pada program menggambar yang berbasis vector, salah satunya Adobe Illustrator atau alternatif gratis Inkscape.

## TUJUAN PEMBELAJARAN

Setelah mengikuti perkuliahan ini, mahasiswa diharapkan mampu:
1. Memahami konsep dasar desain grafis berbasis vector
2. Mengenal interface dan workspace Adobe Illustrator/Inkscape
3. Memahami fungsi tool-tool dasar dalam program vector
4. Menggunakan tool-tool dasar untuk membuat objek sederhana
5. Memahami perbedaan desain vector dan raster

## PENGANTAR MATERI

Desain grafis berbasis vector merupakan salah satu keterampilan fundamental yang harus dikuasai oleh seorang desainer. Berbeda dengan gambar berbasis raster (pixel), desain vector menggunakan perhitungan matematis berdasarkan titik, garis, dan kurva. Hal ini membuat desain vector dapat diperbesar atau diperkecil tanpa kehilangan kualitas.

Adobe Illustrator adalah salah satu software profesional untuk desain vector. Namun, untuk keperluan pembelajaran dan praktik, alternatif gratis seperti Inkscape dapat digunakan dengan fitur yang hampir setara.

Pada pertemuan ini, kita akan mengenal workspace, panel-panel, dan tool-tool dasar yang akan menjadi fondasi untuk pembuatan desain yang lebih kompleks di pertemuan-pertemuan berikutnya.

## MATERI

### A. PENGENALAN DESAIN GRAFIS BERBASIS VECTOR

#### 1. Pengertian Desain Vector
- Desain yang dibuat dengan persamaan matematis
- Terdiri dari anchor point, path, dan fill
- Scalable tanpa kehilangan kualitas
- Format file: AI, EPS, SVG, PDF

#### 2. Perbedaan Vector vs Raster
- Vector: matematika, scalable, file kecil, cocok untuk logo
- Raster: pixel, tidak scalable, detail tinggi, cocok untuk foto

#### 3. Kegunaan Desain Vector
- Pembuatan logo dan identitas visual
- Ilustrasi
- Typography dan lettering
- Icon dan symbol
- Desain untuk cetak (billboard, spanduk, dll)

### B. PENGENALAN SOFTWARE

#### 1. Adobe Illustrator (Berbayar)
- Software profesional industri
- Integrasi dengan Adobe Creative Cloud
- Tools dan fitur lengkap

#### 2. Inkscape (GRATIS - Direkomendasikan untuk Pembelajaran)
- Open source dan gratis
- Cross-platform (Windows, Mac, Linux)
- Fitur hampir setara dengan Illustrator
- Format file SVG native
- Download: https://inkscape.org/

### C. MEMULAI PROGRAM

#### 1. Membuka Program
- Klik ikon Inkscape/Illustrator
- Pilih "New Document" atau "Create New"

#### 2. Mengenal Interface
a. **Menu Bar**: File, Edit, Object, Type, Select, Effect, View, dll
b. **Toolbar**: Berisikan tool-tool untuk menggambar
c. **Control Panel**: Pengaturan tool yang sedang aktif
d. **Panel**: Properties, Layers, Swatches, dll
e. **Artboard/Canvas**: Area kerja utama

#### 3. Pengaturan Document
- Size: A4, A3, Custom
- Orientation: Portrait/Landscape
- Units: mm, cm, inch, px
- Color Mode: RGB (digital), CMYK (cetak)

### D. TOOL-TOOL DASAR

#### 1. Selection Tool (V)
- Memilih dan memindahkan objek
- Resize dengan menarik handle
- Rotate dengan handle pojok

#### 2. Direct Selection Tool (A)
- Memilih anchor point individual
- Memodifikasi path
- Edit bentuk objek

#### 3. Rectangle Tool (M)
- Membuat kotak dan persegi
- Tekan Shift untuk membuat persegi sempurna
- Click-drag untuk membuat

#### 4. Ellipse Tool (L)
- Membuat lingkaran dan oval
- Tekan Shift untuk lingkaran sempurna

#### 5. Polygon Tool
- Membuat segi banyak
- Adjust jumlah sisi

#### 6. Star Tool
- Membuat bentuk bintang
- Adjust jumlah sudut

#### 7. Pen Tool (P)
- Tool paling penting untuk vector
- Membuat path custom
- Click untuk straight line
- Click-drag untuk curve

#### 8. Type Tool (T)
- Menambahkan teks
- Point text dan area text

#### 9. Eyedropper Tool (I)
- Mengambil warna dari objek lain

#### 10. Zoom Tool (Z)
- Zoom in/out canvas
- Shortcut: Ctrl + / Ctrl -

### E. PANEL-PANEL PENTING

#### 1. Color Panel
- Mengatur fill (isi) dan stroke (garis)
- RGB atau CMYK mode

#### 2. Swatches Panel
- Palet warna tersimpan

#### 3. Layers Panel
- Mengorganisir objek dalam layer
- Show/hide, lock/unlock layer

#### 4. Stroke Panel
- Mengatur ketebalan dan style garis

#### 5. Properties Panel
- Informasi dan pengaturan objek terpilih

### F. OPERASI DASAR

#### 1. Membuat Objek
- Pilih tool
- Click dan drag di canvas

#### 2. Memindahkan Objek
- Selection tool (V)
- Click dan drag objek

#### 3. Mengubah Ukuran
- Selection tool
- Drag handle corner (tekan Shift untuk proporsional)

#### 4. Rotasi
- Selection tool
- Click sekali lagi pada objek untuk mode rotate
- Drag handle corner

#### 5. Mengubah Warna
- Pilih objek
- Click warna di Color/Swatches panel
- Double click Fill untuk color picker

#### 6. Copy-Paste
- Ctrl + C (copy)
- Ctrl + V (paste)
- Ctrl + D (duplicate)

#### 7. Delete
- Pilih objek
- Tekan Delete atau Backspace

#### 8. Undo/Redo
- Ctrl + Z (undo)
- Ctrl + Shift + Z (redo)

### G. TIPS WORKFLOW

1. Gunakan keyboard shortcuts untuk efisiensi
2. Organize dengan layers
3. Save secara berkala (Ctrl + S)
4. Gunakan guides dan grid untuk presisi
5. Zoom in untuk detail, zoom out untuk overview

## PRAKTIKUM

### LATIHAN 1: Membuat Bentuk Dasar

**Tujuan**: Mengenal dan menggunakan shape tools

**Langkah-langkah:**
1. Buka Inkscape/Illustrator
2. Buat document baru ukuran A4
3. Buat 5 bentuk berikut:
   - Persegi dengan fill merah, no stroke
   - Lingkaran dengan fill biru, stroke hitam 3pt
   - Segitiga dengan fill kuning, no stroke
   - Bintang 5 sudut dengan fill hijau
   - Hexagon dengan fill purple
4. Atur posisi objek dengan rapi
5. Save dengan nama: Latihan01_NamaMahasiswa

### LATIHAN 2: Manipulasi Objek

**Tujuan**: Belajar memindah, resize, rotate, dan duplicate objek

**Langkah-langkah:**
1. Buat lingkaran kecil
2. Duplicate lingkaran tersebut 7 kali (Ctrl+D)
3. Atur lingkaran-lingkaran membentuk pola melingkar
4. Beri warna berbeda untuk setiap lingkaran
5. Buat sebuah persegi di tengahnya
6. Save dengan nama: Latihan02_NamaMahasiswa

### LATIHAN 3: Bermain dengan Pen Tool

**Tujuan**: Memahami cara kerja pen tool untuk membuat path

**Langkah-langkah:**
1. Pilih Pen Tool (P)
2. Buat garis lurus dengan mengklik beberapa titik
3. Tutup path dengan klik di titik awal
4. Buat path curve dengan click-drag
5. Praktik membuat minimal 3 bentuk custom dengan pen tool
6. Save dengan nama: Latihan03_NamaMahasiswa

# MODUL PERKULIAHAN DESAIN GRAFIS - PERTEMUAN 2

JUDUL: DESAIN LOGO

## SUB-CPMK

## Mahasiswa mampu menguasai pengetahuan teknik-teknik dalam pembuatan logo

dan mampu membuat logo dengan Adobe Illustrator atau Inkscape.

## TUJUAN PEMBELAJARAN

## Setelah mengikuti perkuliahan ini, mahasiswa diharapkan mampu:

1. Memahami definisi, fungsi, dan karakteristik logo yang baik
2. Mengenal berbagai jenis dan kategori logo
3. Memahami prinsip-prinsip desain logo
4. Menerapkan teknik-teknik pembuatan logo menggunakan software vector
5. Membuat sketsa dan eksekusi logo sederhana
6. Memahami pentingnya kesederhanaan dan memorable dalam logo

## PENGANTAR MATERI


Logo adalah identitas visual yang merepresentasikan sebuah perusahaan,
produk, organisasi, atau personal brand. Logo yang baik harus memorable,
sederhana, versatile, appropriate, dan timeless.

Desain logo berbeda dengan desain grafis lainnya karena logo harus dapat
berfungsi dalam berbagai ukuran, media, dan kondisi - mulai dari aplikasi
kecil di kartu nama hingga besar di billboard, dari cetak hingga digital.

Pada pertemuan ini, kita akan mempelajari teori dan praktik pembuatan logo,
mulai dari konsep, sketsa, hingga eksekusi digital menggunakan software
vector yang telah dipelajari di pertemuan sebelumnya.

## MATERI


### A. PENGERTIAN DAN FUNGSI LOGO


#### 1. Definisi Logo

      - Simbol grafis yang merepresentasikan identitas
      - Wajah visual dari brand/organisasi
      - Tool komunikasi visual yang instan

#### 2. Fungsi Logo

      - Identifikasi: Membedakan dari kompetitor
      - Informasi: Mengkomunikasikan value brand
      - Legal Protection: Trademark dan copyright
      - Marketing Tool: Membangun brand awareness
      - Symbol of Quality: Mewakili standar dan reputasi

#### 3. Karakteristik Logo yang Baik (Memorable)

**a. Simple (Sederhana)**

         - Mudah diingat dan dikenali
         - Tidak terlalu banyak detail
         - Efektif dalam ukuran kecil

**b. Memorable (Berkesan)**

         - Unik dan distinctive
         - Mudah diingat setelah sekali lihat

**c. Timeless (Tidak Lekang Waktu)**

         - Tidak mengikuti trend sesaat
         - Tetap relevan bertahun-tahun

**d. Versatile (Fleksibel)**

         - Berfungsi di berbagai media
         - Scalable (besar-kecil)
         - Monochrome atau full color

**e. Appropriate (Sesuai)**

         - Cocok dengan industri/bidang
         - Target audience yang tepat

### B. JENIS-JENIS LOGO


#### 1. Wordmark (Logotype)

      - Hanya menggunakan typography/text
      - Contoh: Google, Coca-Cola, FedEx, Disney
      - Cocok untuk nama brand yang unik dan singkat

#### 2. Lettermark (Monogram)

      - Inisial atau singkatan
      - Contoh: IBM, NASA, HBO, CNN
      - Cocok untuk nama brand yang panjang

#### 3. Pictorial Mark (Logo Symbol)

      - Gambar/icon recognizable
      - Contoh: Apple, Twitter, Target
      - Butuh brand awareness tinggi

#### 4. Abstract Mark

      - Bentuk geometris abstract
      - Contoh: Pepsi, Adidas, Nike
      - Symbolic dan konseptual

#### 5. Mascot

      - Karakter ilustrasi
      - Contoh: KFC Colonel, Tokopedia
      - Fun dan approachable

#### 6. Emblem

      - Text di dalam symbol/badge
      - Contoh: Starbucks, Harley-Davidson
      - Traditional dan prestigious

#### 7. Combination Mark

      - Kombinasi text dan symbol
      - Contoh: Burger King, Lacoste, Doritos
      - Paling versatile

### C. PROSES DESAIN LOGO


#### 1. Riset dan Brief

      - Pahami client/brand
      - Tentukan target audience
      - Analisa kompetitor
      - Define values dan personality

#### 2. Brainstorming dan Konsep

      - Mind mapping
      - Keyword association
      - Mood board
      - Eksplorasi ide

#### 3. Sketching

      - Gambar tangan (paper sketching)
      - Buat 10-20 ide berbeda
      - Jangan langsung ke digital
      - Eksplorasi berbagai pendekatan

#### 4. Digitalisasi

      - Pilih 3-5 sketsa terbaik
      - Trace di software vector
      - Refine bentuk dan proporsi

#### 5. Refinement

      - Adjust spacing dan alignment
      - Test di berbagai ukuran
      - Buat variasi (monochrome, reverse)

#### 6. Presentation

      - Mockup aplikasi logo
      - Show case berbagai media

### D. PRINSIP DESAIN LOGO


#### 1. Kesederhanaan (Simplicity)

      - "Less is more"
      - Hindari detail berlebihan
      - Focus pada 1-2 elemen kunci

#### 2. Keseimbangan (Balance)

      - Visual weight yang seimbang
      - Symmetrical atau asymmetrical

#### 3. Proporsi

      - Relationship antar elemen
      - Golden ratio (1.618)
      - Grid system

#### 4. Typography

      - Pilih font yang sesuai
      - Readability dan legibility
      - Custom lettering untuk uniqueness

#### 5. Warna

      - Color psychology
      - Maksimal 2-3 warna
      - Harus work dalam monochrome
      - Contrast yang cukup

#### 6. Negative Space

      - Ruang kosong yang fungsional
      - Contoh: FedEx arrow, Amazon smile

### E. TEKNIK PEMBUATAN LOGO DI INKSCAPE/ILLUSTRATOR


#### 1. Geometric Logo

      - Gunakan shape tools dasar
      - Combine shapes dengan Pathfinder
      - Align dan distribute dengan presisi

#### 2. Typography Logo

      - Pilih atau custom font
      - Convert text to path
      - Edit anchor points
      - Kerning dan tracking

#### 3. Tracing Sketsa

      - Import sketsa scan/foto
      - Place di layer bawah
      - Trace dengan pen tool
      - Delete layer sketsa

#### 4. Pathfinder Operations (Penting!)

      - Unite: Gabung shapes
      - Subtract: Kurangi shape
      - Intersect: Irisan shapes
      - Exclude: Inverse intersect
      - Divide: Pisah jadi segments

#### 5. Grid dan Guides

      - Enable grid (View > Grid)
      - Snap to grid untuk presisi
      - Create guides dengan ruler

#### 6. Symmetry Tool

      - Untuk logo symmetrical
      - Edit satu sisi, mirror otomatis

### F. COLOR THEORY UNTUK LOGO


#### 1. Color Psychology

      - Merah: Passion, energy, urgency
      - Biru: Trust, calm, professional
      - Kuning: Optimism, cheerful, youth
      - Hijau: Growth, health, nature
      - Orange: Friendly, confidence, warmth
      - Purple: Luxury, creativity, wisdom
      - Hitam: Power, elegance, sophistication
      - Putih: Purity, simplicity, clean

#### 2. Color Combinations

      - Monochromatic: Satu warna, berbeda shade
      - Complementary: Berlawanan di color wheel
      - Analogous: Berdekatan di color wheel
      - Triadic: 3 warna equal distance

#### 3. Tips Warna Logo

      - Test dalam grayscale
      - Contrast untuk readability
      - Pertimbangkan cultural meaning

### G. TECHNICAL SPECIFICATIONS


#### 1. File Format

      - Vector: AI, SVG, EPS, PDF
      - Raster: PNG (transparent), JPG

#### 2. Variasi yang Diperlukan

      - Full color
      - Monochrome (black/white)
      - Reverse (untuk background gelap)
      - Horizontal dan vertical layout

#### 3. Size Testing

      - Test di ukuran favicon (16x16px)
      - Test di ukuran large billboard
      - Harus tetap jelas dan recognizable

## PRAKTIKUM


LATIHAN 1: Geometric Logo Design

Tujuan: Membuat logo geometric sederhana dengan shape tools

Langkah-langkah:
1. Buka Inkscape/Illustrator
2. Buat document baru 1000x1000px
3. Buat logo menggunakan minimal 3 shape dasar (circle, rectangle, triangle)
4. Gunakan Pathfinder untuk combine shapes
5. Tema: "Mountain" atau "Arrow"
6. Gunakan maksimal 2 warna
7. Test logo dalam size kecil (50x50px)
8. Save: Latihan01_GeometricLogo_Nama

LATIHAN 2: Lettermark Logo

Tujuan: Membuat monogram dari inisial nama

Langkah-langkah:
1. Ambil 2-3 huruf inisial nama Anda
2. Buat typography dengan font bold
3. Convert to path (Path > Object to Path)
4. Edit dan customize bentuk huruf
5. Overlap huruf untuk membuat unique design
6. Gunakan negative space dengan cerdas
7. Buat dalam monochrome (hitam/putih)
8. Save: Latihan02_Lettermark_Nama

LATIHAN 3: Logo dengan Grid System

Tujuan: Memahami proporsi dan precision dengan grid

Langkah-langkah:
1. Enable grid (View > Grid)
2. Buat lingkaran sebagai base
3. Gunakan grid untuk membagi proporsi
4. Buat logo geometric dengan proportion yang perfect
5. Setiap elemen harus align dengan grid
6. Tema bebas (animal, icon, symbol)
7. Export dengan guideline visible untuk presentasi
8. Save: Latihan03_GridLogo_Nama

LATIHAN 4: Tracing dan Digitalisasi

Tujuan: Mengubah sketsa manual menjadi vector

Langkah-langkah:
1. Sketsa logo sederhana di kertas (3 ide berbeda)
2. Foto atau scan sketsa
3. Import ke Inkscape/Illustrator
4. Place di layer background, lock layer
5. Buat layer baru di atas untuk tracing
6. Trace dengan pen tool
7. Refine dan clean up
8. Pilih 1 terbaik untuk diselesaikan
9. Save: Latihan04_VectorTrace_Nama

# MODUL PERKULIAHAN DESAIN GRAFIS - PERTEMUAN 3

JUDUL: DESAIN LAYOUT (UNDANGAN DAN BACKDROP)

## SUB-CPMK

## Mahasiswa mampu menguasai pengetahuan dasar tentang penyusunan dari

elemen-elemen desain yang berhubungan ke dalam sebuah bidang sehingga
membentuk susunan artistik, dan mampu membuat desain layout seperti
undangan dan backdrop.

## TUJUAN PEMBELAJARAN

## Setelah mengikuti perkuliahan ini, mahasiswa diharapkan mampu:

1. Memahami prinsip-prinsip layout dan komposisi desain
2. Mengenal grid system dan penggunaannya dalam layout
3. Memahami hierarchy, balance, dan spacing dalam desain
4. Membuat desain undangan dengan berbagai tema
5. Membuat desain backdrop untuk event
6. Mengaplikasikan typography dan color harmony dalam layout
7. Memahami ukuran cetak dan bleed untuk printing

## PENGANTAR MATERI


Layout adalah pengaturan elemen-elemen desain (teks, gambar, warna, ruang)
dalam sebuah bidang untuk mencapai tujuan komunikasi visual yang efektif
dan estetis. Layout yang baik tidak hanya indah dipandang, tetapi juga
fungsional dan mudah dipahami.

Desain undangan dan backdrop adalah aplikasi praktis dari prinsip layout
yang sering dijumpai dalam pekerjaan desain grafis. Undangan memerlukan
kejelasan informasi dengan sentuhan artistik yang sesuai tema acara,
sedangkan backdrop memerlukan visual impact yang kuat dan terbaca dari jarak
jauh.

Pada pertemuan ini, kita akan mempelajari teori layout, prinsip-prinsip
komposisi, dan aplikasi praktis dalam pembuatan desain undangan dan backdrop
menggunakan software vector yang telah dikuasai.

## MATERI


### A. PRINSIP-PRINSIP LAYOUT DESIGN


#### 1. Balance (Keseimbangan)

**a. Symmetrical Balance**

         - Formal, traditional, stable
         - Elemen sama di kedua sisi axis
         - Cocok untuk: Undangan pernikahan formal

**b. Asymmetrical Balance**

         - Dynamic, modern, energetic
         - Visual weight seimbang tapi tidak mirror
         - Cocok untuk: Event casual, contemporary

**c. Radial Balance**

         - Elemen tersebar dari center point
         - Focal point kuat di tengah

#### 2. Hierarchy (Hierarki Visual)

      - Menentukan urutan baca: what to see first
      - Size: Besar = penting
      - Color: Contrast untuk attention
      - Position: Top-center = prominent
      - Typography: Weight dan style berbeda

      Contoh hierarchy undangan:
      Level 1: Nama acara (judul besar)
      Level 2: Nama pengantin/pembicara
      Level 3: Tanggal, waktu, tempat
      Level 4: Info tambahan

#### 3. Contrast (Kontras)

      - Size contrast: Besar vs kecil
      - Color contrast: Light vs dark
      - Font contrast: Bold vs regular
      - Purpose: Menarik perhatian, emphasis

#### 4. Alignment (Perataan)

      - Left align: Natural, easy to read
      - Center align: Formal, elegant
      - Right align: Unconventional, artistic
      - Justified: Clean, professional
      - Consistency adalah kunci!

#### 5. Proximity (Kedekatan)

      - Group related information together
      - White space untuk separasi
      - Related items = close together
      - Unrelated items = further apart

#### 6. Repetition (Pengulangan)

      - Consistency warna, font, style
      - Creates unity dan coherence
      - Pattern dan rhythm

#### 7. White Space (Negative Space)

      - Ruang kosong bukan "wasted space"
      - Memberi breathing room
      - Meningkatkan readability
      - Professional look

#### 8. Unity (Kesatuan)

      - Semua elemen terasa cohesive
      - Consistent style dan theme
      - Harmonis dan tidak chaos

### B. GRID SYSTEM


#### 1. Pengertian Grid

      - Framework untuk organize content
      - Struktur invisible untuk alignment
      - Consistency across design

#### 2. Jenis Grid

**a. Manuscript Grid (Single column)**

         - Paling simple
         - Cocok untuk: Text-heavy

**b. Column Grid**

         - Multiple columns (2, 3, 4, dst)
         - Flexible layout
         - Cocok untuk: Magazine, newsletter

**c. Modular Grid**

         - Rows dan columns
         - Highly structured
         - Cocok untuk: Complex layouts

**d. Hierarchical Grid**

         - Custom, intuitive
         - Breaking conventional grid
         - Cocok untuk: Web, poster

#### 3. Cara Menggunakan Grid

      - Set up columns dan margins
      - Use guides di Inkscape/Illustrator
      - Align elements to grid
      - Know when to break the grid

### C. TYPOGRAPHY DALAM LAYOUT


#### 1. Pemilihan Font

      - Maksimal 2-3 font families
      - Pairing: Serif + Sans-serif
      - Readable dari jarak jauh (untuk backdrop)
      - Match dengan theme event

#### 2. Hierarchy Typography

      - Heading: 24-72pt (tergantung media)
      - Subheading: 18-36pt
      - Body: 10-14pt
      - Caption: 8-10pt

#### 3. Spacing

      - Kerning: Space antar huruf
      - Tracking: Space overall text
      - Leading: Space antar baris (line height)
      - Margin dan padding yang cukup

#### 4. Readability

      - Line length ideal: 50-75 characters
      - Sufficient contrast dengan background
      - Avoid all caps untuk paragraf panjang
      - Left align untuk body text (bahasa LTR)

### D. COLOR HARMONY DALAM LAYOUT


#### 1. Color Schemes

      - Monochromatic: Elegant, sophisticated
      - Analogous: Harmonious, serene
      - Complementary: Vibrant, energetic
      - Triadic: Balanced, colorful

#### 2. Color untuk Berbagai Theme

      - Wedding: Soft pastels, gold, white
      - Corporate: Blue, gray, professional
      - Birthday: Bright, cheerful colors
      - Formal event: Dark, rich colors

#### 3. Tips Penggunaan Warna

      - 60-30-10 rule:
        * 60% dominant color
        * 30% secondary color
        * 10% accent color
      - Gunakan warna untuk hierarchy
      - Test print result (RGB vs CMYK berbeda)

### E. DESAIN UNDANGAN


#### 1. Jenis Undangan

      - Pernikahan (formal/casual)
      - Ulang tahun
      - Seminar/workshop
      - Graduation
      - Corporate event

#### 2. Elemen Penting Undangan

      - Judul/nama acara
      - Nama host/penyelenggara
      - Tanggal dan waktu
      - Lokasi/venue
      - Dress code (jika ada)
      - RSVP info
      - Ornamen dan dekorasi

#### 3. Format Undangan

**a. Flat Card**

         - Single side atau double side
         - Simple dan ekonomis

**b. Folded Card**

         - Bifold atau trifold
         - Lebih banyak space untuk info

**c. Postcard Style**

         - Casual dan modern

#### 4. Ukuran Standar Undangan (dalam cm)

      - 10 x 14 (portrait)
      - 14 x 10 (landscape)
      - 15 x 21 (portrait - common)
      - 21 x 15 (landscape)
      - Custom: Sesuai kebutuhan

#### 5. Tips Desain Undangan

      - Clear hierarchy informasi
      - Sesuaikan dengan theme event
      - Balance antara text dan decoration
      - Test readability
      - Pertimbangkan metode cetak
      - Sisakan bleed 3-5mm untuk print

### F. DESAIN BACKDROP


#### 1. Fungsi Backdrop

      - Background untuk photo session
      - Branding event
      - Dekorasi stage/venue
      - Social media content

#### 2. Jenis Backdrop

      - Step and Repeat (logo pattern)
      - Photo Booth (themed design)
      - Stage Backdrop (speaker background)
      - Banner/Signage

#### 3. Ukuran Umum Backdrop

      - 2 x 3 meter (standard photo booth)
      - 3 x 4 meter (medium event)
      - 4 x 6 meter (large event/stage)
      - Custom: Sesuai venue

#### 4. Elemen Penting Backdrop

      - Logo atau branding (jika corporate)
      - Judul event (besar dan jelas)
      - Tanggal (optional)
      - Visual elements (decoration, pattern)
      - Sponsor logos (jika ada)

#### 5. Tips Desain Backdrop

      - LARGE TEXT: Terbaca dari 3-5 meter
      - HIGH CONTRAST: Jelas untuk foto
      - SIMPLE composition: Jangan terlalu ramai
      - HORIZONTAL layout: Landscape orientation
      - RESOLUTION tinggi: Minimal 150 dpi untuk ukuran cetak
      - SAFE AREA: Jangan letakkan elemen penting di tepi
      - TEST with photography: Pertimbangkan lighting

#### 6. Technical Specs Backdrop

      - Resolution: 150 dpi minimum (100 dpi acceptable untuk very large)
      - Color mode: CMYK untuk cetak
      - File format: PDF atau TIFF untuk printing
      - Bleed: 5-10cm untuk ukuran besar

### G. WORKFLOW PRAKTIS


#### 1. Setup Document

      - Tentukan size dan orientation
      - Set bleed jika untuk print
      - Color mode: CMYK untuk cetak, RGB untuk digital
      - Resolution: 300 dpi untuk undangan, 150 dpi untuk backdrop

#### 2. Create Grid/Guides

      - Set margins
      - Create column grids
      - Add guides untuk alignment

#### 3. Establish Hierarchy

      - Tentukan level informasi
      - Block out dengan shapes dulu
      - Tentukan size relationships

#### 4. Add Content

      - Typography placement
      - Images dan graphics
      - Ornaments dan decorations

#### 5. Refine

      - Fine-tune spacing
      - Adjust colors
      - Check alignment
      - Proofread text!

#### 6. Export

      - PDF untuk printing (CMYK, with bleed)
      - PNG/JPG untuk digital
      - Save source file (SVG/AI) untuk revisi

## MATERI


### A. RASTER vs VECTOR GRAPHICS


#### 1. Raster/Bitmap Graphics

      - Terdiri dari pixel (picture element)
      - Resolution-dependent
      - Semakin di-zoom, semakin pixelated
      - File size besar untuk high resolution
      - Format: JPG, PNG, GIF, TIFF, PSD
      - Best untuk: Photography, complex images

#### 2. Vector Graphics (Review)

      - Terdiri dari mathematical paths
      - Resolution-independent
      - Scalable tanpa quality loss
      - File size relatif kecil
      - Format: AI, SVG, EPS, PDF
      - Best untuk: Logos, icons, illustrations

#### 3. Kapan Menggunakan Apa?

      - Raster: Photos, web graphics, realistic images
      - Vector: Logos, typography, print design yang scalable

### B. KONSEP FUNDAMENTAL


#### 1. Resolution (DPI/PPI)

      - DPI: Dots Per Inch (printing)
      - PPI: Pixels Per Inch (digital)
      - Standard resolutions:
        * Screen/web: 72 ppi
        * Print standar: 300 dpi
        * Print high quality: 600 dpi
      - Higher resolution = larger file size = more detail

#### 2. Image Size

      - Dimensions: Width x Height (in pixels atau cm/inch)
      - Relationship: Size x Resolution = total pixels
      - Contoh: 10cm x 15cm @ 300dpi = 1181 x 1772 pixels

#### 3. Color Modes

**a. RGB (Red, Green, Blue)**

         - For digital display (screen)
         - Color range: 0-255 per channel
         - 16.7 million colors possible

**b. CMYK (Cyan, Magenta, Yellow, Key/Black)**

         - For printing
         - Subtractive color model
         - Smaller color gamut than RGB

**c. Grayscale**

         - Black and white dengan gray shades
         - 256 shades of gray

**d. Bitmap (1-bit)**

         - Only black or white, no gray
         - Smallest file size

#### 4. File Formats

      - PSD: Photoshop native, layers preserved
      - XCF: GIMP native, layers preserved
      - JPG: Compressed, no transparency, lossy
      - PNG: Lossless, supports transparency
      - TIFF: High quality, lossless, large file
      - GIF: 256 colors, animation, transparency
      - RAW: Unprocessed camera data

### C. MENGENAL SOFTWARE


#### 1. Adobe Photoshop (Berbayar)

      - Industry standard
      - Integration dengan Adobe ecosystem
      - Cloud-based assets
      - Regular updates

#### 2. GIMP (GRATIS - DIREKOMENDASIKAN)

### E. LAYER SYSTEM (KONSEP TERPENTING!)


#### 1. Apa itu Layer?

      - Seperti lembaran transparan yang ditumpuk
      - Edit satu layer tanpa affect layer lain
      - Foundation dari non-destructive editing

#### 2. Layers Panel

      - Show/hide layer: Eye icon
      - Lock layer: Lock icon
      - Layer opacity: 0-100%
      - Blending modes
      - Layer order: Drag to reorder

#### 3. Jenis Layer

**a. Background Layer**

         - Default layer, locked
         - No transparency
         - Convert to normal: Double-click

**b. Normal Layer**

         - Standard image layer
         - Supports transparency

**c. Adjustment Layer**

         - Non-destructive color/tone adjustments
         - Can be modified anytime

**d. Text Layer**

         - Editable text
         - Vector-based (scalable)

**e. Shape Layer**

         - Vector shapes

**f. Smart Object**

         - Non-destructive transforms
         - Can embed other files

#### 4. Layer Operations

      - New layer: Ctrl + Shift + N
      - Duplicate layer: Ctrl + J
      - Merge layers: Ctrl + E
      - Merge visible: Ctrl + Shift + E
      - Delete layer: Drag to trash
      - Rename: Double-click name
      - Group layers: Ctrl + G

#### 5. Layer Blend Modes

      - Normal: Default
      - Multiply: Darken
      - Screen: Lighten
      - Overlay: Contrast
      - Soft Light: Subtle overlay
      - Dan masih banyak lagi...

#### 6. Layer Opacity dan Fill

      - Opacity: Overall transparency
      - Fill: Transparency tanpa affect layer styles

### F. SELECTION TOOLS (FUNDAMENTAL!)


#### 1. Rectangular Marquee Tool (M)

      - Select rectangular area
      - Hold Shift: Perfect square
      - Hold Alt: From center

#### 2. Elliptical Marquee Tool (M)

      - Select circular/oval area
      - Hold Shift: Perfect circle

#### 3. Lasso Tool (L)

      - Freehand selection
      - Quick tapi tidak precise

#### 4. Polygonal Lasso Tool

      - Click untuk titik-titik
      - Untuk selection dengan edge lurus

#### 5. Magnetic Lasso Tool

      - Snap to edges automatically
      - Good untuk high contrast subjects

#### 6. Magic Wand Tool (W)

      - Select by color similarity
      - Tolerance setting: Higher = more colors
      - Contiguous: Connected pixels only

#### 7. Quick Selection Tool (W)

      - Brush-like selection
      - Automatically detect edges
      - Very intuitive untuk beginners

#### 8. Object Selection Tool (Photoshop 2020+)

      - AI-powered object detection
      - Drag around object

#### 9. Selection Modifiers

      - Add to selection: Hold Shift
      - Subtract: Hold Alt
      - Intersect: Shift + Alt
      - Feather: Soften edges
      - Grow/Similar: Expand selection

#### 10. Select Menu Commands

       - Select All: Ctrl + A
       - Deselect: Ctrl + D
       - Reselect: Ctrl + Shift + D
       - Inverse: Ctrl + Shift + I
       - Select > Color Range: Select by color
       - Select > Subject: AI selection (Photoshop)

## MATERI


### A. BASIC TOOLS


#### 1. Move Tool (V)

      - Memindahkan layer atau selection
      - Auto-select layer option
      - Hold Ctrl to temporary activate

#### 2. Crop Tool (C)

      - Memotong image
      - Set aspect ratio
      - Rule of thirds overlay
      - Non-destructive (dapat di-adjust)

#### 3. Eyedropper Tool (I)

      - Sample warna dari image
      - Set foreground color
      - Hold Alt untuk background color

#### 4. Brush Tool (B)

      - Paint dengan foreground color
      - Adjust size: [ ]
      - Adjust hardness
      - Opacity dan flow settings

#### 5. Eraser Tool (E)

      - Menghapus pixels
      - Background layer: Erase to background color
      - Normal layer: Erase to transparency

#### 6. Gradient Tool (G)

      - Create color gradient
      - Linear, radial, angle, reflected, diamond

#### 7. Paint Bucket Tool (G)

      - Fill area dengan color
      - Tolerance setting

#### 8. Clone Stamp Tool (S)

      - Copy pixels dari satu area ke area lain
      - Alt + Click: Set source point
      - Good untuk remove objects

#### 9. Healing Brush Tool (J)

      - Similar to clone stamp
      - Blend dengan surrounding automatically
      - Better untuk skin retouching

#### 10. Spot Healing Brush

       - Click or drag over imperfections
       - Automatic healing
       - Easiest untuk quick fixes

#### 11. Blur Tool

       - Soften areas
       - Reduce detail

#### 12. Sharpen Tool

       - Increase clarity
       - Enhance detail

#### 13. Smudge Tool

       - Push pixels around
       - Blend colors

#### 14. Zoom Tool (Z)

       - Zoom in/out
       - Alt + Scroll mouse wheel

#### 15. Hand Tool (H)

       - Pan around canvas
       - Hold Spacebar untuk temporary activate

### B. ADJUSTMENTS (NON-DESTRUCTIVE EDITING)


#### 1. Brightness/Contrast

      - Basic lightness adjustment
      - Not recommended (destructive)

#### 2. Levels

      - Adjust tonal range
      - Shadows, midtones, highlights
      - Histogram view
      - Better than brightness/contrast

#### 3. Curves

      - Most powerful adjustment
      - Precise control atas tones
      - RGB dan individual channels

#### 4. Hue/Saturation

      - Change colors
      - Adjust saturation
      - Colorize option

#### 5. Color Balance

      - Add color tints
      - Adjust shadows/midtones/highlights separately

#### 6. Vibrance

      - Smart saturation adjustment
      - Protect skin tones

#### 7. Black & White

      - Convert to grayscale
      - Control individual color contribution

#### 8. Photo Filter

      - Add warming/cooling filter
      - Simulate camera filters

#### 9. Adjustment Layers (Recommended!)

      - Layer > New Adjustment Layer
      - Non-destructive
      - Can be edited anytime
      - Use masks untuk selective adjustment
      - Benefits:
        * Can turn on/off
        * Adjust opacity
        * Apply to multiple layers
        * Save for later

### C. TRANSFORMASI


#### 1. Free Transform (Ctrl + T)

      - Scale, rotate, skew, distort
      - Hold Shift: Constrain proportions
      - Hold Alt: Transform from center
      - Enter: Apply
      - Esc: Cancel

#### 2. Transform Menu (Edit > Transform)

      - Scale
      - Rotate
      - Skew
      - Distort
      - Perspective
      - Warp
      - Rotate 90/180
      - Flip Horizontal/Vertical

### D. MASKS (ADVANCED CONCEPT)


#### 1. Layer Masks

      - Non-destructive hiding
      - Black: Hide, White: Show, Gray: Partial
      - Add mask: Layer panel bottom
      - Edit mask: Click mask thumbnail
      - Disable mask: Shift + Click
      - Apply mask: Right-click > Apply

#### 2. Clipping Masks

      - Use layer below as visibility boundary
      - Alt + Click between layers
      - Or: Right-click > Create Clipping Mask

#### 3. Menggunakan Masks

      - Brush dengan black untuk hide areas
      - Brush dengan white untuk reveal
      - Soft brush untuk smooth edges
      - Gradient untuk gradual transition
      - Selection + Fill untuk precise masking

### E. WORKFLOW BEST PRACTICES


#### 1. Non-Destructive Editing

      - Use adjustment layers
      - Use smart objects
      - Use masks instead of eraser
      - Duplicate original layer

#### 2. Organization

      - Name layers descriptively
      - Group related layers
      - Use color coding
      - Delete unused layers

#### 3. Performance

      - Work at appropriate resolution
      - Merge layers when no longer need editing
      - Purge history jika file lambat
      - Save as you go (Ctrl + S)

#### 4. Keyboard Shortcuts

      - Learn common shortcuts
      - Speed up workflow significantly
      - Customize shortcuts jika perlu

#### 5. Saving dan Export

      - Save working file as PSD/XCF (preserve layers)
      - Export final as:
        * JPG: Photos, smaller file size
        * PNG: Need transparency
        * TIFF: High quality, archival
      - Save different versions untuk different uses
      - Naming convention: Project_Version_Date

## MATERI


### A. FOTOGRAFI JURNALISTIK - KONSEP DAN ETIKA


#### 1. Pengertian Fotografi Jurnalistik

      - Foto yang menceritakan berita atau peristiwa
      - Visual storytelling yang faktual
      - Dokumentasi kejadian aktual
      - Menyampaikan informasi melalui gambar
      - Komplemen dari text journalism

#### 2. Karakteristik Foto Jurnalistik

      - Actual: Kejadian nyata
      - Objective: Tidak bias
      - Timely: Berita terkini
      - Newsworthy: Punya nilai berita
      - Clear: Mudah dipahami
      - Impactful: Memberi dampak emosional

#### 3. Jenis Fotografi Jurnalistik

**a. Spot News**

         - Breaking news, kejadian mendadak
         - Accident, disaster, crime

**b. General News**

         - Planned events
         - Press conference, ceremony

**c. Feature Photography**

         - Human interest stories
         - Slice of life

**d. Sports Photography**

         - Athletic events
         - Action shots

**e. Portrait/Personality**

         - Profile subjects
         - Environmental portraits

#### 4. Prinsip Komposisi Foto Jurnalistik

      - Rule of Thirds
      - Leading Lines
      - Framing
      - Decisive Moment (Henri Cartier-Bresson)
      - Context dan environment
      - Foreground-background relationship

#### 5. Etika Foto Jurnalistik

a. Dibolehkan:

         - Crop untuk improve composition
         - Adjust exposure, contrast (minimal)
         - Convert to B&W
         - Sharpening
         - Color correction untuk match scene

b. TIDAK Dibolehkan:

         - Add atau remove content
         - Excessive manipulation
         - Change reality dari scene
         - Misleading editing
         - Staging events

c. Prinsip Utama:

         - Accuracy
         - Honesty
         - Independence
         - Minimize harm
         - Accountability

#### 6. Langkah-langkah Foto Jurnalistik

      - Riset: Pahami event/issue
      - Planning: Location scouting, timing
      - Shooting: Multiple angles, tell story
      - Selection: Choose most impactful images
      - Editing: Minimal, ethical enhancement
      - Captioning: Accurate, informative
      - Distribution: Timely delivery

### B. PHOTO EDITING - ENHANCEMENT DAN CORRECTION


#### 1. Workflow Photo Editing Profesional


      Step 1: Organization
      - Import dan sort photos
      - Rate dan flag keepers
      - Delete obvious rejects

      Step 2: Basic Adjustments
      - Exposure correction
      - White balance
      - Contrast
      - Highlights/Shadows recovery

      Step 3: Detail Work
      - Sharpening
      - Noise reduction
      - Lens correction

      Step 4: Color Grading
      - Creative color adjustments
      - Mood enhancement

      Step 5: Final Touches
      - Vignette
      - Grain (optional)
      - Output sharpening

#### 2. Exposure dan Tone Adjustment


**a. Levels**

         - Adjust black point, white point, midtones
         - Input levels vs Output levels
         - Per-channel adjustments
         - Auto levels (quick fix)

**b. Curves (Most Powerful!)**

         - S-curve untuk contrast
         - Highlight/shadow control
         - Color grading dengan RGB curves
         - Precise control atas tonal range

**c. Brightness/Contrast**

         - Simple tapi limited
         - Use sparingly

**d. Exposure**

         - Simulate camera exposure adjustment
         - Offset, Gamma correction

**e. Shadows/Highlights**

         - Recover detail di shadows
         - Tame bright highlights
         - Preserve midtones

#### 3. Color Correction


**a. White Balance**

         - Fix color cast
         - Eyedropper pada neutral area
         - Temperature dan Tint sliders

**b. Color Balance**

         - Adjust cyan-red, magenta-green, yellow-blue
         - Separate control for shadows/midtones/highlights

**c. Hue/Saturation**

         - Adjust specific colors
         - Desaturate atau over-saturate
         - Colorize monochrome

**d. Vibrance vs Saturation**

         - Vibrance: Smart saturation, protect skin tones
         - Saturation: Uniform boost semua colors

**e. Selective Color**

         - Adjust CMYK values untuk specific colors
         - Professional color grading

#### 4. Sharpening


**a. Unsharp Mask**

         - Amount: Strength
         - Radius: Edge width
         - Threshold: Protect smooth areas

**b. Smart Sharpen**

         - Better algorithm
         - Reduce noise options

**c. High Pass Sharpening**

         - Filter > Other > High Pass
         - Set blend mode to Overlay
         - More control

      Tips Sharpening:
      - Always last step
      - Don't over-sharpen
      - Sharpen for output (screen different from print)
      - Duplicate layer untuk non-destructive

#### 5. Noise Reduction

      - Filter > Noise > Reduce Noise
      - Luminance noise vs Color noise
      - Preserve detail slider
      - Balance noise reduction dengan sharpness

#### 6. Retouching Techniques


**a. Spot Healing Brush**

         - Quick fixes
         - Small blemishes
         - Content-aware healing

**b. Healing Brush**

         - More control
         - Set source point
         - Blend dengan surrounding

**c. Clone Stamp**

         - Exact copy dari source
         - Hard edges
         - Good untuk patterns

**d. Patch Tool**

         - Drag selection to clean area
         - Good untuk larger areas

**e. Content-Aware Fill**

         - Select area, Edit > Fill > Content-Aware
         - AI-powered removal
         - Amazing results untuk backgrounds

**f. Frequency Separation (Advanced!)**

         - Separate texture dari color
         - Perfect skin retouching
         - Process:
           * Duplicate layer 2x
           * Low frequency: Gaussian Blur
           * High frequency: Apply Image (Subtract, Add blending, Scale 2, Offset 128)
           * Set high freq to Linear Light blend mode
           * Retouch color on low freq layer
           * Retouch texture on high freq layer

#### 7. Portrait Retouching

      - Remove blemishes: Spot healing
      - Even skin tone: Frequency separation
      - Whiten teeth: Hue/Saturation + mask
      - Brighten eyes: Dodge tool atau adjustment layer
      - Enhance lips: Saturation + sharpness
      - Smooth skin: Blur + mask (subtle!)
      - Dodge and Burn: Sculpt face dengan light
      - Important: Keep it natural!

## MATERI


### C. COLOR GRADING


#### 1. Color Grading vs Color Correction

      - Color Correction: Fix problems, make natural
      - Color Grading: Creative decisions, mood, style

#### 2. Popular Color Grades

      - Teal and Orange (cinematic)
      - Matte finish (faded blacks)
      - Warm vintage
      - Cool blue
      - High contrast B&W
      - Monochromatic

#### 3. Techniques


**a. Curves Color Grading**

         - Adjust individual RGB channels
         - Lift shadows (add color to darks)
         - Lower highlights (add color to lights)

**b. Color Lookup Tables (LUTs)**

         - Pre-made color grades
         - Layer > New Adjustment Layer > Color Lookup
         - Instant cinematic look

**c. Gradient Maps**

         - Map tones to colors
         - Great untuk dramatic effects
         - Blend modes untuk subtlety

**d. Split Toning**

         - Different colors untuk shadows dan highlights
         - Hue/Saturation adjustment layers + masks
         - Or use Color Balance

#### 4. Creating Consistent Look

      - Save adjustment layer presets
      - Batch processing untuk consistency
      - Reference image untuk match

### D. MASKING TECHNIQUES (ESSENTIAL!)


#### 1. Layer Masks Basics (Review)

      - Black hides, white reveals
      - Gray = partial transparency
      - Paint dengan brush untuk refine

#### 2. Quick Mask Mode (Q)

      - Visualize selection sebagai red overlay
      - Paint selection dengan brush
      - Exit quick mask = active selection

#### 3. Color Range Masking

      - Select > Color Range
      - Select by color
      - Refine dengan fuzziness
      - Great untuk skies, solid colors

#### 4. Luminosity Masking

      - Select by brightness
      - Ctrl + Click RGB channel
      - Refine dengan levels atau curves
      - Professional technique untuk landscapes

#### 5. Channel Masking

      - Use color channels untuk difficult selections
      - Duplicate channel
      - Increase contrast (levels/curves)
      - Load as selection
      - Perfect untuk hair, trees, smoke

#### 6. Select and Mask Workspace

      - Select > Select and Mask
      - Refine Edge tools
      - Adjust edge detection
      - Decontaminate colors
      - Output to layer mask
      - Best untuk complex selections

### E. IMAGE MANIPULATION DAN COMPOSITING


#### 1. Planning Composites

      - Concept sketch
      - Gather source images
      - Match perspective
      - Match lighting direction
      - Match color temperature

#### 2. Blending Techniques


**a. Layer Masks**

         - Primary technique
         - Soft brushes untuk edges
         - Feather untuk smooth transition

**b. Blend Modes**

         - Multiply: Darken
         - Screen: Lighten
         - Overlay: Contrast blend
         - Soft Light: Subtle blend
         - Experiment!

**c. Gradient Masks**

         - Smooth transition
         - Blend in smoke, fog, etc.

#### 3. Matching Elements


**a. Match Colors**

         - Image > Adjustments > Match Color
         - Or manual dengan curves

**b. Match Lighting**

         - Add shadows: Multiply layer dengan soft brush
         - Add highlights: Screen layer dengan soft brush
         - Dodge and Burn

**c. Match Perspective**

         - Edit > Transform > Perspective
         - Distort untuk match angles

**d. Match Sharpness**

         - Blur sharp element
         - Or sharpen soft element
         - Consistency adalah key

#### 4. Creating Shadows

      - Duplicate object layer
      - Fill black
      - Transform (skew/distort) untuk ground shadow
      - Blur dengan Gaussian Blur
      - Lower opacity (20-40%)
      - Layer mask untuk fade edges

#### 5. Creating Reflections

      - Duplicate object layer
      - Flip vertical
      - Position below object
      - Layer mask gradient (fade to transparent)
      - Lower opacity
      - Add blur untuk distance

#### 6. Adding Depth

      - Depth of Field (Lens Blur)
      - Atmospheric perspective (haze)
      - Size relationships
      - Overlap elements
      - Foreground/midground/background

#### 7. Adjustment Layer Techniques

      - Use adjustment layers untuk affect whole composite
      - Clip adjustment to specific layers
      - Unify dengan overall color grade
      - Vignette untuk focus

#### 8. Final Polish

      - Overall sharpening
      - Add grain untuk texture
      - Color grading untuk mood
      - Dodge and burn untuk depth

### F. FILTERS DAN EFFECTS


#### 1. Blur Filters

      - Gaussian Blur: Smooth, general purpose
      - Motion Blur: Speed effect
      - Radial Blur: Zoom or spin
      - Lens Blur: Simulate camera DoF
      - Field Blur/Tilt-Shift: Selective focus

#### 2. Sharpen Filters

      - Unsharp Mask
      - Smart Sharpen

#### 3. Artistic Filters

      - Oil Paint
      - Watercolor
      - Poster Edges
      - Cutout
      - (Use sparingly, often obvious)

#### 4. Filter Gallery

      - Preview multiple filters
      - Stack filters
      - Experiment dengan combinations

#### 5. Smart Filters

      - Convert to Smart Object dulu
      - Filter > [Any Filter]
      - Becomes editable!
      - Can adjust opacity
      - Can mask filter
      - Non-destructive

#### 6. Neural Filters (Photoshop 2021+)

      - AI-powered filters
      - Skin smoothing
      - Style transfer
      - Colorize B&W
      - Super zoom
      - Sky replacement

### G. SPECIAL TECHNIQUES


#### 1. Double Exposure Effect

      - Two images stacked
      - Blend mode (Screen, Lighten)
      - Masks untuk control
      - Popular artistic effect

#### 2. Dispersion Effect

      - Subject breaking into particles
      - Use Liquify filter
      - Particle brushes
      - Multiple layers

#### 3. Levitation Effect

      - Shoot two photos: with stool, without
      - Composite in Photoshop
      - Remove support in editing
      - Add shadow on ground

#### 4. Miniature/Tilt-Shift Effect

      - Blur top and bottom
      - Increase saturation
      - Boost contrast
      - Simulate shallow DoF

#### 5. Duotone/Tritone

      - Grayscale image
      - Gradient Map adjustment
      - Map to two or three colors
      - Vintage poster look
