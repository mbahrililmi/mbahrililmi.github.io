Pengantar Kuis Praktikum

Tujuan Evaluasi

Kuis praktikum pertemuan 11 bertujuan untuk mengevaluasi pemahaman dan kemampuan mahasiswa dalam menggunakan Blender 3D dasar. Ini bukan UTS atau UAS – ini adalah kuis tengah modul untuk memastikan semua mahasiswa telah menguasai fondasi sebelum melanjutkan ke materi advanced di pertemuan 12-15.

Yang Diuji
Interface Blender, navigasi, transformasi objek, mesh editing dasar, modifier, material, dan rendering.
Format
Praktikum langsung di Blender. Membuat objek 3D sesuai skenario dalam waktu yang ditentukan.
Output
File .blend + render PNG. Dikumpulkan melalui platform yang ditentukan oleh dosen.
2

Format & Aturan Kuis

Ketentuan Umum
  • Kuis dilaksanakan secara individual
  • Durasi: 90-120 menit (sesuai arahan dosen)
  • Software: Blender versi 3.x atau 4.x
  • Tidak boleh menggunakan asset/file dari luar kecuali ditentukan
  • Boleh membuka dokumentasi Blender (blender.org/docs)
  • Dilarang copy-paste dari teman atau internet
Peringatan

Ketidakjujuran akademik (mencontek, plagiarisme) akan mendapat nilai 0 dan sanksi akademik sesuai peraturan institusi. Kerjakan sendiri dengan kemampuan yang telah dipelajari.

Format Pengumpulan
  • File .blend dengan nama: Kuis11_NIM_Nama.blend
  • Render PNG: Kuis11_NIM_Nama.png
  • Resolusi render: minimal 1280×720 (HD)
  • Deadline: Sesuai arahan dosen
  • Upload ke platform: Google Classroom / LMS

Rubrik Penilaian

Kriteria Bobot Deskripsi
Kelengkapan Objek 25% Semua objek yang diminta hadir dalam scene. Sesuai deskripsi skenario.
Teknik Modeling 25% Penggunaan Edit Mode, Extrude, Loop Cut, Bevel, dan tools lainnya dengan benar.
Modifier 15% Minimal 2 modifier digunakan secara tepat dan efektif.
Material 15% Minimal 3 material berbeda. Warna, metallic, roughness diatur dengan tepat.
Lighting & Camera 10% Pencahayaan memadai, kamera well-positioned, komposisi render baik.
Kualitas Render 10% Render bersih, tidak terlalu gelap/terang, resolusi cukup.
Nilai A (85-100)
Semua kriteria terpenuhi dengan baik. Teknik tepat, hasil rapi, creative execution.
Nilai B (70-84)
Mayoritas kriteria terpenuhi. Beberapa minor issue pada teknik atau material.
Nilai C (55-69)
Objek ada tapi teknik kurang tepat. Material atau lighting kurang optimal.

Skenario Tugas (Contoh)

Skenario A: Still Life – Meja Kerja ▼

Deskripsi: Buat scene still life berupa meja kerja sederhana dengan objek-objek di atasnya.

Objek yang harus dibuat:

  • Meja (1 buah) – gunakan Cube, edit proporsi
  • Laptop atau buku (1 buah)
  • Gelas minum (1 buah) – gunakan Cylinder
  • Lampu meja (1 buah) – optional bonus

Persyaratan teknis: Edit Mode wajib digunakan, minimal 2 modifier, minimal 3 material berbeda, render dari sudut yang menarik.

Skenario B: Scene Outdoor Sederhana ▼

Deskripsi: Buat scene outdoor sederhana dengan beberapa elemen alam dan buatan manusia.

  • Ground plane
  • Minimal 2 pohon sederhana (silinder + sphere)
  • 1 bangunan/rumah sederhana
  • Jalur/path (plane yang di-edit)

Persyaratan teknis: Array modifier untuk elemen berulang, material dengan warna alam (hijau, coklat, abu-abu), Sun light untuk pencahayaan outdoor.

Skenario C: Product Display ▼

Deskripsi: Buat display produk 3D sederhana (bebas memilih produk: botol, kaleng, kotak, dll).

  • Produk utama (1 buah, detail cukup)
  • Alas/platform display
  • Background/backdrop sederhana

Persyaratan teknis: Bevel modifier untuk edge halus, material realistis (metallic untuk logam, transmission untuk kaca), 3-point lighting setup, kamera dengan focal length 85mm.

Review Materi Kuis

Poin Kunci yang Harus Dikuasai

Pastikan Anda menguasai semua poin berikut sebelum kuis. Gunakan checklist ini sebagai persiapan.

  • Interface: Mengenal semua panel – 3D Viewport, Outliner, Properties, Timeline
  • Navigasi: Orbit (Middle Mouse), Pan (Shift+MMB), Zoom (Scroll), Numpad views
  • Add Object: Shift+A → Mesh → pilih primitives
  • Transform: G (move), R (rotate), S (scale) + X/Y/Z untuk axis
  • Edit Mode: Tab untuk masuk, 1/2/3 untuk vertex/edge/face select
  • Extrude: E dalam Edit Mode untuk tarik face/edge
  • Loop Cut: Ctrl+R untuk tambah edge loop
  • Inset: I untuk buat face lebih kecil dalam face
  • Bevel: Ctrl+B untuk buat edge rounded
  • Modifiers: Properties → Wrench tab → Add Modifier
  • Material: Properties → Material tab → New → atur Base Color, Metallic, Roughness
  • Lighting: Shift+A → Light → Point/Sun/Area
  • Camera: Numpad 0 untuk camera view, F12 untuk render
  • Save: Ctrl+S untuk simpan, Ctrl+Shift+S untuk Save As

Tips & Strategi Kuis

Manajemen Waktu
  • 15 menit: Baca skenario, rencanakan
  • 45 menit: Modeling semua objek
  • 20 menit: Material dan lighting
  • 10 menit: Camera setup dan render
  • 5-10 menit: Review dan kumpul
Prioritas
  • Selesaikan semua objek DULU, baru detail
  • Simple tapi complete > complex tapi tidak selesai
  • Jangan terlalu lama di satu objek
  • Render dulu, polish nanti jika ada waktu
Common Mistakes yang Harus Dihindari ▼
  • Lupa keluar dari Edit Mode sebelum add object baru
  • Scale di Object Mode – gunakan Apply Scale (Ctrl+A) sebelum modifier
  • Render terlalu gelap – pastikan ada minimal 1 light yang cukup kuat
  • Modifier tidak di-apply saat diperlukan
  • Lupa beri material – semua objek harus punya material
  • Tidak save file – simpan berkala (Ctrl+S)
  • Menggunakan Default Cube tanpa modifikasi – edit dulu!
Shortcut Paling Penting untuk Kuis ▼
  • Shift+A: Add object
  • Tab: Edit/Object mode toggle
  • G/R/S: Transform
  • X/Y/Z setelah transform: Axis constraint
  • E: Extrude
  • I: Inset
  • Ctrl+R: Loop cut
  • Ctrl+B: Bevel
  • A: Select All
  • Alt+Click: Loop select
  • Numpad 0: Camera view
  • F12: Render
  • Ctrl+S: Save
  • Z: Viewport shading pie

Persiapan Sebelum Kuis

Checklist Persiapan Teknis
  • Blender terinstall dan berjalan normal di komputer
  • Familiar dengan interface – bisa navigasi tanpa berpikir lama
  • Latihan modeling minimal 3 objek sederhana sebelum kuis
  • Tahu cara render dan save file .blend
  • Sudah berlatih semua shortcut yang akan digunakan
  • Folder output siap untuk menyimpan file kuis
Pesan Dosen

Kuis ini bukan untuk menakuti, melainkan untuk memastikan fondasi Blender Anda kuat sebelum masuk ke materi advanced. Yang terpenting adalah Anda memahami workflow dasar dan dapat menggunakannya secara praktis. Practice makes perfect – latih terus sebelum kuis!
