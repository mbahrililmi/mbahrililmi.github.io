1 Tujuan Pembelajaran
Setelah mengikuti perkuliahan ini, mahasiswa mampu:
  1. Memahami prinsip-prinsip layout dan komposisi desain
  2. Mengenal grid system dan penggunaannya dalam layout
  3. Memahami hierarchy, balance, dan spacing dalam desain
  4. Membuat desain undangan dengan berbagai tema
  5. Membuat desain backdrop untuk event
  6. Mengaplikasikan typography dan color harmony dalam layout
  7. Memahami ukuran cetak dan bleed untuk printing
💡 Layout yang baik tidak hanya indah dipandang, tetapi juga fungsional dan mudah dipahami. Visual hierarchy menentukan apa yang dilihat pertama kali oleh mata pembaca.
2 Prinsip-Prinsip Layout Design

1. Balance

Keseimbangan visual — symmetrical (formal), asymmetrical (dynamic), atau radial

2. Hierarchy

Urutan baca: size besar = penting. Top-center = prominent

3. Contrast

Perbedaan size, color, atau font untuk menarik perhatian

4. Alignment

Left, center, right, atau justified — konsisten adalah kunci

5. Proximity

Elemen yang terkait ditempatkan berdekatan, beri white space untuk separasi

6. Repetition

Konsistensi warna, font, dan style untuk unity dan coherence

7. White Space

Ruang kosong bukan "wasted space" — memberi breathing room dan professional look

8. Unity

Semua elemen terasa cohesive, consistent style, harmonis dan tidak chaos

Jenis BalanceKarakteristikCocok untuk
SymmetricalFormal, traditional, stable — elemen sama di kedua sisiUndangan pernikahan formal
AsymmetricalDynamic, modern, energetic — visual weight seimbang tapi tidak mirrorEvent casual, contemporary design
RadialElemen tersebar dari center point — focal point kuat di tengahCircular compositions, mandalas

Contoh hierarchy untuk undangan event:

LevelElemenVisual Treatment
Level 1Nama acara / judulFont paling besar, bold, eye-catching
Level 2Nama pengantin / pembicaraMedium-large, distinct style
Level 3Tanggal, waktu, tempatMedium, clear dan readable
Level 4Info tambahan, RSVP, dress codeKecil, secondary importance
3 Grid System

Grid adalah framework invisible untuk mengorganisir content dengan terstruktur. Grid memberikan consistency dan alignment yang profesional.

Manuscript Grid

Single column, text-heavy

Column Grid

Multi-column, magazine/newsletter

Modular Grid

Rows + columns, complex layouts

✅ Cara menggunakan grid di Inkscape/Illustrator: Set margins, buat column grids, tambahkan guides untuk alignment, dan know when to break the grid untuk emphasis visual.
4 Typography dalam Layout
AspekPanduan
Pemilihan FontMaksimal 2-3 font families. Pairing: Serif + Sans-serif bekerja dengan baik
Ukuran Heading24–72pt (tergantung media). Backdrop: sangat besar, terbaca dari 3-5 meter
Ukuran Subheading18–36pt
Ukuran Body10–14pt untuk print, minimal 16px untuk digital
KerningSpasi antar huruf — perlu diperhatikan untuk display text besar
LeadingSpasi antar baris (line height) — 1.5x font size adalah standar yang nyaman
Line LengthIdeal 50-75 karakter per baris untuk body text
5 Color Harmony dalam Layout
ThemeColor Recommendation
Wedding FormalSoft pastels, gold, white, ivory
Corporate EventBlue, gray, professional tones
Birthday PartyBright, cheerful colors
GraduationGold, navy, maroon — prestigious
Casual EventVibrant, complementary colors
🎨 Rule 60-30-10: 60% dominant color (background) · 30% secondary color (main elements) · 10% accent color (highlights, CTA). Proporsi ini menciptakan komposisi warna yang seimbang dan profesional.
6 Desain Undangan
📋 Elemen Penting Undangan ▼
  • Judul/nama acara — paling menonjol secara visual
  • Nama host/penyelenggara
  • Tanggal dan waktu — harus sangat jelas
  • Lokasi/venue
  • Dress code (jika ada)
  • RSVP info (contact dan deadline)
  • Ornamen dan dekorasi sesuai tema

📏 Ukuran Standar Undangan

  • 10 × 14 cm (portrait)
  • 14 × 10 cm (landscape)
  • 15 × 21 cm (portrait — paling umum)
  • 21 × 15 cm (landscape)
  • Custom sesuai kebutuhan

⚙️ Spesifikasi Cetak Undangan

  • Resolution: 300 dpi
  • Color mode: CMYK
  • Bleed: 3mm di semua sisi
  • Safe area: 5mm dari tepi
  • Format export: PDF, PNG
7 Desain Backdrop
🖼️ Jenis dan Ukuran Backdrop ▼
JenisUkuran UmumPenggunaan
Photo Booth2 × 3 meterPhoto session informal
Medium Event3 × 4 meterSeminar, workshop, gathering
Stage Backdrop4 × 6 meterConference, concert, large event
Step and RepeatCustomPress conference, sponsor backdrop
⚠️
Checklist Desain Backdrop:
  • LARGE TEXT — harus terbaca dari jarak 3-5 meter
  • HIGH CONTRAST — jelas untuk foto
  • SIMPLE composition — jangan terlalu ramai
  • HORIZONTAL layout — landscape orientation
  • RESOLUTION tinggi — minimal 150 dpi untuk ukuran cetak
  • SAFE AREA — jangan letakkan elemen penting di tepi
8 Workflow Praktis
1

Setup Document

Tentukan size, orientation, bleed. Color mode: CMYK untuk cetak, RGB untuk digital

2

Create Grid/Guides

Set margins, buat column grids, tambahkan guides untuk alignment

3

Establish Hierarchy

Block out dengan shapes dulu, tentukan level informasi dan size relationships

4

Add Content

Typography placement, images dan graphics, ornaments dan dekorations

5

Refine

Fine-tune spacing, adjust colors, check alignment, proofread text berkali-kali!

6

Export

PDF untuk printing (CMYK, with bleed), PNG/JPG untuk digital, simpan SVG/AI source

11 Ringkasan
📝
Poin-poin kunci Pertemuan 3:
  1. 8 Prinsip Layout — Balance, hierarchy, contrast, alignment, proximity, repetition, white space, unity
  2. Grid System — Manuscript, column, modular, dan hierarchical grid untuk organizing content
  3. Typography — Pemilihan font, hierarchy tipografi, spacing, dan readability
  4. Color Harmony — Color schemes dan rule 60-30-10 untuk komposisi warna
  5. Desain Undangan — Elemen penting, format, ukuran standar, dan spesifikasi cetak
  6. Desain Backdrop — Large format considerations: ukuran text, contrast, resolution, safe area

Layout yang baik adalah tentang mengorganisir informasi secara visual sehingga mudah dipahami. Prinsip ini bersifat universal untuk semua jenis desain.

12 Tugas

Tugas Individu: Desain Undangan & Backdrop untuk Event

Buat 1 set desain lengkap untuk satu event pilihan: Seminar/Workshop, Pernikahan, Ulang Tahun, Company Gathering, atau Graduation Party.

Deliverables

  • Undangan 15×21 cm, 300dpi, CMYK, PDF
  • Backdrop 200×300 cm, 150dpi, CMYK, PDF
  • Design system: warna, font, style konsisten
  • PDF presentation 1 halaman

Kriteria Penilaian

  • Konsep & kreativitas: 20%
  • Penerapan prinsip layout: 25%
  • Typography & readability: 20%
  • Technical execution: 20%
  • Consistency & cohesiveness: 10%
  • Presentation: 5%
