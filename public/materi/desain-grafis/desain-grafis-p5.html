1 Tujuan Pembelajaran
Setelah mengikuti perkuliahan ini, mahasiswa mampu:
  1. Menggunakan basic tools untuk editing dan retouching
  2. Memahami dan menggunakan adjustment layers secara non-destructive
  3. Melakukan cropping, resizing, dan transformasi
  4. Menggunakan masks untuk non-destructive editing
  5. Memahami file formats dan best practices untuk export

🎯 Tema utama pertemuan ini: Non-Destructive Editing — selalu edit dengan cara yang bisa diubah atau di-undo kapan saja, tanpa merusak image original. Ini adalah pendekatan yang digunakan para profesional.

3 Adjustment Layers (Non-Destructive!)

Buat adjustment via Layer > New Adjustment Layer. Adjustment layers adalah cara profesional untuk edit karena dapat dimodifikasi, dimatikan, atau dihapus kapan saja.

AdjustmentFungsiPower Level
LevelsAdjust shadows, midtones, highlights — lebih baik dari Brightness/Contrast⭐⭐⭐
CurvesKontrol tonal range paling presisi — S-curve untuk contrast boost⭐⭐⭐⭐⭐
Hue/SaturationUbah warna, saturation, dan lightness — bisa per color channel⭐⭐⭐⭐
Color BalanceAdd color tints ke shadows/midtones/highlights secara terpisah⭐⭐⭐
VibranceSmart saturation — protect skin tones, enhance dull colors⭐⭐⭐⭐
Black & WhiteConvert ke grayscale dengan kontrol per color channel⭐⭐⭐
Photo FilterSimulate camera filter — warming/cooling⭐⭐
✅ Keuntungan Adjustment Layers: Dapat turn on/off, adjust opacity, apply ke multiple layers, dan save untuk digunakan kembali. Gunakan masks pada adjustment layer untuk selective adjustments!

5 Layer Masks (Advanced!)

🎭 Konsep Layer Mask

  • Black hides (menyembunyikan) — paint hitam untuk hide area
  • White reveals (menampilkan) — paint putih untuk show area
  • Gray = partial transparency — semakin gelap, semakin transparan
  • Soft brush untuk smooth, natural edges
  • Gradient pada mask untuk gradual transition yang halus
Operasi MaskCara
Add layer maskKlik icon di Layers panel (atau Layer > Layer Mask > Reveal All)
Edit maskKlik thumbnail mask (bukan thumbnail layer)
Disable maskShift + Click thumbnail mask
Apply mask permanentRight-click > Apply Layer Mask
Clipping MaskAlt + Click antara dua layer (layer atas clipped ke bawahnya)
💡 Mengapa menggunakan Mask daripada Eraser? Mask bersifat non-destructive — pixel asli tidak terhapus. Anda bisa selalu "reveal" kembali area yang disembunyikan dengan paint putih di mask.
6 Workflow Best Practices
🛡️

Non-Destructive Editing

Gunakan adjustment layers, smart objects, masks instead of eraser, duplicate original layer sebelum edit apapun

📂

Organization

Name layers secara descriptive, group related layers, gunakan color coding di Layers panel, delete unused layers

⚡

Keyboard Shortcuts

Pelajari shortcuts umum untuk speed up workflow. B (brush), E (eraser), V (move), Ctrl+Z (undo), [ ] (brush size)

💾

Saving & Export

Save working file sebagai PSD/XCF (preserve layers). Export final sebagai JPG (photos), PNG (transparency), atau TIFF (high quality archival)

9 Ringkasan

  2. Adjustments — Levels, Curves, Hue/Saturation, Color Balance, Vibrance — selalu gunakan Adjustment Layers

10 Tugas

Tugas Individu: Photo Editing & Layer Compositing

Buat komposisi image dengan menggabungkan minimal 3 foto berbeda menggunakan layers, selections, masks, dan adjustments.

Pilih 1 Theme:

🌌 "Dunia Fantasi" — Surreal composition 📦 "Product Mockup" — Product in environment ✨ "Before-After Transformation" 🎭 "Double Exposure Effect" 🔭 "Miniature World"

Requirements

  • Minimal 3 foto berbeda
  • Proper selection dan isolation
  • Layer masks untuk blending
  • Color correction untuk consistency
  • Adjustment layers untuk mood

Spesifikasi Teknis

  • Size: 1920×1080px atau 1080×1920px
  • Resolution: 300 ppi, RGB
  • Format: PSD/XCF + PNG/JPG
  • Organized layers dengan nama deskriptif
