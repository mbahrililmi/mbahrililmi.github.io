Pengantar Animasi 3D

Apa itu Animation?

Animation adalah seni menghidupkan objek statis melalui gerakan terhadap waktu. Dengan animasi, Anda dapat bercerita, showcase produk, membuat engaging content, dan mengkomunikasikan ide secara dinamis. Skill animasi sangat valuable dalam advertising, film, game, dan digital content creation.

Frame Rate
24 fps (film), 30 fps (TV/video), 60 fps (game). Blender default: 24 fps. Set di Output Properties.
Timeline
Panel bawah. Playback controls, current frame indicator (garis biru), frame range start/end.
Dope Sheet
Overview semua keyframe. Adjust timing dengan mudah. Shift/scale keyframe blocks.

12 Principles of Animation

Dikembangkan oleh Disney Animators (Frank Thomas & Ollie Johnston)

Prinsip-prinsip ini dikembangkan untuk animasi tradisional tapi tetap fundamental dan relevan untuk 3D animation modern. Menguasai konsep ini membuat animasi terasa lebih hidup dan natural.

1
Squash & Stretch
Fleksibilitas. Objek gepeng saat impact, memanjang saat terbang.
2
Anticipation
Persiapan sebelum aksi utama. Berlawanan dengan gerakan utama.
3
Staging
Presentasi yang jelas. Satu ide per moment.
4
Pose to Pose
Animasi dari pose ke pose. Isi in-between setelah.
5
Follow Through
Gerakan sekunder setelah gerakan utama berhenti.
6
Slow In & Out
Easing. Mulai lambat, tengah cepat, akhir lambat.
7
Arcs
Gerakan mengikuti jalur lengkung, bukan garis lurus.
8
Secondary Action
Gerakan pendukung yang memperkuat aksi utama.
9
Timing
Kecepatan mendefinisikan karakter dan mood.
10
Exaggeration
Lebay untuk efek dramatis. Lebih dari realita.
11
Solid Drawing
Form, berat, volume yang konsisten.
12
Appeal
Karisma. Sesuatu yang menarik untuk ditonton.

Keyframe Animation

Konsep Keyframe

Keyframe adalah "snapshot" dari state objek pada frame tertentu. Blender secara otomatis menginterpolasi (menghitung) gerakan antara dua keyframe. Semakin banyak keyframe = semakin banyak kontrol, tapi lebih rumit dikelola.

Workflow Dasar Keyframe

  1. Pergi ke frame pertama (ketik 1 di frame input atau drag timeline)
  2. Posisikan objek sesuai pose awal
  3. Tekan I untuk Insert Keyframe menu
  4. Pilih: Location, Rotation, Scale, atau LocRotScale
  5. Pindah ke frame berikutnya
  6. Ubah pose objek
  7. Tekan I lagi untuk keyframe baru
  8. Ulangi hingga animasi selesai
  9. Tekan Space untuk play
Playback Controls
Space Play/pause
←/→ Previous/next frame
Shift+←/→ Jump ke keyframe
Alt+A Play backward
Auto-Keyframe
Aktifkan record icon di Timeline. Semua perubahan transform otomatis di-key. Lebih cepat tapi hati-hati – semua perubahan ter-record!

Graph Editor

Mengapa Graph Editor?

Graph Editor menampilkan animation curves (F-Curves) secara visual. Di sinilah Anda mengontrol timing dan easing – apakah gerakan linear (robotic) atau bezier (natural). Ini adalah perbedaan antara animasi yang terasa "kaku" dan yang terasa "hidup".

Jenis Interpolasi

Bezier (Default)
Kurva smooth dengan handle. Natural, organic. Ease in/out otomatis. Paling sering digunakan.
Linear
Kecepatan konstan. Terasa mechanical/robotic. Bagus untuk conveyor belt, robot, mekanis.
Constant
Tidak ada interpolasi. Langsung snap ke value. Untuk flip-book effect atau blinking.

Mengontrol Easing

  • Select keyframe dalam Graph Editor
  • V untuk handle type menu: Auto, Vector, Aligned, Free
  • Drag handle untuk control curve shape
  • Ease In: Handle di awal keyframe ditarik ke atas/bawah → mulai lambat
  • Ease Out: Handle di akhir keyframe → berhenti lambat
  • Kanan-klik keyframe → Interpolation Mode untuk ganti mode

Animating Different Properties

Location
Objek bergerak dalam space. X, Y, Z bisa di-key independen. Keyframe: I → Location.
Rotation
Objek berputar. Euler angles (lebih intuitive) atau Quaternion (lebih smooth). Keyframe: I → Rotation.
Scale
Objek membesar/mengecil. Squash & stretch effect. Scale dari 0 untuk logo reveal. Keyframe: I → Scale.
  • Keyframe material property: Hover atas parameter → kanan-klik → Insert Keyframe
  • Alpha animation: Objek fade in/out dengan animate Alpha 0→1
  • Color animation: Base Color berubah warna seiring waktu
  • Emission strength: Animasikan untuk glow on/off effect
  • Roughness: Surface berubah dari matte ke shiny
  • Camera position: Animate G untuk camera move
  • Camera rotation: Animate R untuk pan/tilt
  • Focal length: Animate untuk zoom effect (pull focus)
  • Depth of field focus: Animate focus distance untuk rack focus
  • Camera reveal: Start dari belakang objek, move ke depan

Simple Animation Projects

Bouncing Ball – Animation 101 ▼
  1. Buat bola (UV Sphere)
  2. Frame 1: Bola di atas – keyframe Location Z tinggi
  3. Frame 12: Bola di bawah (touch ground) – keyframe Location Z rendah + Scale XY lebih besar, Z lebih kecil (squash)
  4. Frame 24: Bola naik lagi – keyframe Location Z (lebih rendah dari awal)
  5. Ulangi pattern dengan amplitude menurun
  6. Gunakan Graph Editor: timing lebih cepat saat turun, lebih lambat di puncak (ease in/out)
Logo Reveal Animation ▼
  1. Buat logo 3D atau text
  2. Frame 1: Scale = 0 atau position off-screen
  3. Frame 20-30: Scale = 1 atau position on-screen
  4. Tambah rotation untuk dinamisme: R Z dari 180° ke 0°
  5. Animate emission untuk glow in effect
  6. Ease out di akhir untuk smooth landing
Product Turntable ▼
  1. Posisikan produk di center scene
  2. Metode 1: Animate produk – Frame 1: Rotation Z = 0, Frame 120: Z = 360
  3. Metode 2: Animate camera orbit – gunakan Empty sebagai pivot, parent camera ke empty
  4. Gunakan Linear interpolation untuk smooth rotation konstan
  5. Total: 5 detik (120 frame at 24fps)
  6. Pastikan lighting dari multiple angles
Camera Fly-Through ▼
  1. Buat scene dengan beberapa objek
  2. Pergi ke frame 1, posisikan camera di titik start
  3. Keyframe camera location dan rotation
  4. Pindah ke frame 30, pindah camera ke posisi baru
  5. Keyframe lagi, ulangi untuk path keseluruhan
  6. Gunakan Bezier interpolation untuk smooth camera motion
  7. Aktifkan follow path untuk lebih smooth (optional)

Rendering Animation

Render ke Video (MP4)
  1. Output Properties → Output path: pilih folder tujuan
  2. File Format: FFmpeg Video
  3. Encoding → Container: MPEG-4, Codec: H.264
  4. Output Quality: High Quality atau Perceptually Lossless
  5. Frame range: atur Start dan End frame
  6. Ctrl+F12 atau Render → Render Animation
  7. Tunggu hingga selesai – bisa lama untuk scene complex!
Render ke Image Sequence (Safer)
  • Format: PNG
  • Render frame individual – 0001.png, 0002.png, dst
  • Lebih aman: bisa resume jika crash
  • Compile ke video setelah selesai (Video Editor atau external software)
  • Rekomendasi untuk render panjang atau complex scene
Tips Render Animation

Selalu test render beberapa frame dulu sebelum full render. Render malam hari untuk tidak mengganggu penggunaan komputer. Lower samples + denoise untuk balance kecepatan dan kualitas. Untuk Cycles: GPU rendering jauh lebih cepat dari CPU.

Praktikum Pertemuan 14

Latihan 1: Bouncing Ball Animation
  • Buat bola dengan material yang menarik
  • Animasikan 3-5 pantulan dengan squash & stretch
  • Gunakan Graph Editor untuk proper easing
  • Durasi: 2-3 detik (48-72 frame)
  • Render sebagai video MP4
  • Waktu: 60-90 menit
Latihan 2: Logo Reveal
  • Logo sederhana atau text 3D
  • Animasikan entry: scale, position, atau rotation
  • Tambah material animation (emission glow in)
  • Smooth easing (ease in/out)
  • Durasi: 3-5 detik
  • Waktu: 60-90 menit
Latihan 3: Product Turntable
  • Gunakan produk atau objek model apapun
  • 360° smooth rotation
  • Professional lighting
  • Loop-able animation
  • Durasi: 5-10 detik
  • Waktu: 45-60 menit
