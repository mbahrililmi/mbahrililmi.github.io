Melihat Clip & Proxy Workflow

Thumbnail View
Default view; hover clip untuk scrub preview; klik untuk load di Source Viewer.
List View
View > Show List View — info detail: duration, resolusi, codec, frame rate, audio; kolom bisa di-sort.
Proxy Workflow untuk Playback Smooth ▼

Jika edit footage 4K atau komputer kesulitan playback, gunakan PROXY FILES:

  1. Pilih clip dalam Media Pool
  2. Klik kanan > Generate Optimized Media/Proxy
  3. Pilih format proxy: "Quarter Resolution" atau "Half Resolution"
  4. DaVinci akan generate file proxy resolusi rendah
  5. Edit dengan proxy (smooth), export tetap menggunakan file original (kualitas terjaga)

Toggle proxy mode: Playback > Proxy Mode > Quarter Resolution (otomatis kembali ke resolusi penuh saat export).

Timeline Editing Fundamentals

Menambahkan Clip ke Timeline

Drag dari Media Pool
Drag clip dari Media Pool ke timeline, letakkan di awal atau setelah clip lain.
Three-Point Editing
Load clip di Source Viewer (double-click di Media Pool), mark IN point di timeline (tombol I). Insert: menggeser clip berikutnya. Overwrite: menimpa clip di timeline.
Append at End (F12)
Pilih clip di Media Pool, tekan F12 — clip otomatis ditambahkan ke akhir timeline.

Best practice: mulai dengan drag sederhana, lalu pelajari three-point editing seiring makin terbiasa (jauh lebih cepat untuk editing profesional).

Basic Editing Operations

Blade Tool (B) — Memotong Clip ▼

Fungsi: memotong clip menjadi beberapa bagian.

Cara pakai: tekan B (Blade tool), klik di posisi yang ingin dipotong.

Shortcut: Ctrl+B (blade di playhead), Ctrl+F (blade semua track di playhead).

Use case: menghapus bagian tak diinginkan, split clip untuk rearrange, isolasi bagian untuk efek.

Trim Tool — Normal, Ripple, Roll ▼

Fungsi: menyesuaikan edit point (awal/akhir clip). Hover dekat tepi clip — cursor berubah jadi ikon trim.

  • Normal Trim: drag tepi clip — extend/shorten, meninggalkan gap atau overlap dengan clip sebelahnya
  • Ripple Edit (Alt+drag): menyesuaikan panjang clip, otomatis menggeser clip berikutnya (tanpa gap) — paling sering dipakai, gunakan 90% waktu!
  • Roll Edit (Ctrl+drag pada boundary dua clip): menyesuaikan batas antar dua clip, total durasi tidak berubah — satu clip diperpanjang sementara clip sebelah dipersingkat
Slip & Slide Tool ▼

Slip Tool (Y): mengubah konten di dalam clip sambil mempertahankan posisi dan durasi di timeline. Contoh: clip interview 10 detik di timeline berasal dari sumber 30 detik — slip tool memilih bagian 10 detik mana yang ditampilkan tanpa memindahkan clip di timeline.

Slide Tool (Shift+Alt): memindahkan clip kiri/kanan sambil otomatis menyesuaikan clip sebelahnya agar tidak ada gap. Use case: mengatur ulang timing tanpa membuat gap.

Selection Tool & Delete Options ▼

Selection Tool (A): klik clip = select; drag clip = move (meninggalkan gap); Alt+drag = move dengan ripple (tanpa gap); Ctrl+Click = multi-select; klik track header = select semua clip di track.

  • Delete: hapus clip, meninggalkan gap
  • Backspace: hapus clip, menutup gap otomatis (ripple delete)
  • Ctrl+H: disable clip terpilih (membuatnya invisible tanpa menghapus)
Practice Exercise
  1. Import 5-10 clip video
  2. Susun di timeline secara berurutan
  3. Latih blade tool untuk memotong beberapa bagian
  4. Latih ripple edit (Alt+drag) untuk trim
  5. Latih rearrange clip dengan Alt+drag
  6. Hapus section dengan Backspace (ripple delete)

Goal: terbiasa dengan operasi timeline sampai jadi muscle memory.

Advanced Timeline Techniques

Snapping (N)
Saat enabled, clip akan snap ke playhead, edit point, dan marker. Sebaiknya dibiarkan ON untuk editing presisi.
Linked Selection (Ctrl+Shift+L)
Saat enabled, video dan audio dari clip yang sama terpilih bersama. Disable untuk split edit (audio berbeda dari video berbeda).
Track Locking
Klik kanan track header > Lock Track. Mencegah editing tak disengaja, misal mengunci track musik saat menyusun ulang video.
Track Targeting
Indikator merah di track header menentukan track mana yang terpengaruh operasi. Shift+Click untuk target satu track secara eksklusif.

Transitions

Apa itu Transition?

Transition adalah efek visual antara dua clip — bagaimana satu clip berubah menjadi clip lain.

Cut (Tanpa Transisi)
Perubahan instan dari clip 1 ke clip 2. Transisi paling umum (90% edit). Pacing tercepat. Default untuk kebanyakan scene.
Fade / Dissolve
Blend gradual dari clip 1 ke clip 2. Transisi halus dan lembut. Cocok untuk passage of time, perubahan mood, transisi chapter.
Dips (Black/White)
Dip to Black: fade ke hitam lalu fade dari hitam. Dip to White: serupa dengan putih. Cocok untuk perubahan scene besar, akhir sequence.
Wipes
Satu gambar "menyapu" layar menggantikan yang lain. Banyak variasi (linear, circular, barn doors). Sering terkesan cheesy — gunakan secukupnya untuk look retro/playful.
Golden Rule

Gunakan cut 90% dari waktu. Transisi harus MOTIVATED — punya alasan story, emosi, atau gaya. Jangan gunakan transisi untuk "membuat footage membosankan jadi menarik" — transisi tidak memperbaiki konten yang membosankan!

Menerapkan Transition di DaVinci Resolve

Lokasi: Effects Library > Video Transitions

Kategori: Dissolves, Iris, Motion, Shape, Warp, dan lainnya.

Metode 1: Drag & Drop
Cari transisi di Effects Library, drag ke edit point antara dua clip di timeline. Transisi muncul berwarna ungu di timeline.
Metode 2: Default Transition
Posisikan playhead di edit point, tekan shortcut default transition (Ctrl+T) — transisi default (biasanya Cross Dissolve) langsung diterapkan. Ubah default: klik kanan transisi di Effects Library > Set as Default Transition.
Mengatur Durasi & Curve Transition ▼

Mengubah durasi: drag tepi transisi di timeline (default biasanya 1 detik), atau set angka langsung di Inspector > Transition Duration (misal 0.5s, 2s).

  • Pace cepat: 0.25–0.5 detik
  • Standard: 1 detik
  • Dramatis: 2–3 detik

Transition Curve (Inspector > Transition Curve) menentukan progresi transisi: Linear (rate konstan), Ease In (mulai lambat, akselerasi), Ease Out (mulai cepat, deselerasi), Ease In & Out (paling natural — rekomendasi untuk kebanyakan transisi).

Audio Transitions (Crossfade)

Membuat Audio Crossfade

Pilih kedua audio clip di edit point, terapkan crossfade otomatis. Untuk kontrol lebih, gunakan manual keyframing pada parameter volume.

Tip: selalu gunakan crossfade audio singkat (5-10 frame) bahkan untuk video cut biasa, untuk menghindari klik/pop pada audio.

Basic Video Effects & Keyframing

Inspector Panel — Video Properties

Pilih clip di timeline lalu buka Inspector (panel kanan).

Transform
Position X/Y (geser horizontal/vertikal — reframing, picture-in-picture), Rotation Angle (Dutch angle, creative framing), Scale X/Y (zoom in/out — kunci link icon agar aspect ratio terjaga), Anchor Point (pivot rotasi/scaling).
Crop
Left/Right/Top/Bottom — crop dari tepi untuk hilangkan area tak diinginkan/reframe. Soft — feather pada tepi crop, untuk vignette/gradual crop.
Dynamic Zoom
Zoom yang keyframe-able — Ken Burns effect pada still images.
Composite
Opacity (transparansi 0-100 untuk overlay/fade), Blend Mode (Normal, Screen, Multiply, Overlay, dll — bereksperimen dengan mode berbeda).

Effects Library & Efek Umum

Window > Effects Library (Shift+5). Kategori: OpenFX (plugin pihak ketiga), Resolve FX (built-in), Light, Blur, Sharpen, Stylize, Color, dan lainnya. Cara terapkan: drag efek dari Effects Library ke clip di timeline.

Gaussian Blur
Effects Library > OpenFX > Blur > Gaussian Blur. Parameter: Blur Strength (0-100). Use case: blur background untuk fokus subjek, soften skin, sensor wajah.
Sharpen
Effects Library > OpenFX > Sharpen > Sharpen. Parameter: Intensity. Use case: tajamkan footage agak soft. Warning: over-sharpening menimbulkan artifact jelek!
Glow
Effects Library > OpenFX > Light > Glow. Parameter: Intensity, Threshold, Size. Use case: look dreamy/ethereal, highlight pada lampu/logam/air, scene romantis lembut.
Vignette
Effects Library > OpenFX > Color > Vignette. Parameter: Size, Strength. Use case: menarik fokus ke tengah, look sinematik, menutupi masalah di tepi footage.

Transform (OpenFX) serupa dengan Inspector transform, namun bisa di-stack (diterapkan berulang) — berguna untuk animasi multi-layer kompleks.

Keyframing — Animation Basics

Apa itu Keyframe?

Keyframe adalah marker yang memberi tahu DaVinci Resolve: "Pada titik INI di timeline, parameter harus bernilai INI." Di antara dua keyframe, DaVinci akan interpolasi (animasi otomatis secara smooth).

Cara Membuat & Mengelola Keyframe ▼
  1. Pilih clip di timeline
  2. Pindahkan playhead ke titik animasi mulai
  3. Di Inspector, klik ikon diamond di sebelah parameter (misal "Position X") — diamond berubah merah (keyframe dibuat)
  4. Pindahkan playhead ke titik animasi berakhir, ubah nilai parameter — keyframe baru otomatis terbuat

Melihat keyframe: Inspector > klik nama parameter — membuka animation curve editor secara grafis.

Menghapus keyframe: pindahkan playhead ke posisi keyframe, klik kanan diamond > Delete Keyframe.

Curve types: Linear (speed konstan), Bezier (smooth ease in/out), Hold (nilai tertahan sampai keyframe berikutnya tanpa interpolasi). Klik kanan keyframe di curve editor untuk mengubah tipe.

Zoom In
Awal clip: Scale 100 (keyframe). Akhir clip: Scale lebih besar (keyframe). Hasil: efek zoom in halus.
Slide In dari Kiri
Awal: Position X = -1920 (off-screen, keyframe). 1 detik kemudian: Position X = 0 (keyframe). Hasil: clip masuk dari kiri.
Fade In
Awal: Opacity 0 (keyframe). 1 detik kemudian: Opacity 100 (keyframe). Hasil: clip fade in.

Speed Effects (Simple)

Mengubah Speed Clip

Klik kanan clip di timeline > Change Clip Speed.

  • Speed: 50% = setengah speed (slow motion); 200% = double speed (fast forward); -100% = reverse playback
  • Ripple Timeline: enabled = durasi timeline ikut berubah; disabled = durasi clip tetap, ujung terpotong
  • Frame Blending: enabled = slow motion smooth (blend antar frame); disabled = bisa terlihat patah jika footage shot di 30fps

Tip: untuk slow motion terbaik, shoot di frame rate tinggi (60fps, 120fps), lalu conform ke timeline 24/30fps — hasil smooth tanpa artifact.

Export – Delivering Your Video

Deliver Page

Klik tab "DELIVER" di bawah. Interface: kiri (render settings), tengah (preview), kanan (render queue).

Export Preset YouTube 1080p

  1. Pilih preset: YouTube 1080p — Format MP4, Codec H.264, Resolusi 1920x1080, Bit rate tinggi (8-12 Mbps), Audio AAC 320kbps stereo
  2. Pilih lokasi output
  3. Ubah nama dari "Timeline 1" ke nama deskriptif misal "MyVideo_Final_1080p"
  4. Klik Add to Render Queue, lalu klik Render All

Render time bervariasi sesuai panjang video, efek yang dipakai, dan spek komputer (1-5 menit untuk video 5 menit pada komputer rata-rata).

Custom Export Settings (Detail)

  • MP4: paling universal, bekerja di mana saja — rekomendasi untuk 99% use case
  • QuickTime: kualitas tinggi, file lebih besar, ramah Mac
  • MKV: format open, fleksibel
  • AVI: format lama, biasanya lebih besar
  • H.264: paling kompatibel, standar untuk YouTube/Instagram/web — rekomendasi untuk kompatibilitas maksimal
  • H.265 (HEVC): kompresi lebih baik, file lebih kecil, namun kompatibilitas tidak universal (device lama mungkin tidak bisa play)
  • 1920x1080 (1080p): HD, standar YouTube/kebanyakan platform
  • 3840x2160 (4K): Ultra HD, didukung YouTube, file lebih besar
  • 1280x720 (720p): kualitas cukup, file lebih kecil
  • 640x480 (SD): kualitas rendah, hindari kecuali perlu
  • Instagram Feed: 1080x1080 (square) · Story/Reel: 1080x1920 (vertikal) · IGTV: 1080x1920 atau 1920x1080
  • Frame rate: match timeline — biasanya 30fps atau 24fps (cinematic), 60fps untuk smooth motion
  • 1080p 30fps: 8-12 Mbps (kualitas tinggi) atau 5-8 Mbps (kualitas baik, file lebih kecil)
  • 4K 30fps: 35-50 Mbps
  • Auto: DaVinci memilih bit rate yang sesuai (biasanya cukup baik)
  • Audio Codec: AAC (kompatibilitas terbaik), bit rate 192-320 kbps, sample rate 48kHz, channel stereo (2)

Export Troubleshooting

Export Lama / Lambat ▼
  • Reduce timeline resolution (proxy mode)
  • Reduce export quality/bit rate
  • Tutup program lain (bebaskan RAM)
  • Aktifkan Hardware Acceleration (Settings)
Video Pecah/Pixelated ▼
  • Naikkan bit rate (8-12 Mbps untuk 1080p)
  • Cek kualitas footage sumber (tidak bisa export lebih tinggi dari sumber)
  • Pastikan resolusi sesuai timeline
Audio Tidak Sinkron ▼
  • Cek frame rate sesuai footage sumber
  • Hapus render cache lalu re-export
File Size Terlalu Besar ▼
  • Turunkan bit rate
  • Gunakan Variable Bit Rate (VBR) bukan Constant (CBR)
  • Gunakan H.265 jika kompatibilitas bukan masalah
Upload YouTube Terlihat Lebih Jelek ▼
  • YouTube re-encode — kehilangan kualitas itu normal
  • Tunggu 15-30 menit setelah upload untuk pemrosesan HD
  • Export dengan bit rate lebih tinggi (YouTube punya overhead)

Praktikum: First Video Editing Project

Project & Objective

Buat video 1-2 menit dari 5-10 raw clip. Tujuan: praktik workflow lengkap dari import hingga export.

Step 1-3: Gather, Create Project, Import (20 menit) ▼

Step 1 — Gather Footage (10 menit): Opsi A: rekam sendiri (smartphone OK), 5-10 klip pendek (10-30 detik), tema bebas (day-in-life, tutorial, storytelling, montage). Opsi B: stock footage gratis dari Pexels Videos, Pixabay Videos, atau Videvo — unduh 5-10 klip dengan tema serupa agar kohesif.

Step 2 — Create Project (5 menit): Buka DaVinci Resolve, set Timeline Resolution 1920x1080, Frame Rate 30fps.

Step 3 — Import & Organize (5 menit): Import klip (Ctrl+I atau drag & drop), organisir ke dalam bins.

Step 4-5: Rough Cut & Refine Edit (60 menit) ▼

Step 4 — Rough Cut (30 menit): Tonton semua klip di Source Viewer, susun ke timeline secara berurutan. Goal: konten tersusun, jangan khawatir soal kesempurnaan.

Step 5 — Refine Edit (30 menit): Tonton timeline dari awal sampai akhir (Spacebar), perketat pacing dengan ripple delete pada dead space, susun ulang klip untuk flow lebih baik (Alt+drag), tambahkan B-roll cutaway untuk visual interest bila perlu. Goal: pacing yang smooth dan engaging.

Step 6-7: Transitions & Effects (25 menit) ▼

Step 6 — Add Transitions (10 menit): Identifikasi titik yang butuh transisi (biasanya 1-3 titik). Ingat: gunakan secukupnya! Mayoritas edit harus straight cut.

Step 7 — Add Basic Effects (15 menit): Pilih 1-2 efek untuk dipraktikkan. Contoh zoom: keyframe Scale 100 di awal clip, Scale 120 di akhir clip (zoom in lembut). Contoh fade in: keyframe Opacity 0 di awal clip pertama, Opacity 100 satu detik kemudian.

Step 8-10: Audio, Music & Export (40 menit) ▼

Step 8 — Audio (15 menit): Dengarkan playback, sesuaikan level audio (pilih clip, Inspector > Audio > Volume), target -6 hingga -12 dB, tambahkan crossfade singkat (3-5 frame) di edit point audio.

Step 9 — Add Music, opsional (15 menit): Sumber musik gratis: YouTube Audio Library, Free Music Archive, Incompetech. Import track musik, atur volume di bawah dialog jika ada (-18 hingga -24 dB), fade in di awal dan fade out di akhir.

Step 10 — Export (10 menit): Buka Deliver page, pilih preset YouTube 1080p, render.

FAQ

DaVinci Resolve tidak bisa dibuka/crash ▼

Update driver GPU (NVIDIA/AMD/Intel) via Preferences > Memory and GPU > GPU Configuration > Auto. Jika masih bermasalah, coba Kdenlive sebagai alternatif yang lebih ringan.

Playback choppy/lag ▼

Turunkan timeline quality (Playback > Timeline Proxy Mode > Half Resolution), generate optimized media (klik kanan clip > Generate Optimized Media), atau render cache segmen bermasalah (mark in/out, Playback > Render Cache > Smart).

Bisa pakai Kdenlive sebagai pengganti? ▼

Ya! Semua konsep (timeline editing, transisi, efek, export) sama. Interface sedikit berbeda tapi fundamentalnya identik. Kdenlive lebih ringan dan berjalan lebih baik di komputer lama.

Berapa disk space yang dibutuhkan? ▼

Software ~4GB. Footage 1080p 1 jam (rekaman) ~10-20GB (tergantung codec). Project files ~1-5GB (render cache, optimized media). Export 1 menit 1080p ~100-200MB. Rekomendasi: 50GB+ ruang kosong untuk editing yang nyaman.

Transisi menampilkan black flash/glitch ▼

Transisi membutuhkan "handles" — footage ekstra pada kedua clip di luar edit point. Solusi: pastikan clip punya frame ekstra tersedia (tidak ditrim sampai batas mutlak), drag tepi clip keluar sedikit jika footage sumber masih punya frame ekstra.

Audio crackling/stuttering saat playback ▼

Ini umum terjadi saat editing (tidak mempengaruhi hasil export): render audio cache (Playback > Render Cache > Smart), atau naikkan Audio Buffer Size di Preferences > Audio.

Cara menambah teks/judul, dan edit video vertikal? ▼

Cara cepat: Effects Library > Titles > Basic Title, drag ke timeline, lalu Inspector > Edit text (modul mendatang akan membahas titles secara detail). Untuk video vertikal 9:16: Project Settings > Timeline Resolution > Custom: 1080x1920, atau edit di 16:9 lalu crop saat delivery.

Watermark & DaVinci Resolve vs Resolve Studio ▼

DaVinci Resolve FREE tidak memiliki watermark! Pastikan diunduh dari blackmagicdesign.com resmi dan menggunakan "DaVinci Resolve" (bukan trial "Resolve Studio"). Resolve Studio (berbayar $295 sekali bayar) menambahkan lebih banyak efek/transisi, noise reduction, kolaborasi multi-user, HDR grading, dan tools stereoscopic 3D — namun versi FREE sudah lebih dari cukup untuk pembelajaran dan 99% project, termasuk untuk mata kuliah ini.

Practice makes perfect — semakin sering mengedit, semakin cepat dan intuitif.
