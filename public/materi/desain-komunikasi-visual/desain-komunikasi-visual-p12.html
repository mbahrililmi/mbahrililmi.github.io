Tujuan & Fungsi In House Journal

Definisi

In house journal adalah publikasi internal organisasi (majalah, buletin, newsletter) yang ditujukan untuk karyawan, anggota, atau stakeholder internal — sarana komunikasi korporat yang membangun budaya, engagement, dan keterbukaan informasi.

Informasi
Kebijakan baru, perkembangan organisasi, pengumuman penting.
Engagement
Profil karyawan, prestasi tim, kegiatan sosial — rasa memiliki.
Budaya
Memperkuat nilai-nilai, visi-misi, dan identitas organisasi.
In House Journal vs Publikasi Eksternal ▼
  • Audiens: internal (karyawan/anggota) vs publik/customer
  • Tone: lebih personal dan familial vs formal/promotional
  • Konten: internal news, profil orang dalam vs produk dan brand story
  • Tujuan: engagement dan budaya vs marketing dan reputasi
  • Karakteristik efektif: relevan bagi pembaca internal, konsisten terbit, desain menarik, kontribusi dua arah
3

Corporate Journalism Principles

Story Types
News (pengumuman, milestone), Feature (profil, behind-the-scenes), How-to (tips kerja), Opinion/editorial, Photo story (event coverage), Q&A/interview.
Writing Style
Inverted pyramid untuk news (terpenting dulu). Lead menarik. Bahasa jelas, hindari jargon korporat berlebihan. Kutipan langsung menghidupkan cerita. Human interest angle.
Ethical Considerations ▼
  • Akurasi — verifikasi fakta meski publikasi internal
  • Keseimbangan antara kepentingan manajemen dan suara karyawan
  • Privasi — izin sebelum mempublikasi foto/cerita personal
  • Kredit untuk penulis dan fotografer

Magazine Layout & Editorial Design

Anatomy of Magazine Layout

Headline & Deck
Judul utama + subjudul/ringkasan yang menarik pembaca masuk.
Body Text & Columns
Teks artikel dalam kolom-kolom yang readable.
Images & Captions
Foto/ilustrasi dengan caption — visual anchor halaman.
Pull Quotes
Kutipan menarik diperbesar — visual break dan hook.
Folios
Nomor halaman, nama publikasi, edisi di tepi halaman.
White Space
Ruang napas — kunci layout elegan dan readable.
Common Magazine Layouts ▼
  • Classic/Symmetrical: formal, seimbang — cocok untuk konten serius
  • Modular: blok-blok konten dalam grid — newsletter, berita pendek
  • Picture-dominant: foto besar mendominasi spread — feature, photo story
  • Type-dominant: tipografi sebagai hero — opinion, editorial
  • Spread design: dua halaman sebagai satu kanvas — feature utama
5

Grid Systems

Mengapa Grid?

Grid memberi struktur, konsistensi, dan kecepatan kerja. Pembaca merasakan keteraturan tanpa menyadarinya. Grid bukan penjara — setelah dikuasai, melanggar grid secara sengaja menjadi teknik desain yang kuat.

Komponen Grid
Margins: tepi halaman. Columns: kolom vertikal konten. Gutters: jarak antar kolom. Rows/baseline grid: ritme vertikal. Modules: unit kotak hasil perpotongan.
Jenis Grid untuk Majalah
2-column: simple, formal. 3-column: paling umum, fleksibel. 4+ column: kompleks, banyak kombinasi. Modular grid: kontrol horizontal + vertikal.
Implementing Grid (Scribus/Canva)
  1. New Document → set page size (A4 umum untuk journal)
  2. Page → Manage Guides → buat grid kolom
  3. Set columns (3-4 tipikal) dan gutter width (4-5mm)
  4. Set margins (15-20mm)
  5. Save sebagai template untuk konsistensi antar halaman
  6. Elemen boleh span 1, 2, atau 3 kolom — variasi dalam keteraturan

Typography untuk Publikasi

Typography Hierarchy
Headline (24-48pt) → Deck (14-18pt) → Subheading (12-14pt bold) → Body (9-11pt) → Caption (8-9pt italic). Konsisten di seluruh publikasi.
Font Selection
Serif untuk body cetak (readable panjang — Georgia, PT Serif). Sans-serif untuk heading dan caption (Inter, Open Sans). Max 2-3 font families.
Typography Best Practices ▼
  • Line length 45-75 karakter per baris
  • Leading (line spacing) 120-145% dari font size
  • Hindari widow/orphan (kata tunggal di akhir/awal kolom)
  • Justified text butuh hyphenation; left-align lebih aman
  • Drop caps untuk pembuka artikel feature
  • Pull quotes 18-24pt dengan style berbeda

Production Workflow

Dari Konsep ke Distribusi
  1. Planning: tema edisi, rubrik, penanggung jawab konten, timeline
  2. Content development: penulisan artikel, fotografi, wawancara
  3. Design phase: layout per halaman mengikuti grid dan template
  4. Review & approval: proofreading, persetujuan manajemen/redaksi
  5. Production: export print-ready PDF (CMYK, 300 DPI, bleed) atau versi digital
  6. Distribution: cetak fisik, email PDF, intranet, atau web version

Praktikum & Kuis

Praktikum 1: Grid System Analysis & Application
  • Koleksi contoh spread majalah, identifikasi grid yang digunakan
  • Comparative analysis antar majalah
  • Desain grid Anda sendiri (3-column, A4) sebagai template
Praktikum 2: In House Journal Spread Design
  • Pilih konteks: kampus, perusahaan fiktif, atau organisasi
  • Buat konten: 1 artikel + foto + caption + pull quote
  • Desain spread 2 halaman mengikuti grid
  • Requirements: hierarchy jelas, typography konsisten, white space terjaga
  • Tools: Scribus, Canva, atau Inkscape
