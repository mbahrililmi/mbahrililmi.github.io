### B. PENGANTAR DESAIN KOMUNIKASI VISUAL

#### 1. Apa itu Desain Komunikasi Visual?

**Definisi**:

Desain Komunikasi Visual (Visual Communication Design) adalah proses kreatif yang mengombinasikan elemen-elemen visual - seperti gambar, tipografi, warna, dan layout - untuk menyampaikan pesan, informasi, atau ide kepada audience tertentu dengan tujuan spesifik.

**Kata Kunci dalam Definisi**:
- **Proses Kreatif**: Bukan hanya kebetulan, tetapi pendekatan sistematis
- **Elemen Visual**: Tools yang digunakan (image, type, color, shape, dll)
- **Menyampaikan Pesan**: Fokus pada komunikasi, bukan hanya estetika
- **Audience Tertentu**: Targeted, bukan untuk semua orang
- **Tujuan Spesifik**: Ada objective yang ingin dicapai

**Perbedaan dengan Fine Art**:

| Aspek | Fine Art | Desain Komunikasi Visual |
|-------|----------|--------------------------|
| Tujuan Utama | Ekspresi pribadi artist | Komunikasi efektif kepada audience |
| Client | Diri sendiri/gallery | Client dengan kebutuhan spesifik |
| Fungsi | Tidak harus fungsional | Harus solve problem komunikasi |
| Interpretasi | Terbuka, subjektif | Jelas, targeted message |
| Success Metric | Nilai estetis, emosional | Mencapai tujuan komunikasi |
| Audience | General public, art lovers | Target audience spesifik |

**Evolusi Istilah**:
- Dulu: "Graphic Design" (fokus pada print)
- Sekarang: "Visual Communication Design" (lebih luas, termasuk digital, motion, interactive)
- Trend: "Experience Design", "Communication Design" (semakin integrated)

Contoh: Kampanye anti-rokok dengan visual yang impactful - mengubah perilaku masyarakat.

#### 3. Elemen Dasar dalam DKV

**a. LINE (Garis)**:
- Membentuk shapes dan structure
- Mengarahkan eye movement
- Menciptakan texture dan pattern
- Mengimplikasikan motion atau direction

**b. SHAPE (Bentuk)**:
- Geometric shapes: circle, square, triangle
- Organic shapes: natural, free-form
- Abstract shapes: stylized, simplified
- Positive dan negative shapes

**c. COLOR (Warna)**:
- Hue (warna itu sendiri: red, blue, dll)
- Saturation (intensity warna)
- Value (lightness atau darkness)
- Psychology dan meaning dari warna

**d. TEXTURE (Tekstur)**:
- Visual texture: appearance of surface
- Tactile texture: actual feel (dalam print)
- Adds interest dan realism
- Creates depth dan dimension

**e. SPACE (Ruang)**:
- Positive space: object itu sendiri
- Negative space (white space): area di sekitar object
- Depth: illusion of 3D dalam 2D medium
- Proximity dan grouping

**f. TYPOGRAPHY (Tipografi)**:
- Typeface choice (font selection)
- Hierarchy (size, weight, style)
- Readability dan legibility
- Spacing (kerning, leading, tracking)

**g. IMAGERY**:
- Photography
- Illustration
- Icons dan symbols
- Data visualization

#### 4. Prinsip-Prinsip Desain

**a. BALANCE (Keseimbangan)**:
- Symmetrical: formal, stable
- Asymmetrical: dynamic, interesting
- Radial: circular, center focus

**b. CONTRAST (Kontras)**:
- Size contrast: large vs small
- Color contrast: dark vs light, complementary colors
- Type contrast: serif vs sans-serif, bold vs light
- Purpose: create focal point, add visual interest

**c. EMPHASIS (Penekanan)**:
- Focal point: di mana mata tertarik pertama kali
- Hierarchy: apa yang paling penting?
- Techniques: size, color, placement, contrast

**d. MOVEMENT (Pergerakan)**:
- Eye flow: bagaimana mata bergerak through design
- Direction: guiding viewer's attention
- Rhythm: repetition yang creates pattern

**e. PATTERN dan REPETITION**:
- Consistency: elements yang diulang
- Unity: creating cohesive look
- Variation: repetition with slight changes untuk interest

**f. PROPORTION dan SCALE**:
- Relative size: elemen dalam relation to each other
- Golden ratio: 1:1.618 (natural, pleasing proportion)
- Hierarchy: importance indicated by size

**g. UNITY (Kesatuan)**:
- Cohesiveness: semua elemen work together
- Consistency: style, color, typography aligned
- Gestalt: whole is greater than sum of parts

**h. WHITE SPACE (Negative Space)**:
- Breathing room: space around elements
- Focus: draws attention to important elements
- Elegance: sophisticated, not cluttered

#### 6. Visual Literacy

**Apa itu Visual Literacy?**

Visual literacy adalah kemampuan untuk interpret, understand, dan create visual messages. Dalam era yang dibanjiri informasi visual, skill ini essential.

**Mengapa Penting?**
- 90% informasi yang dikirim ke otak adalah visual
- Visual diproses 60,000x lebih cepat daripada text
- Majority of communication di social media adalah visual
- Effective communication requires visual competency

**Komponen Visual Literacy**:

**a. Visual Perception**:
- Bagaimana kita see dan process visual information
- Gestalt principles (perception psychology)
- Color theory dan perception
- Cultural influences pada interpretation

**b. Visual Interpretation**:
- Reading visual metaphors
- Understanding symbols dan semiotics
- Analyzing composition dan meaning
- Critical viewing

**c. Visual Creation**:
- Menggunakan visual tools untuk communicate
- Applying design principles effectively
- Technical skills dengan software
- Creative problem solving

**d. Visual Thinking**:
- Conceptualizing ideas visually
- Mind mapping dan visual brainstorming
- Storyboarding
- Visual note-taking

## PENGANTAR MATERI

Komunikasi visual adalah fondasi dari semua yang kita lakukan sebagai
desainer. Namun untuk berkomunikasi secara efektif secara visual, kita
harus terlebih dahulu memahami bagaimana audiens kita menerima dan
memproses informasi visual. Di sinilah pemahaman tentang human visual
system menjadi sangat penting.

Mata manusia adalah organ yang luar biasa kompleks, mampu memproses
informasi visual dengan kecepatan yang menakjubkan - sekitar 10 juta bit
informasi per detik. Namun, cara kita "melihat" bukan hanya soal fisiologi
mata, tetapi juga melibatkan proses persepsi yang dipengaruhi oleh otak,
pengalaman, budaya, dan konteks. Memahami bagaimana mata dan otak bekerja
sama dalam memproses visual akan memberikan Anda keunggulan sebagai desainer.

abad ke-20, memberikan kerangka kerja yang sangat valuable untuk memahami
bagaimana manusia mengorganisir informasi visual. Prinsip-prinsip ini
bukan hanya teori abstrak - mereka adalah tools praktis yang dapat langsung
diterapkan dalam setiap desain yang Anda buat.

persepsi, kemudian mengeksplorasi bagaimana pengetahuan ini dapat diterapkan

mengapa beberapa desain "bekerja" secara intuitif, sementara yang lain
terasa membingungkan atau tidak nyaman untuk dilihat.

## MATERI

### A. KONSEP KOMUNIKASI VISUAL

#### 1. Definisi Komunikasi Visual

   Komunikasi visual adalah proses penyampaian ide, informasi, atau pesan
   melalui elemen-elemen visual seperti gambar, warna, bentuk, tipografi,
   dan tata letak. Berbeda dengan komunikasi verbal yang mengandalkan kata-
   kata, komunikasi visual memanfaatkan kekuatan visual untuk menciptakan
   pemahaman yang cepat dan efektif.

   Karakteristik komunikasi visual:
   - Processed faster: Otak memproses visual 60,000x lebih cepat dari teks
   - Universal potential: Visual dapat melampaui batasan bahasa
   - Emotional impact: Visual memicu respons emosional yang kuat
   - Memorable: 80% orang mengingat apa yang mereka lihat vs 20% yang
     mereka baca

#### 2. Model Komunikasi Visual

   Proses komunikasi visual dapat dipahami melalui model komunikasi yang
   diadaptasi:

   SENDER (Desainer/Komunikator)
        ↓
   ENCODING (Proses mengubah ide menjadi visual)
        ↓
   MESSAGE (Visual design - poster, logo, website, dll)
        ↓
   CHANNEL (Medium penyampaian - print, digital, billboard)
        ↓
   DECODING (Audiens menginterpretasi visual)
        ↓
   RECEIVER (Audiens target)
        ↓
   FEEDBACK (Respons/reaksi audiens)
        ↑
   NOISE (Faktor pengganggu - distraksi, misinterpretasi, dll)

   Faktor-faktor yang mempengaruhi efektivitas komunikasi visual:

**a. Context (Konteks)**

      Di mana dan kapan visual dilihat mempengaruhi interpretasi

**b. Cultural Background**

      Simbol dan warna memiliki makna berbeda di budaya berbeda

**c. Personal Experience**

      Pengalaman pribadi audiens mempengaruhi cara mereka melihat

**d. Attention Span**

      Berapa lama audiens akan fokus pada visual Anda

**e. Competing Visuals**

      Apa lagi yang bersaing untuk mendapatkan perhatian audiens

#### 3. Fungsi Komunikasi Visual

**a. Informative (Menginformasikan)**

      - Infografis, signage, instructional graphics
      - Tujuan: menyampaikan informasi dengan jelas dan cepat
      - Contoh: peta, diagram, chart, wayfinding

**b. Persuasive (Membujuk)**

      - Advertising, marketing materials, propaganda
      - Tujuan: mengubah sikap atau perilaku audiens
      - Contoh: iklan produk, kampanye sosial, poster politik

**c. Educational (Mendidik)**

      - Educational materials, tutorials, explainer graphics
      - Tujuan: mengajarkan konsep atau skill
      - Contoh: buku pelajaran, tutorial video, infografis edukatif

**d. Identificatory (Mengidentifikasi)**

      - Branding, logos, corporate identity
      - Tujuan: membedakan dan mengidentifikasi entitas
      - Contoh: logo perusahaan, uniform, packaging

**e. Decorative (Menghias)**

      - Mainly aesthetic purpose dengan komunikasi minimal
      - Tujuan: mempercantik atau menciptakan atmosphere
      - Contoh: wallpaper, pattern, ornamental design

### B. HUMAN VISUAL SYSTEM: ANATOMI DAN FISIOLOGI

#### 1. Anatomi Mata Manusia

   Mata adalah organ kompleks yang bekerja seperti kamera:

**a. Cornea (Kornea)**

      - Lapisan transparan di depan mata
      - Fungsi: memfokuskan cahaya yang masuk
      - Refraksi cahaya pertama kali terjadi di sini

**b. Iris dan Pupil**

      - Iris: bagian berwarna mata (pigmentasi)
      - Pupil: bukaan di tengah iris
      - Fungsi: mengatur jumlah cahaya yang masuk
      - Pupil membesar (dilasi) di tempat gelap, mengecil di tempat terang

**c. Lens (Lensa)**

      - Struktur transparan yang flexible
      - Fungsi: memfokuskan cahaya ke retina
      - Dapat berubah bentuk (accommodation) untuk fokus berbeda

**d. Retina**

      - Lapisan paling dalam mata yang sensitive terhadap cahaya
      - Mengandung photoreceptor cells (sel penerima cahaya)
      - Di sinilah cahaya diubah menjadi sinyal elektrik

**e. Photoreceptors: Rods dan Cones**

      Rods (Sel Batang):
      - Jumlah: ~120 juta per mata
      - Fungsi: penglihatan dalam kondisi cahaya rendah
      - Tidak sensitive terhadap warna (black & white vision)
      - Lebih sensitive terhadap gerakan
      - Concentrated di peripheral vision

      Cones (Sel Kerucut):
      - Jumlah: ~6-7 juta per mata
      - Fungsi: penglihatan warna dan detail
      - 3 jenis: S-cones (blue), M-cones (green), L-cones (red)
      - Concentrated di fovea (pusat penglihatan tajam)
      - Memerlukan cahaya lebih terang

**f. Fovea**

      - Area kecil di tengah retina
      - Konsentrasi cones tertinggi
      - Area dengan ketajaman visual maksimum
      - Ketika kita "fokus" pada sesuatu, kita mengarahkan fovea ke objek

**g. Optic Nerve (Saraf Optik)**

      - Mengirim sinyal dari retina ke otak
      - Blind spot: area tanpa photoreceptors di mana optic nerve keluar

#### 2. Proses Penglihatan

   Step by step bagaimana kita melihat:

#### 1. Cahaya dari objek masuk melalui cornea

#### 2. Iris mengatur jumlah cahaya yang masuk melalui pupil

#### 3. Lens memfokuskan cahaya ke retina

#### 4. Photoreceptors (rods & cones) mendeteksi cahaya

#### 5. Sinyal visual diubah menjadi impuls elektrik

#### 6. Sinyal dikirim melalui optic nerve ke otak

#### 7. Visual cortex di otak memproses sinyal

#### 8. Otak menginterpretasikan informasi - kita "melihat"

   Fun facts:
   - Image yang terbentuk di retina sebenarnya terbalik dan reversed
   - Otak yang kemudian "membalikkan" image sehingga kita melihat dengan
     benar
   - Kita sebenarnya "melihat" dengan otak, bukan mata - mata hanya
     alat sensor

#### 3. Karakteristik Penglihatan Manusia yang Relevan untuk Desain

**a. Field of Vision (Lapangan Pandang)**

      - Horizontal: ~200 degrees (dengan kedua mata)
      - Vertical: ~135 degrees
      - Central vision: ~2 degrees (foveal vision - ketajaman maksimum)
      - Peripheral vision: sisanya (lower resolution, sensitive to motion)

      Implikasi desain:
      - Elemen penting harus di central vision area
      - Peripheral vision tools dapat digunakan untuk context atau
        supporting elements

**b. Visual Acuity (Ketajaman Visual)**

      - Paling tajam di fovea, menurun drastis di periphery
      - Tergantung pada pencahayaan, kontras, dan ukuran objek

      Implikasi desain:
      - Font size minimum untuk readability
      - Contrast requirements untuk visibility
      - Detail halus harus cukup besar

**c. Color Vision**

      - Trichromatic theory: 3 jenis cone untuk red, green, blue
      - Color blindness: ~8% pria, ~0.5% wanita (mostly red-green)

      Implikasi desain:
      - Jangan rely hanya pada warna untuk komunikasi penting
      - Test designs untuk color blindness accessibility
      - Use sufficient contrast

**d. Motion Detection**

      - Peripheral vision sangat sensitive terhadap gerakan
      - Automatic attention reflex terhadap motion

      Implikasi desain:
      - Animation dan motion graphics sangat effective
      - Hati-hati dengan animated ads yang distract

**e. Temporal Resolution**

      - Flicker fusion threshold: ~50-90 Hz
      - Anything faster appears as continuous motion

      Implikasi desain:
      - Video frame rates (24, 30, 60 fps considerations)
      - Animation smoothness

### C. PERSEPSI VISUAL

#### 1. Perbedaan Sensation vs Perception

   Sensation (Sensasi):
   - Proses fisiologis deteksi stimulus oleh sensor (mata)
   - Bottom-up processing
   - Raw data yang diterima retina

   Perception (Persepsi):
   - Proses psikologis interpretasi sensory information
   - Top-down processing
   - Melibatkan memory, expectation, context
   - "Making sense" dari sensory data

   Contoh:
   - Sensation: Mata mendeteksi pola garis-garis hitam di atas putih
   - Perception: Otak menginterpretasi ini sebagai "huruf" dan "kata"

#### 2. Faktor yang Mempengaruhi Persepsi Visual

**a. Past Experience (Pengalaman Masa Lalu)**

      - Kita melihat apa yang kita "expect" untuk dilihat
      - Familiar objects lebih cepat dikenali

**b. Attention (Perhatian)**

      - Selective attention: we focus on certain stimuli
      - Inattentional blindness: we miss things we don't attend to
      - Famous example: "Invisible Gorilla" experiment

**c. Motivation dan Interest**

      - Kita lebih notice hal yang relevan dengan kebutuhan kita
      - "When you're hungry, you notice food signs"

**d. Cultural Background**

      - Budaya mempengaruhi bagaimana kita interpret symbols dan colors
      - Reading direction (left-to-right vs right-to-left) affects layout
        perception

**e. Context**

      - Same stimulus dapat dipersepsikan berbeda dalam context berbeda
      - Contoh: "13" vs "B" dalam sequence "12 13 14" vs "A B C"

**f. Emotional State**

      - Mood mempengaruhi perception
      - Anxious people lebih notice threatening stimuli

#### 3. Visual Illusions dan Implikasinya

   Optical illusions terjadi ketika perception berbeda dari reality -
   demonstrasi bahwa "seeing is NOT always believing".

**a. Geometric Illusions**

      - Müller-Lyer illusion: garis dengan arrow tips berbeda panjang
      - Ponzo illusion: parallel lines tampak tidak parallel
      - Ebbinghaus illusion: circle sama tampak berbeda ukuran

      Implikasi: Konteks dan surrounding elements mempengaruhi perceived
      size dan proportion

**b. Color dan Brightness Illusions**

      - Simultaneous contrast: warna sama tampak berbeda di background
        berbeda
      - Hermann grid: phantom spots di intersections

      Implikasi: Warna tidak exist in isolation - always consider context

**c. Motion Illusions**

      - Peripheral drift: static patterns tampak bergerak
      - Rotating snakes: illusion of rotation

      Implikasi: Pattern design dapat create sense of movement

**d. Ambiguous Figures**

      - Necker cube: 3D cube yang dapat dilihat dari 2 perspectives
      - Rubin's vase: vase vs two faces
      - Old woman/young woman illusion

      Implikasi: Desain dapat memiliki multiple interpretations - pastikan
      intended message jelas

   Mengapa illusions penting untuk desainer?
   - Menunjukkan bahwa perception is constructed, not just recorded
   - Memahami "quirks" visual system membantu kita communicate lebih
     efektif
   - Dapat digunakan kreatif dalam desain (trompe-l'œil effects, dll)

### D. GESTALT PRINCIPLES OF PERCEPTION

Gestalt psychology menyatakan bahwa "the whole is greater than the sum of
its parts". Otak kita secara otomatis mengorganisir elemen visual menjadi
unified wholes berdasarkan certain principles.

#### 1. Law of Proximity (Kedekatan)

   Prinsip: Objek yang berdekatan cenderung dipersepsikan sebagai grup.

   Contoh:
   ● ● ●    ● ● ●    ● ● ●
   Kita melihat 3 grup of 3 dots, bukan 9 individual dots

   Aplikasi dalam desain:
   - Grouping related information dengan spacing
   - White space untuk separate sections
   - Navigation menus: related items didekatkan
   - Form design: related fields grouped together

   Contoh praktis:
   - Business card: info digrup (name/title, contact info, address)
   - Website layout: related content didekatkan

#### 2. Law of Similarity (Kesamaan)

   Prinsip: Objek yang mirip (warna, bentuk, ukuran) dipersepsikan sebagai
   grup atau pattern.

   Contoh:
   ● ○ ● ○ ● ○
   ● ○ ● ○ ● ○
   Kita melihat vertical columns, bukan horizontal rows

   Aplikasi dalam desain:
   - Consistent styling untuk similar elements
   - Color coding untuk categories
   - Icon sets dengan consistent style
   - Typography hierarchy

   Contoh praktis:
   - All buttons di website styled sama
   - Product categories dengan color coding
   - Call-to-action elements dengan styling distinctive

#### 3. Law of Continuity (Kontinuitas)

   Prinsip: Mata mengikuti smooth, continuous paths rather than abrupt
   changes in direction.

   Contoh:
   Kita melihat wavy line yang crosses, bukan separate angled lines

   Aplikasi dalam desain:
   - Leading the eye through layout dengan visual flow
   - Logo design yang menggunakan continuous lines
   - Curved elements untuk smooth transitions
   - Alignment untuk create visual paths

   Contoh praktis:
   - Nike swoosh: continuous curve
   - Magazine layouts: eye flows from headline → image → body text
   - Arrows dan lines untuk guide attention

#### 4. Law of Closure (Penutupan)

   Prinsip: Otak "melengkapi" informasi yang missing untuk membuat complete
   shapes.

   Contoh:
   Circle dengan gaps → kita tetap melihat circle, bukan arcs

   Aplikasi dalam desain:
   - Negative space logos (clever use of gaps)
   - Minimal design yang suggest form tanpa complete outlines
   - Interactive designs (incomplete progress bars)

   Contoh praktis:
   - WWF logo: panda dari black shapes
   - FedEx logo: hidden arrow
   - NBC logo: peacock dari colored shapes

#### 5. Law of Figure-Ground (Figur-Latar)

   Prinsip: Mata memisahkan objects (figures) dari their backgrounds
   (ground).

   Contoh:
   Rubin's vase: bisa lihat vase (figure) atau faces (figure)

   Aplikasi dalam desain:
   - Negative space utilization
   - Contrast untuk separate figure from ground
   - Depth perception dan layering
   - Reversible designs untuk creative effect

   Contoh praktis:
   - Logo dengan negative space
   - Text on background: ensure readability
   - Photography: subject vs background bokeh

#### 6. Law of Common Fate (Nasib Bersama)

   Prinsip: Objects moving in same direction dipersepsikan sebagai grup.

   Aplikasi dalam desain:
   - Animation: elements yang move together seen as related
   - Motion graphics
   - Scrolling effects di websites
   - Carousels dan sliders

   Contoh praktis:
   - Parallax scrolling websites
   - Loading animations
   - Transition effects

#### 7. Law of Symmetry (Simetri)

   Prinsip: Symmetrical elements dipersepsikan sebagai belonging together,
   creating sense of order.

   Aplikasi dalam desain:
   - Balanced layouts
   - Logo design (many logos are symmetrical)
   - Centered compositions
   - Pattern design

   Contoh praktis:
   - Mandala designs
   - Corporate logos
   - Classical website layouts

#### 8. Law of Common Region (Wilayah Bersama)

   Prinsip: Elements dalam defined boundary dipersepsikan sebagai grup.

   Aplikasi dalam desain:
   - Boxes dan containers untuk group content
   - Cards dalam UI design
   - Borders dan backgrounds
   - Highlighted sections

   Contoh praktis:
   - Card-based layouts (Pinterest, Trello)
   - Call-out boxes dalam articles
   - Pricing tables
   - Form sections

### E. COLOR PERCEPTION

#### 1. Bagaimana Kita Melihat Warna

   - Cahaya dengan wavelength berbeda dipersepsikan sebagai warna berbeda
   - Visible spectrum: ~380-700 nanometers
   - 3 jenis cones (S, M, L) respond to different wavelengths
   - Otak combines signals dari 3 cone types untuk perceive millions of
     colors

   Color perception is relative:
   - Simultaneous contrast: background affects perceived color
   - Successive contrast: after-images (stare at red → see cyan)
   - Color constancy: we perceive object colors as constant despite
     lighting changes

#### 2. Color Vision Deficiencies

   - Deuteranomaly (most common): reduced sensitivity to green (~5% males)
   - Protanomaly: reduced sensitivity to red (~1% males)
   - Tritanomaly: reduced sensitivity to blue (rare)
   - Complete color blindness (achromatopsia): extremely rare

   Design considerations:
   - Don't rely solely on color untuk critical information
   - Use color + shape/icon/text
   - Test designs dengan color blindness simulators
   - Sufficient contrast ratios (WCAG guidelines: 4.5:1 minimum)

#### 3. Cultural Aspects of Color Perception

   Warna memiliki associations berbeda di budaya berbeda:

   - Red:
     * Western: love, danger, excitement
     * China: luck, celebration, prosperity
     * Middle East: danger, caution

   - White:
     * Western: purity, cleanliness, innocence
     * Eastern: mourning, death

   - Yellow:
     * Western: optimism, caution
     * Egypt: mourning
     * Japan: courage

   Implikasi: Always consider cultural context dari target audience

- Black and white only (atau max 2 colors)
- Save source files (.xcf or .svg)
- Export as PNG

   - Bagian B: 4 compositions Anda dengan brief explanation each

## RINGKASAN

Pertemuan kedua ini memberikan fondasi scientific understanding tentang
bagaimana komunikasi visual bekerja dan bagaimana manusia memproses
informasi visual:

#### 1. Komunikasi Visual

   - Proses penyampaian pesan melalui elemen visual
   - Model komunikasi: sender → encoding → message → channel → decoding
     → receiver
   - Fungsi: informative, persuasive, educational, identificatory,
     decorative
   - Efektivitas dipengaruhi context, culture, dan experience

#### 2. Human Visual System

   - Anatomi mata: cornea, iris/pupil, lens, retina (rods & cones), fovea
   - Rods: low-light vision, peripheral, motion-sensitive
   - Cones: color vision, detail, concentrated di fovea
   - Proses: cahaya → photoreceptors → optic nerve → brain processing
   - Kita "melihat" dengan otak, bukan hanya mata

#### 3. Visual Perception

   - Sensation (physiological) vs Perception (psychological)
   - Dipengaruhi: past experience, attention, cultural background, context
   - Optical illusions demonstrate perception is constructed
   - Understanding "quirks" membantu design lebih efektif

#### 4. Gestalt Principles

   - Proximity: nearby objects grouped together
   - Similarity: similar objects grouped together
   - Continuity: we follow smooth paths
   - Closure: we complete incomplete shapes
   - Figure-Ground: we separate objects from backgrounds
   - Common Fate: objects moving together grouped together
   - Symmetry: symmetrical elements create order
   - Common Region: elements dalam boundary grouped together

   Applications: grouping content, visual hierarchy, negative space, flow

#### 5. Color Perception

   - 3 cone types (S, M, L) untuk perceive millions of colors
   - Color perception is relative dan context-dependent
   - Color blindness considerations penting untuk accessibility
   - Cultural meanings of colors influence interpretation

Pemahaman tentang human visual system dan perception principles adalah
foundation untuk menjadi desainer yang efektif. Dengan memahami bagaimana
audiens melihat dan memproses visual, Anda dapat design komunikasi yang
lebih impactful dan efektif.

## PENGANTAR MATERI

Di era yang dipenuhi informasi visual - dari social media feeds hingga
billboard di jalan, dari infografis hingga memes - kemampuan untuk "membaca"
dan "menulis" secara visual bukan lagi optional skill, tetapi kebutuhan
fundamental. Inilah yang disebut visual literacy: kemampuan untuk
menginterpretasikan, negotiate meaning, dan create dengan visual.

Visual literacy goes beyond sekedar mengerti "apa" yang ditampilkan dalam
gambar. Ia melibatkan pemahaman mendalam tentang "bagaimana" visual bekerja
untuk menyampaikan makna, "mengapa" visual tertentu dipilih dibanding yang
lain, dan "untuk siapa" pesan visual tersebut ditujukan. Seorang desainer
yang visual literate tidak hanya membuat desain yang "cantik", tetapi
desain yang "berbicara" dengan jelas dan efektif kepada audiensnya.

Visual rhetoric membawa visual literacy satu langkah lebih jauh dengan
fokus pada aspek persuasif dari komunikasi visual. Sejak zaman ancient
Greece, rhetoric telah dipelajari sebagai seni persuasi melalui bahasa.
Dalam konteks modern, kita menyadari bahwa visual memiliki rhetorical power
yang sama kuatnya - bahkan mungkin lebih kuat - dari bahasa verbal. Sebuah
foto dapat memicu emosi instant, sebuah logo dapat membangun trust, sebuah
infografis dapat mengubah opini.

Dalam pertemuan ini, kita akan explore bagaimana visual "berargumen",
bagaimana desain dapat membujuk, dan bagaimana Anda sebagai desainer dapat
craft visual messages yang tidak hanya inform, tetapi juga persuade dan
inspire action. Kita akan menganalisis iklan, poster, kampanye sosial, dan
berbagai bentuk komunikasi visual lainnya untuk understand strategies yang
membuat mereka efektif - atau tidak efektif.

Pertemuan ini juga akan memberikan introduction ke semiotics, ilmu tentang
signs dan symbols, yang merupakan foundation untuk understanding bagaimana
makna dikonstruksi dan dikomunikasikan melalui visual. Dengan toolkit
analytical ini, Anda akan dapat deconstruct setiap visual yang Anda lihat
dan understand bagaimana ia bekerja on multiple levels.

## MATERI

### A. VISUAL LITERACY

#### 1. Definisi Visual Literacy

   Visual literacy adalah kemampuan untuk:
   - Interpret (membaca dan memahami) makna dalam visual
   - Evaluate (menilai) kualitas dan credibility visual
   - Create (memproduksi) komunikasi visual yang efektif
   - Use (menggunakan) visual untuk thinking dan learning

   Komponen visual literacy:

**a. Visual Reading**

      - Kemampuan decode visual messages
      - Mengidentifikasi elemen dan prinsip desain
      - Understanding conventions dan symbols
      - Recognize context dan purpose

**b. Visual Writing**

      - Kemampuan encode ideas ke dalam visual
      - Memilih appropriate visual strategies
      - Compose effective visual messages
      - Consider audience dan context

**c. Critical Visual Thinking**

      - Analyze visual arguments
      - Evaluate visual evidence
      - Detect bias dan manipulation
      - Compare competing visual messages

#### 2. Mengapa Visual Literacy Penting?

**a. Information Overload**

      - Kita terekspos 5000+ visual messages per hari
      - Visual memory lebih kuat dari verbal memory
      - 90% informasi ke otak adalah visual

**b. Digital Age Communication**

      - Social media dominated oleh visual content
      - Instagram, TikTok, Pinterest - visual platforms
      - Emoji dan memes sebagai new visual language
      - Visual communication crosses language barriers

**c. Misinformation era**

      - Fake news often uses misleading visuals
      - Deep fakes dan photo manipulation
      - Critical visual literacy melindungi dari deception
      - Kemampuan fact-check visual crucial

**d. Professional Necessity**

      - Hampir semua profesi melibatkan visual communication
      - Presentations, reports, websites, social media
      - Visual literacy = competitive advantage
      - Career opportunities di visual industries expanding

3. "Grammar" of Visual Language

   Seperti bahasa verbal memiliki grammar, visual juga memiliki "rules"
   yang govern how meaning is made:

**a. Visual Vocabulary (Elemen)**

      - Titik/Dot: focus, emphasis, location
      - Garis/Line: connection, division, movement, direction
      - Shape: basic forms, recognition
      - Form: 3D shapes, volume, mass
      - Space: positive/negative, depth
      - Texture: surface quality, tactile sensation
      - Color: emotion, symbolism, organization
      - Typography: tone of voice, personality

**b. Visual Syntax (Prinsip Arrangement)**

      - Balance: visual weight distribution
      - Contrast: difference creates interest
      - Emphasis: focal point, hierarchy
      - Unity: coherence, consistency
      - Movement: visual flow, eye path
      - Rhythm: repetition creates pattern
      - Proportion: size relationships

**c. Visual Context**

      - Cultural context: symbols dan colors berbeda meaning di
        cultures berbeda
      - Historical context: visual references ke past events atau styles
      - Social context: who is communicating to whom, power dynamics
      - Medium context: print vs digital, static vs motion

#### 4. Reading Visual Messages: Levels of Meaning

**a. Denotative Level (What you see)**

      - Literal, descriptive meaning
      - Factual inventory of what's present
      - Objective observation

      Contoh: "An apple" = round, red fruit

**b. Connotative Level (What it suggests)**

      - Associated meanings, symbolic interpretations
      - Cultural dan personal associations
      - Emotional responses

      Contoh: Apple → health, temptation, technology (Apple Inc.),
      teacher's gift

**c. Ideological Level (What beliefs it promotes)**

      - Underlying values, assumptions, worldviews
      - Power structures, social norms
      - Often implicit, requires critical analysis

      Contoh: Apple in context of "an apple a day" ad → beliefs about
      health, nature, wellness lifestyle

#### 5. Framework untuk Visual Analysis

   Ketika menganalisis sebuah visual, consider:

   CONTENT:
   - Apa yang literally shown?
   - Siapa atau apa subjects?
   - Objects, settings, colors, text present?

   CONTEXT:
   - Di mana visual ini appear?
   - Kapan dibuat dan distributed?
   - Siapa creator/sponsor?
   - Siapa intended audience?
   - Apa cultural/historical context?

   COMPOSITION:
   - Bagaimana elements arranged?
   - Apa focal point?
   - How is eye guided through image?
   - What is visual hierarchy?
   - Color scheme dan lighting?

   CODES & CONVENTIONS:
   - Symbols atau icons used?
   - Genre conventions (advertising, editorial, etc.)?
   - Cultural codes (dress, gestures, settings)?

   CONNOTATIONS:
   - What are associated meanings?
   - What emotions evoked?
   - What values promoted?

   RHETORIC:
   - What is argument atau message?
   - How does it persuade?
   - What strategies used?

   IDEOLOGY:
   - What worldview presented?
   - Whose interests served?
   - What assumptions made?
   - What is included/excluded?

### B. VISUAL RHETORIC

#### 1. Apa itu Visual Rhetoric?

   Visual rhetoric adalah study tentang bagaimana visual persuade, argue,
   dan influence audiences. Ia menggabungkan:
   - Rhetorical theory (dari classical rhetoric)
   - Visual communication theory
   - Semiotics (study of signs)

   Key premises:
   - Visuals are NOT neutral atau objective
   - Visuals make arguments dan advance positions
   - Visual choices are rhetorical choices
   - Understanding visual rhetoric membantu both analysis dan creation

#### 2. Classical Rhetoric Applied to Visual

   Aristotle mengidentifikasi 3 modes of persuasion (rhetorical appeals):

**a. ETHOS (Credibility/Character)**

      Definisi: Appeal based on credibility, authority, trustworthiness
      dari speaker/brand

      Visual strategies untuk build ethos:

      - Professional design quality
        * Clean, polished layouts
        * Consistent branding
        * High-quality imagery

      - Authoritative imagery
        * Lab coats, diplomas (untuk healthcare/education)
        * Professional settings
        * Expert testimonials dengan photos

      - Brand symbols
        * Established logos
        * Award badges, certifications
        * "As seen in" media logos

      - Design conventions
        * Adhering to industry standards
        * Expected layouts untuk specific contexts

      Contoh ethos dalam visual:
      - Medical ads showing doctors di hospitals
      - Financial services usando corporate blue colors
      - Academic publications dengan formal typography
      - Sustainability brands dengan natural, earthy visual tones

      Red flags (weak ethos):
      - Poor design quality, typography errors
      - Stock photos yang obviously generic
      - Inconsistent branding
      - Lack of contact info atau transparency

**b. PATHOS (Emotion)**

      Definisi: Appeal to audience's emotions, values, beliefs

      Visual strategies untuk evoke pathos:

      - Emotional imagery
        * Faces showing emotions (joy, sadness, anger, fear)
        * Human interest stories through photos
        * Before/after transformations
        * Cute animals, babies (universal emotional triggers)

      - Color psychology
        * Red: excitement, urgency, passion
        * Blue: trust, calm, stability
        * Yellow: optimism, energy
        * Black: luxury, sophistication
        * Warm colors: intimacy, comfort
        * Cool colors: professionalism, distance

      - Symbolic imagery
        * Cultural symbols (flag, heart, dove)
        * Religious imagery
        * Nature imagery (sunrise, storm)

      - Visual metaphors
        * Journey, growth, barriers, bridges
        * Light vs darkness
        * Up vs down, big vs small

      - Composition techniques
        * Low angle = power, dominance
        * High angle = vulnerability, weakness
        * Close-up = intimacy, connection
        * Wide shot = isolation, context
        * Lighting = mood (dark/light, harsh/soft)

      Contoh pathos dalam visual:
      - Charity campaigns showing vulnerable children atau animals
      - Insurance ads depicting family safety dan security
      - Motivational posters dengan inspiring imagery
      - Political campaigns dengan hopeful atau fearful imagery

      Ethical considerations:
      - Manipulation vs legitimate emotional appeal
      - Exploitation of vulnerable subjects
      - Fear-mongering tactics
      - Balancing emotion dengan facts

**c. LOGOS (Logic/Reason)**

      Definisi: Appeal to reason, logic, facts, evidence

      Visual strategies untuk convey logos:

      - Data visualization
        * Charts, graphs, infographics
        * Statistics presented visually
        * Comparison tables
        * Timelines dan flowcharts

      - Demonstrative images
        * Before/after comparisons
        * Step-by-step illustrations
        * Diagrams showing how things work
        * Product features highlighted

      - Textual evidence
        * Quotes dari experts
        * Research findings
        * Testimonials dengan specifics
        * Numbered lists dan bullet points

      - Clear visual hierarchy
        * Logical flow of information
        * Prioritization of key points
        * Organized layout

      - Minimal, clean design
        * Reducing emotional manipulation
        * Focusing on facts
        * Professional, sober aesthetic

      Contoh logos dalam visual:
      - Technical manuals dengan detailed diagrams
      - Scientific posters dengan data visualization
      - Product comparison charts
      - Educational infographics

      Balance dengan other appeals:
      - Pure logos can be dry dan unengaging
      - Combine dengan ethos (credible source) dan pathos (why it matters)
      - Make data visual menarik tanpa misleading

#### 3. Visual Argumentation

   Visual arguments berbeda dari verbal arguments:

   Verbal argument:
   - Linear, sequential
   - Explicit claims dan evidence
   - Logical connectors stated

   Visual argument:
   - Simultaneous, spatial
   - Implicit claims (often)
   - Viewer constructs connections
   - Multiple interpretations possible

   Types of visual arguments:

**a. Categorical Claims**

      "X is a member of category Y"
      Visual shows relationship atau classification

      Contoh: Ad showing product among luxury items → "This is a luxury
      product"

**b. Causal Claims**

      "X causes Y"
      Visual shows cause-effect relationship

      Contoh: Smoking ad showing cigarettes → damaged lungs

**c. Evaluative Claims**

      "X is good/bad"
      Visual presents value judgment

      Contoh: Environmental poster showing pristine nature vs pollution

**d. Proposal Claims**

      "We should do X"
      Visual advocates for action atau policy

      Contoh: Public health poster showing vaccination benefiting community

#### 4. Visual Rhetorical Strategies

**a. Juxtaposition**

      - Placing contrasting elements side by side
      - Creates meaning through comparison
      - Examples: before/after, us vs them, problem vs solution

**b. Framing**

      - What's included vs excluded
      - How subject is presented
      - Camera angle, distance, context
      - Literal frames (borders, windows) yang focus attention

**c. Metaphor dan Symbolism**

      - Visual representations of abstract concepts
      - Cultural symbols dengan shared meanings
      - Metaphorical imagery (journey, war, growth, etc.)

**d. Identification**

      - Creating connection antara viewer dan subject
      - "People like you" imagery
      - Aspirational images (who you want to be)
      - First-person perspective shots

**e. Narrative**

      - Telling story through visual sequence
      - Single image suggesting larger story
      - Character development through imagery

**f. Repetition dan Pattern**

      - Reinforcing message through repeated visuals
      - Creating campaign consistency
      - Pattern breaking untuk emphasis

**g. Scale dan Proportion**

      - Size indicates importance
      - Exaggeration untuk effect
      - Microdetail vs grand overview

**h. Absence dan Silence**

      - What's NOT shown
      - Empty space speaking volumes
      - Minimalism as rhetorical choice

#### 5. Analyzing Rhetorical Visual: Case Study Approach

   Framework untuk analyze visual rhetoric:

   STEP 1: Describe
   - What do you see? (objective description)
   - Inventory elemen-elemen visual

   STEP 2: Contextualize
   - Where/when was this created dan circulated?
   - Who is creator/sponsor?
   - What is medium dan venue?
   - Who is intended audience?

   STEP 3: Identify Rhetorical Situation
   - What is problem atau issue addressed?
   - What is exigence (urgency, timing)?
   - What are constraints (limitations)?

   STEP 4: Analyze Rhetorical Appeals
   - How does it establish ethos?
   - How does it evoke pathos?
   - How does it use logos?

   STEP 5: Identify Strategies
   - What rhetorical strategies employed?
   - How are visual elements working argumentatively?

   STEP 6: Evaluate Effectiveness
   - How effective is the visual untuk intended audience?
   - What are strengths dan weaknesses?
   - Ethical considerations?

   STEP 7: Compare dan Contrast
   - How does this compare to similar visuals?
   - What makes it distinctive?

### C. INTRODUCTION TO SEMIOTICS

#### 1. Apa itu Semiotics?

   Semiotics adalah study tentang signs dan how they create meaning.

   Founder: Ferdinand de Saussure (linguistics) dan Charles Sanders Peirce
   (philosophy)

   Core concept: Everything can be a "sign" - words, images, sounds,
   objects, behaviors - anything that carries meaning.

   Relevance untuk desain:
   - Designers create signs
   - Understanding semiotic processes membantu intentional meaning-making
   - Helps predict how audiences will interpret designs

#### 2. Basic Semiotic Concepts

**a. Sign**

      - Anything that stands for something else
      - Consists of:
        * SIGNIFIER: the form (image, word, sound)
        * SIGNIFIED: the concept atau meaning

      Example:
      - Red octagonal sign (signifier) → STOP (signified)
      - Heart shape (signifier) → love, affection (signified)

**b. Relationship bukan fixed:**

      - Same signifier dapat have different signifieds dalam konteks
        berbeda
      - Heart → love (Valentine) vs heart → vital organ (medical context)
      - Cultural dan contextual factors affect interpretation

**c. Arbitrariness vs Iconicity**

      Arbitrary signs:
      - No natural connection antara signifier dan signified
      - Relationship is conventional (agreed upon)
      - Example: Kata "tree" tidak seperti tree visual
      - Most language is arbitrary

      Iconic signs:
      - Resemblance antara signifier dan signified
      - Example: Photograph resembles subject
      - Warning signs dengan pictorial representation

#### 3. Peirce's Trichotomy of Signs

   Charles Peirce mengklasifikasi signs menjadi 3 types:

**a. ICON**

      - Resembles atau imitates what it represents
      - Visual similarity

      Examples:
      - Photograph
      - Realistic painting atau illustration
      - Map (spatially resembles territory)
      - Onomatopoeia dalam language (words that sound like what they mean)

      Dalam desain:
      - Icons di user interfaces (trash can icon)
      - Product packaging showing product inside
      - Pictorial road signs

**b. INDEX**

      - Direct connection atau causal relationship dengan what it
        represents
      - "Points to" atau indicates

      Examples:
      - Smoke → fire (smoke indicates fire exists)
      - Footprints → someone walked here
      - Thermometer reading → temperature
      - Red, watery eyes → crying atau allergies

      Dalam desain:
      - Progress bars (index of completion)
      - Loading spinners (index of processing)
      - Pointer cursor (index of clickable item)
      - "Before" photos di ads (index of past condition)

**c. SYMBOL**

      - Arbitrary, conventional relationship
      - Meaning is learned, cultural
      - No resemblance atau direct connection

      Examples:
      - Language (words)
      - National flags
      - Religious symbols (cross, crescent, Star of David)
      - Traffic light colors (red=stop, green=go)
      - Mathematical symbols

      Dalam desain:
      - Logos (Nike swoosh)
      - Color coding (blue untuk boys, pink untuk girls - cultural)
      - Typographic choices (Gothic font = medieval, horror)

   Note: Many signs are combinations:
   - Photograph is icon (resembles) DAN index (caused by light from
     subject)
   - Logo dapat be symbolic DAN iconic (Apple logo)

#### 4. Codes dan Conventions

   Codes: Systems of signs governed oleh agreed-upon rules

   Types of codes dalam visual communication:

**a. Technical Codes**

      - Camera techniques: angles, distance, focus
      - Lighting: high-key (bright, positive) vs low-key (dark, dramatic)
      - Editing: transitions, pace

**b. Symbolic Codes**

      - Objects, settings, body language, clothing
      - Colors, shapes, typography
      - Cultural specific meanings

**c. Written Codes**

      - Language, typography
      - Headlines, captions, slogans

   Understanding codes helps:
   - Decode existing visual messages
   - Encode new messages effectively
   - Subvert atau challenge conventions untuk creative effect

## MATERI

### A. VISUAL SYNTAX: THE GRAMMAR OF VISUAL DESIGN

#### 1. Apa itu Visual Syntax?

   Visual syntax adalah system of rules dan conventions yang govern
   bagaimana visual elements dikombinasikan untuk create meaningful
   compositions.

   Components dari visual syntax:

**a. Visual Grammar**

      - "Rules" tentang bagaimana elements berinteraksi
      - Compositional structures yang convey meaning
      - Spatial relationships dan their significance

**b. Reading Paths**

      - How viewers navigate through visual space
      - Cultural conventions (left-to-right, top-to-bottom di Western
        cultures)
      - Designer's intentional guidance

**c. Visual Coherence**

      - How elements "belong together"
      - Unity vs fragmentation
      - Consistency creating meaning

#### 2. Spatial Dimensions in Visual Syntax

   Kress & van Leeuwen (2006) mengidentifikasi beberapa spatial dimensions
   yang carry meaning:

**a. LEFT vs RIGHT**

      Western reading cultures:
      - LEFT = Given (known information, starting point)
      - RIGHT = New (new information, goal, destination)

      Applications:
      - Before/after: before on left, after on right
      - Problem/solution: problem left, solution right
      - Text layout: introduction left, details right

      Example: Product ads often show product on right (as "new"
      information atau goal)

**b. TOP vs BOTTOM**

      - TOP = Ideal (aspirational, promise, generalized claim)
      - BOTTOM = Real (practical, evidence, specific information)

      Applications:
      - Top: emotive imagery, aspirational message
      - Bottom: factual details, product specs, call-to-action

      Example: Magazine ads often have emotive image at top, product info
      at bottom

**c. CENTER vs MARGIN**

      - CENTER = Nucleus (most important, core message)
      - MARGIN = Supporting information, context

      Applications:
      - Centered logos for emphasis
      - Hero elements in center
      - Supporting details di periphery

      Example: Website hero sections center key message

**d. FOREGROUND vs BACKGROUND**

      - FOREGROUND = Salient, immediate, important
      - BACKGROUND = Context, atmosphere, supporting

      Applications:
      - Subject in foreground, context in background
      - Depth creating hierarchy
      - Figure-ground relationships

#### 3. Visual Modality (How "Real" Does It Look?)

   Modality refers to "reality value" atau "truth claim" dari
   representation.

   Factors affecting modality:

**a. Color Saturation**

      - High saturation = high modality (looks "real")
      - Low saturation = low modality (abstract, conceptual)
      - Black & white = variable (can be artistic atau documentary)

**b. Contextualization**

      - Detailed background = high modality
      - Abstract/minimal background = low modality

**c. Detail**

      - High detail = high modality
      - Simplified/stylized = low modality

**d. Depth**

      - Perspective dan depth cues = high modality
      - Flat, two-dimensional = low modality

   Implications:
   - Photorealistic rendering claims "this is how it is"
   - Abstraction claims "this is a concept atau idea"
   - Designers manipulate modality untuk rhetorical purposes

   Examples:
   - Scientific illustration: high modality untuk accuracy
   - Infographic icons: low modality untuk abstraction
   - Advertising photography: high modality untuk desirability

#### 4. Vectors dan Processes

   Visual elements dapat represent different types of processes:

**a. Narrative Vectors**

      - Lines, eyelines, pointing gestures create direction
      - Show action atau connection
      - "Actor" dan "goal" dalam visual narrative

      Example: Person looking at product creates vector connecting them

**b. Conceptual Structures**

      - Classification (this belongs to that category)
      - Analytical (part-whole relationships)
      - Symbolic (what something means)

      Example: Diagram showing components of system

#### 5. Interactive Meanings: Contact dan Distance

**a. Gaze (Eye Contact)**

      DEMAND:
      - Subject looks directly at viewer
      - Creates direct address, involvement
      - "Demands" response dari viewer
      - Common dalam advertising: "This is for YOU"

      OFFER:
      - Subject looks away (doesn't acknowledge viewer)
      - Viewer becomes observer
      - More detached, informative
      - Common dalam documentary atau reportage

**b. Social Distance (Framing)**

      CLOSE-UP:
      - Intimate distance
      - Personal, emotional connection
      - Focus on detail, expression

      MEDIUM SHOT:
      - Social distance
      - Comfortable, conversational
      - Shows person dan some context

      LONG SHOT:
      - Public distance
      - Impersonal, objective
      - Shows full context, environment

**c. Angle of Interaction**

      EYE LEVEL (HORIZONTAL):
      - Equality, peer relationship
      - Neutral, objective

      HIGH ANGLE (looking down):
      - Viewer has power
      - Subject vulnerable, weak, submissive
      - Used to diminish

      LOW ANGLE (looking up):
      - Subject has power
      - Imposing, dominant, authoritative
      - Used to amplify, intimidate, inspire awe

**d. Perspective (Frontal vs Oblique)**

      FRONTAL:
      - Involvement, inclusion
      - "You are part of this"

      OBLIQUE:
      - Detachment, observation
      - "You are outside looking in"

### B. SEMIOTICS IN DEPTH

#### 1. Saussure's Semiotic Theory

   Ferdinand de Saussure: founder of modern linguistics dan structuralist
   semiotics

   Core concepts:

**a. The Sign = Signifier + Signified**

      SIGNIFIER (Signifiant):
      - The form, the "sound-image" atau visual form
      - Material vehicle of meaning
      - What we perceive dengan senses

      SIGNIFIED (Signifié):
      - The concept, mental representation
      - Meaning dalam mind
      - What we understand

      Example:
      - Tree image (signifier) → concept of "tree" (signified)
      - Word "tree" (signifier) → concept of "tree" (signified)

**b. Arbitrariness of the Sign**

      - No natural/inevitable connection antara signifier dan signified
      - Relationship is conventional, established by social agreement
      - Why "tree" means tree? Because English speakers agree it does
      - Different languages use different signifiers untuk same signified

      Exceptions:
      - Onomatopoeia (words that sound like what they mean)
      - Photographic images (have indexical connection)

**c. Sign Gets Meaning from Difference**

      - Sign doesn't have meaning in isolation
      - Meaning comes from difference dari other signs dalam system
      - "Hot" means what it does because it's NOT "cold"
      - Red means what it does in contrast to green, blue, dll

      Implications untuk design:
      - Context matters crucially
      - Contrast creates meaning
      - Choices are always relative

#### 2. Peirce's Semiotic Theory (Review + Expansion)

   Charles Sanders Peirce: American philosopher, founder of pragmatist
   semiotics

**a. Triadic Model of Sign**

#### 1. REPRESENTAMEN (Sign vehicle)

         - The form taken by sign
         - What represents

#### 2. OBJECT

         - What sign refers to
         - What is represented

#### 3. INTERPRETANT

         - The sense made of sign
         - Mental effect atau understanding
         - NOT the interpreter, but the interpretation

      Key insight: Meaning tidak "dalam" sign atau object, tapi dalam
      interpretant - the process of meaning-making

**b. Icon, Index, Symbol (Detailed)**

      ICON (similarity):
      - Photographs
      - Realistic paintings
      - Diagrams, maps
      - Mimetic representations

      Design considerations:
      - Used when clarity dan immediate recognition needed
      - Can be culturally specific (some icons not universal)
      - Balance realism vs stylization

      INDEX (causal/physical connection):
      - Smoke indicating fire
      - Footprints indicating presence
      - Shadows indicating objects
      - Symptoms indicating disease

      Design considerations:
      - Powerful because of real-world connection
      - Can be used metonymically (part stands for whole)
      - Examples: flag representing country, crown representing royalty

      SYMBOL (conventional):
      - Words, numbers
      - National flags
      - Religious symbols
      - Brand logos
      - Colors (meanings are cultural)

      Design considerations:
      - Must be learned - not immediately obvious
      - Can be very powerful within culture
      - Risk of misinterpretation across cultures
      - Can be combined with iconic/indexical elements

**c. Unlimited Semiosis**

      - Interpretant of one sign becomes representamen of another
      - Meaning is never final, always generates more meaning
      - Chain of interpretations

      Example:
      - Photo of smoking cigarette (icon)
      - Signifies "smoking" (object)
      - Interpreted as "danger" (interpretant)
      - Which then becomes sign untuk "health risk" (new object)
      - Which is interpreted as "should quit" (new interpretant)
      - And so on...

#### 3. Denotation dan Connotation (Roland Barthes)

   Roland Barthes: French literary theorist dan semiologist

**a. DENOTATION (First Order Signification)**

      - Literal, obvious, commonsense meaning
      - Descriptive, definitional
      - What is represented
      - "Objective" level (though this is problematic)

      Example:
      - Image shows red rose
      - Denotation: "This is a red rose" (flower, plant, red color)

**b. CONNOTATION (Second Order Signification)**

      - Associated meanings, cultural dan emotional
      - Subjective, interpretive
      - What is suggested atau implied
      - Meanings layered pada denotation

      Example:
      - Red rose connotations:
        * Love, passion, romance
        * Valentine's Day
        * Beauty, fragility
        * English symbol (War of Roses)
        * Luxury (expensive flowers)

      Key insight: Connotations are where ideology works

**c. Barthes' Myth (Third Order Signification)**

      Myth is not "false story" but rather: system of second-order
      signification that makes cultural/ideological meanings seem natural,
      universal, inevitable

      How myth works:

      FIRST ORDER:
      Signifier (image) + Signified (concept) = Sign

      SECOND ORDER (Myth):
      Sign (from first order becomes signifier) + New Signified (ideology)
      = Myth

      Example: Advertisement Analysis

      FIRST ORDER:
      - Image of woman in kitchen (signifier)
      - Woman cooking (signified)
      - = Sign: "woman cooking"

      SECOND ORDER (Myth):
      - "Woman cooking" (now signifier)
      - "Women's natural role is domestic" (signified)
      - = Myth: naturalizes gender roles

      The myth makes constructed social arrangement seem "natural"

**d. Demythologizing**

      Designer's responsibility:
      - Be aware of myths being perpetuated
      - Question "natural" representations
      - Challenge or subvert problematic myths
      - Create counter-narratives

      Example: Advertising showing men in domestic roles challenges
      traditional gender myth

#### 4. Cultural Codes

   Codes are systems of meaning shared oleh members of culture atau group.

**a. Types of Cultural Codes:**

      BODILY CODES:
      - Gestures, postures, facial expressions
      - Personal space, touch
      - Can be very culturally specific

      Example: Thumbs up = positive (Western), offensive (Middle East)

      DRESS CODES:
      - Clothing as signifier of identity, status, role
      - Professional vs casual
      - Subculture markers

      Example: Business suit signifies professionalism, authority

      BEHAVIORAL CODES:
      - Manners, etiquette
      - What's appropriate dalam different contexts
      - Rituals dan ceremonies

      COLOR CODES:
      - Symbolic meanings of colors
      - Vary greatly across cultures

      Example: White = purity (West), mourning (East Asia)

      TECHNICAL CODES (Media-specific):
      - Camera angles, editing, lighting dalam photography/video
      - Typography conventions
      - Layout patterns

      TEXTUAL CODES:
      - Written language conventions
      - Typography carrying meaning
      - Text placement dan hierarchy

**b. Code Switching:**

      - Using different codes untuk different audiences atau contexts
      - Designers must be fluent dalam multiple codes
      - Code mixing dapat create new meanings

### C. APPLYING SEMIOTICS TO DESIGN

#### 1. Semiotic Design Strategy

   Step-by-step approach:

   STEP 1: Define Communication Goal
   - What do you want to communicate?
   - To whom?
   - For what purpose?

   STEP 2: Identify Signifieds (Concepts)
   - What concepts need to be conveyed?
   - List all meanings yang harus dikomunikasikan

   STEP 3: Select Signifiers
   - What visual forms akan represent concepts?
   - Icons, indexes, symbols?
   - Consider cultural codes relevant to audience

   STEP 4: Build Connotative Layers
   - What associated meanings desired?
   - How to evoke these through visual choices?

   STEP 5: Check for Unintended Significations
   - What else might this signify?
   - Potential misinterpretations?
   - Problematic myths reinforced?

   STEP 6: Test dan Refine
   - Does audience interpret as intended?
   - Adjust berdasarkan feedback

#### 2. Designing dengan Intentional Polysemy

   Polysemy: multiple meanings coexisting dalam single sign

   Can be strategic:
   - Engages viewer dalam meaning-making
   - Creates depth dan interest
   - Appeals to multiple audiences simultaneously

   But also risky:
   - Message dapat become unclear
   - Unintended meanings dapat dominate

   Balance needed:
   - Core meaning should be clear
   - Secondary meanings add richness
   - Avoid confusion

#### 3. Case Study: Analyzing Logo Semiotics

   Take any well-known logo dan analyze:

   Example: Apple Inc. Logo

   DENOTATION:
   - Bitten apple shape
   - Silver/gray color (atau rainbow dalam old version)
   - Clean, simple silhouette

   CONNOTATIONS (Multiple layers):
   - Apple fruit: knowledge, education, health
   - Bite: accessibility, user-friendly (you can "take a bite")
   - Historical: Adam & Eve, forbidden fruit (taking knowledge)
   - Clean design: simplicity, elegance, modern
   - Silver: premium, high-tech, sleek

   INDEXICAL:
   - Tech company (through established association)

   SYMBOLIC:
   - Brand values: innovation, thinking different

   CULTURAL CODES:
   - Minimalist design = sophisticated (design culture code)
   - Apple as fruit of knowledge (Western cultural narrative)

   MYTH:
   - Apple users are creative, innovative, sophisticated
   - Owning Apple = being part of elite, forward-thinking group
   - Technology should be beautiful dan intuitive

#### 4. Ethical Considerations dalam Semiotic Design

**a. Stereotypes**

      - Visual shorthand often relies on stereotypes
      - Can perpetuate harmful representations
      - Designer responsibility: challenge rather than reinforce

**b. Cultural Appropriation**

      - Using symbols dari cultures you're not part of
      - Risk of misuse, offense, commodification
      - Need for research, respect, proper context

**c. Manipulation**

      - Semiotic knowledge dapat digunakan untuk manipulate
      - Ethical line antara persuasion dan deception
      - Transparency vs hidden persuasion

**d. Representation dan Inclusion**

      - Who is visible dalam visual?
      - Whose perspective privileged?
      - Are diverse identities represented fairly?

## MATERI

### A. PSYCHOLOGY OF ATTENTION

#### 1. Apa itu Attention?

   Attention adalah cognitive process dari selectively concentrating pada
   satu aspect dari environment while ignoring others.

   Karakteristik attention:
   - LIMITED resource: tidak bisa attend to everything simultaneously
   - SELECTIVE: we choose (consciously atau unconsciously) what to
     attend to
   - DIRECTED: can be guided internally atau externally
   - DYNAMIC: shifts frequently

   Dalam konteks visual design:
   - Designers compete untuk limited attention
   - Goal: capture attention quickly
   - Challenge: hold attention long enough untuk communicate message
   - Strategy: make it easy untuk audience to process information

#### 2. Types of Attention

**a. BOTTOM-UP ATTENTION (Stimulus-Driven)**

      Definisi:
      - Automatic, involuntary attention
      - Triggered oleh stimulus characteristics
      - "Captured" oleh salient features
      - Pre-attentive processing

      Triggers:
      - Sudden onset (something appears)
      - Movement
      - High contrast
      - Bright colors
      - Large size
      - Novelty, unexpectedness
      - Personal relevance (your name, face)

      Examples dalam daily life:
      - Flashing lights automatically grab attention
      - Loud noise makes you look
      - Moving object di peripheral vision catches eye

      Design applications:
      - Pop-ups dan notifications (though can be annoying)
      - Animated elements
      - High-contrast buttons
      - Large headlines
      - Bright colors untuk urgent information

      Advantages:
      - Fast, automatic
      - Effective untuk capturing initial attention
      - Works across audiences

      Disadvantages:
      - Can be distracting
      - If overused, becomes noise
      - May not result dalam sustained attention

**b. TOP-DOWN ATTENTION (Goal-Driven)**

      Definisi:
      - Voluntary, controlled attention
      - Driven oleh goals, expectations, knowledge
      - "Searching" untuk specific information
      - Attentive processing

      Triggers:
      - Task requirements
      - Personal goals
      - Prior knowledge
      - Expectations
      - Instructions

      Examples dalam daily life:
      - Looking untuk specific product di supermarket
      - Searching untuk friend di crowd
      - Reading specific section dalam newspaper

      Design applications:
      - Clear labeling dan navigation
      - Consistent design patterns (users know where to look)
      - Search functionality
      - Table of contents, menus
      - Meeting user expectations

      Advantages:
      - Sustained, focused attention
      - Efficient information seeking
      - User is motivated

      Disadvantages:
      - Requires user effort
      - User must know what they're looking for
      - Slow if information hard to find

**c. DESIGNING FOR BOTH:**

      Effective design balances both types:
      - Bottom-up untuk initial capture
      - Top-down untuk sustained engagement dan information finding

      Example: Website homepage
      - Large hero image (bottom-up: captures attention)
      - Clear navigation menu (top-down: helps users find what they seek)
      - Consistent layout (top-down: meets expectations)
      - Call-to-action button dalam bright color (bottom-up: draws eye)

#### 3. Attention Span dan Implications

   Research findings:
   - Average attention span: ~8 seconds (decreased dari 12 seconds dalam
     2000)
   - 55% of page visitors spend less than 15 seconds on webpage
   - First few seconds critical untuk engagement
   - Mobile screens further reduce attention span

   Implications untuk design:

**a. IMMEDIATE IMPACT:**

      - First impression happens dalam 50 milliseconds
      - Design must communicate quickly
      - Clarity over complexity
      - Strong focal point immediately visible

**b. PROGRESSIVE DISCLOSURE:**

      - Don't overwhelm dengan all information at once
      - Reveal information gradually
      - Use visual hierarchy untuk prioritize
      - Make it easy to scan quickly

**c. MICRO-MOMENTS:**

      - Design untuk brief interactions
      - Each element should have clear purpose
      - Reduce cognitive load
      - Streamline user journey

**d. MOBILE-FIRST THINKING:**

      - Small screens demand even clearer hierarchies
      - Thumb-friendly interaction zones
      - Reduce scrolling untuk key information

### B. VISUAL HIERARCHY

#### 1. Definisi Visual Hierarchy

   Visual hierarchy adalah arrangement of elements dalam way that implies
   importance. It guides viewer's eye dalam order of importance, creating
   clear path through information.

   Purpose:
   - Direct attention efficiently
   - Communicate importance
   - Improve comprehension
   - Enhance user experience
   - Speed up information processing

#### 2. Principles untuk Creating Visual Hierarchy

**a. SIZE dan SCALE**

      Principle: Larger elements attract more attention dan appear more
      important.

      Applications:
      - Headlines larger than body text
      - Primary CTA button larger than secondary
      - Hero images dominate layout
      - Important products shown larger

      Guidelines:
      - Create clear size differences (subtle differences confusing)
      - 3-level hierarchy minimum: primary, secondary, tertiary
      - Proportional scaling: follow mathematical ratios (1.5x, 2x, 3x)

      Typography scale example:
      - H1: 48pt
      - H2: 32pt
      - H3: 24pt
      - Body: 16pt
      - Caption: 12pt

**b. COLOR dan CONTRAST**

      Principle: High contrast dan saturated colors capture attention;
      similar colors recede.

      Applications:
      - Brightly colored CTAs stand out
      - Dark text pada light background (atau vice versa)
      - Accent color untuk important elements
      - Grayscale untuk less important information

      Guidelines:
      - Use color sparingly untuk emphasis
      - Maintain sufficient contrast untuk readability (WCAG: 4.5:1
        minimum)
      - Warm colors advance, cool colors recede
      - Saturated colors grab attention more than muted ones

      Color hierarchy strategy:
      - Primary action: brand color, saturated
      - Secondary action: neutral, less saturated
      - Tertiary: gray, subtle

**c. POSITION dan PLACEMENT**

      Principle: Certain positions naturally receive more attention.

      Attention zones (typical):
      - TOP > Bottom
      - LEFT > Right (dalam Western cultures)
      - CENTER > Edges
      - FOREGROUND > Background

      Applications:
      - Logo di top-left
      - Primary headline at top
      - Main CTA "above the fold" (visible without scrolling)
      - Most important content di center atau top

      Guidelines:
      - Place most important elements dalam prime positions
      - Use conventions (users have learned expectations)
      - Consider reading patterns (Z, F)

**d. TYPOGRAPHY**

      Principle: Typography variations create hierarchy through weight,
      style, size, case.

      Tools:
      - Weight: Bold > Regular > Light
      - Style: Italic untuk emphasis
      - Case: ALL CAPS untuk short, urgent text (but less readable untuk
        long text)
      - Font family: Display fonts untuk headlines, readable fonts untuk
        body

      Applications:
      - Bold headlines
      - Regular body text
      - Light captions
      - Italic untuk quotes atau emphasis

      Guidelines:
      - Limit font families (2-3 max)
      - Maintain consistent styling untuk same hierarchy levels
      - Ensure readability at all sizes

**e. SPACING dan WHITE SPACE**

      Principle: Elements dengan more space around them receive more
      attention; grouped elements seen as related.

      Applications:
      - Generous margins around key elements
      - White space sets off important content
      - Tight spacing groups related items
      - Breathing room improves focus

      Guidelines:
      - Use space lavishly untuk important elements
      - Consistent spacing within categories
      - More space between categories than within
      - Don't fear empty space

**f. REPETITION dan PATTERN**

      Principle: Repeated elements create pattern; breaking pattern
      attracts attention.

      Applications:
      - Consistent button styling
      - Regular grid layouts
      - Pattern of similar images
      - Breaking pattern untuk emphasis (one different element)

      Guidelines:
      - Establish pattern first
      - Break pattern intentionally, not accidentally
      - Use sparingly untuk maximum effect

**g. ALIGNMENT**

      Principle: Aligned elements perceived as organized dan connected;
      misalignment draws attention.

      Applications:
      - Grid-based layouts
      - Flush left atau right text blocks
      - Aligned columns
      - Centered titles

      Guidelines:
      - Choose alignment strategy dan stick to it
      - Use alignment dalam service of hierarchy
      - Intentional misalignment dapat create dynamic interest

**h. DEPTH dan LAYERING**

      Principle: Foreground elements appear more important than background.

      Applications:
      - Drop shadows create depth
      - Overlapping elements show priority
      - Blur background, focus foreground
      - Depth cues (perspective, size diminishment)

      Guidelines:
      - Use subtly untuk effect
      - Ensure readability maintained
      - Consistent lighting direction dalam shadows

#### 3. Evaluating Visual Hierarchy

   Good hierarchy characteristics:
   - Clear primary focal point
   - Logical progression through content
   - Easy to scan dan find information
   - Importance immediately apparent
   - Supports content, doesn't distract

   Testing hierarchy:

**a. SQUINT TEST:**

      - Squint at design (blur vision)
      - What stands out?
      - Should be most important elements
      - If wrong things stand out, hierarchy fails

**b. 5-SECOND TEST:**

      - Show design untuk 5 seconds
      - Ask: what did you see? What was most prominent?
      - Feedback reveals what actually captures attention

**c. GRAYSCALE TEST:**

      - Remove all color
      - Does hierarchy still work?
      - Should rely on more than color alone

**d. INFORMATION ARCHITECTURE TEST:**

      - List content elements dalam order of importance
      - Check if visual presentation matches
      - Adjust if mismatch

### C. CONTRAST PRINCIPLES

#### 1. Types of Contrast

**a. Size Contrast**

      - Large vs small
      - Creates immediate hierarchy
      - Most straightforward form

**b. Color Contrast**

      - Hue contrast (different colors)
      - Value contrast (light vs dark)
      - Saturation contrast (vivid vs muted)
      - Temperature contrast (warm vs cool)

**c. Shape Contrast**

      - Geometric vs organic
      - Simple vs complex
      - Regular vs irregular

**d. Texture Contrast**

      - Smooth vs rough
      - Plain vs patterned
      - Minimal vs detailed

**e. Direction Contrast**

      - Horizontal vs vertical
      - Diagonal amongst horizontal
      - Static vs dynamic

**f. Density Contrast**

      - Dense vs sparse
      - Busy vs calm
      - Complex vs simple

#### 2. Using Contrast Effectively

   Principles:
   - Contrast must be STRONG enough to be noticed
   - Too much contrast everywhere = no hierarchy
   - Strategic contrast guides attention
   - Contrast creates visual interest

   Guidelines:
   - Contrast important against unimportant
   - Use multiple types of contrast together
   - Maintain consistency dalam categories
   - Test accessibility (color contrast ratios)

### D. EYE-TRACKING PATTERNS

#### 1. Common Reading Patterns

**a. Z-PATTERN**

      Description:
      - Eyes start top-left
      - Scan horizontally to top-right
      - Diagonal down to bottom-left
      - Scan horizontally to bottom-right
      - Forms "Z" shape

      When it occurs:
      - Pages dengan minimal text
      - Layouts tanpa clear focal points
      - Homogeneous content distribution
      - Landing pages

      Design applications:
      - Place logo top-left
      - Important info/CTA top-right
      - Supporting info mid-left
      - Secondary CTA bottom-right

      Example contexts:
      - Billboards
      - Magazine ads
      - Simple landing pages
      - Presentations slides

**b. F-PATTERN**

      Description:
      - Eyes start top-left
      - Horizontal scan across top (forms top of "F")
      - Vertical scan down left side
      - Shorter horizontal scan (forms middle of "F")
      - Continue down left side
      - Forms "F" shape

      When it occurs:
      - Text-heavy pages
      - Lists dan menus
      - Search results
      - Blog posts, articles

      Design applications:
      - Most important info dalam first two paragraphs
      - Front-load headings with key info
      - Left-align important content
      - Summaries atau highlights in opening

      Example contexts:
      - Websites dengan much content
      - News sites
      - E-commerce product listings
      - Search engine results

**c. GUTENBERG DIAGRAM (Z-PATTERN variant)**

      Divides layout into 4 quadrants:
      - PRIMARY OPTICAL AREA: Top-left (most attention)
      - STRONG FALLOW AREA: Top-right
      - WEAK FALLOW AREA: Bottom-left
      - TERMINAL AREA: Bottom-right (last stop, good untuk CTA)

      Applications:
      - Quadrant-based layouts
      - Print design (newspapers)
      - Traditional websites

#### 2. Designing untuk Eye-tracking Patterns

   Strategy:
   - Understand likely patterns untuk your content type
   - Place important elements dalam high-attention zones
   - Use visual cues to guide eyes (arrows, lines, gaze direction)
   - Create intentional path through design
   - Test dengan users if possible

   Breaking patterns:
   - Strong visual can interrupt pattern
   - Use strategically untuk important elements
   - But don't fight natural tendencies too much

### E. CALL-TO-ACTION (CTA) DESIGN

#### 1. Anatomy of Effective CTA

   Components:
   - CLEAR ACTION VERB: "Buy", "Download", "Subscribe", "Learn More"
   - VALUE PROPOSITION: What user gets
   - URGENCY/INCENTIVE (optional): "Limited time", "Free", "Save 50%"
   - VISUAL PROMINENCE: Button stands out

   Design principles:

**a. Visual Contrast**

      - CTA berbeda dari surrounding elements
      - Often brightly colored
      - Button style vs text link
      - Sufficient white space around it

**b. Size**

      - Large enough to be noticed
      - But not overwhelming
      - Proportional to importance

**c. Position**

      - Natural endpoint dalam visual flow
      - Often bottom-right (terminal area)
      - "Above the fold" untuk primary CTA
      - Repeated if long page

**d. Shape**

      - Recognizable as button
      - Rounded corners often more inviting
      - Adequate padding (clickable area)

**e. Text**

      - Short, action-oriented
      - Benefit-focused when possible
      - First-person "My" atau second-person "Your" effective
      - Avoid generic "Submit" atau "Click here"

#### 2. CTA Best Practices

   - ONE primary CTA per page/section
   - Secondary CTAs less prominent
   - Consistent CTA styling across site/campaign
   - Test colors (A/B testing)
   - Mobile-friendly size (min 44x44px touch target)
   - Loading/success states considered

### F. ATTENTION OPTIMIZATION STRATEGIES

#### 1. Reduce Cognitive Load

   - Simplify layouts
   - Limit choices (paradox of choice)
   - Clear, concise copy
   - Progressive disclosure
   - Consistent patterns
   - Familiar UI elements

#### 2. Create Clear Focus

   - One main message per design
   - Eliminate unnecessary elements
   - Direct gaze dengan visual cues
   - Strong focal point

#### 3. Optimize untuk Scanning

   - Break up long text
   - Use headings dan subheadings
   - Bullet points untuk lists
   - Short paragraphs (3-4 lines online)
   - Highlight key information
   - Use spacing generously

#### 4. Make Text Readable

   - Appropriate font sizes (min 16px untuk body online)
   - High contrast
   - Adequate line spacing (1.5x font size)
   - Optimal line length (50-75 characters)
   - Left alignment untuk most text
   - No all-caps for long text

#### 5. Guide with Visual Cues

   - Arrows dan directional shapes
   - Lines connecting related elements
   - Gaze direction (people looking at product/CTA)
   - Color coding
   - Numbered steps

## RINGKASAN

Pertemuan kelima ini focused pada bagaimana capture dan direct attention
effectively:

#### 1. Psychology of Attention

   - Attention is limited, selective, dan dynamic resource
   - Bottom-up (stimulus-driven): automatic capture via contrast, motion,
     novelty
   - Top-down (goal-driven): voluntary focus based on task dan expectations
   - Average attention span: ~8 seconds - design must work fast
   - Design untuk both types untuk complete experience

#### 2. Visual Hierarchy

   - Arrangement implying importance
   - Techniques:
     * Size/scale: larger = more important
     * Color/contrast: high contrast = attention
     * Position: top/center/left prioritized
     * Typography: weight, size, style differences
     * Spacing: more space = more importance
     * Repetition + breaking pattern
     * Alignment: organized perception
     * Depth: foreground dominates
   - Testing: squint test, 5-second test, grayscale test

#### 3. Contrast Principles

   - Multiple types: size, color, shape, texture, direction, density
   - Must be strong enough to notice
   - Strategic use guides attention
   - Contrast important against unimportant

#### 4. Eye-tracking Patterns

   - Z-pattern: minimal text, landing pages (top-left → top-right →
     bottom-left → bottom-right)
   - F-pattern: text-heavy pages (top horizontal scan → left vertical →
     shorter horizontal)
   - Gutenberg diagram: quadrant-based understanding
   - Design placement considering natural eye movements

#### 5. Call-to-Action Design

   - Clear action verb + value proposition
   - Visual prominence: contrast, size, position, shape
   - One primary CTA per page/section
   - Best practices: benefit-focused copy, adequate touch targets,
     consistent styling

#### 6. Attention Optimization

   - Reduce cognitive load
   - Create clear focus
   - Optimize untuk scanning
   - Ensure readability
   - Guide dengan visual cues

Dengan understanding ini, Anda dapat:
- Design yang immediately captures attention
- Guide viewers through information efficiently
- Prioritize content effectively
- Create clear dan compelling CTAs
- Optimize untuk short attention spans
- Test dan improve hierarchy

- often menjadi first point of contact antara brand dan audience. Pikirkan

Namun, effective branding goes far beyond just membuat logo yang "cantik".

touchpoint. Dari business cards hingga billboards, dari websites hingga

recognition, dan build emotional connections.

logo design. Terlalu sering, designers fokus pada trends atau personal
aesthetic preferences, forgetting tentang strategic role yang logo dan brand

      - Reputation, associations, feelings
      - Promise yang dibuat ke customers
      - Intangible asset dengan real value
      - Lived experience, not just visuals

      Brand includes:
      - What you do (products/services)
      - How you do it (processes, quality)
      - Why you do it (mission, values)
      - Who you are (personality, culture)
      - What people think dan feel about you

      Key insight: Brand exists dalam people's minds, bukan dalam your
      control fully. You influence, but don't fully control.

**b. BRANDING**

      Definisi:
      Branding adalah active process dari building dan managing brand.

      Components:
      - Brand strategy (positioning, messaging, values)
      - Brand identity (visual manifestation)
      - Brand experience (all touchpoints)
      - Brand management (consistency, evolution)
      - Brand communication (marketing, storytelling)

      Branding activities:
      - Market research
      - Competitive analysis
      - Positioning development
      - Identity design
      - Messaging framework
      - Implementation guidelines
      - Ongoing management

      Goal: Create strong, positive, consistent perceptions

**c. BRAND IDENTITY**

      Definisi:
      Brand identity adalah visible elements of brand - collection dari
      visual, verbal, dan sensory expressions yang represent brand.

      Core components:

      VISUAL IDENTITY:
      - Logo (primary symbol)
      - Color palette
      - Typography
      - Imagery style (photography/illustration)
      - Graphic elements (patterns, icons, shapes)
      - Layout principles

      VERBAL IDENTITY:
      - Brand name
      - Tagline/slogan
      - Tone of voice
      - Key messages
      - Naming conventions

      SENSORY IDENTITY (when applicable):
      - Sound (sonic logo, brand music)
      - Scent (retail environments)
      - Tactile (materials, finishes)
      - Taste (F&B brands)

      Purpose:
      - Make brand recognizable
      - Communicate brand personality
      - Differentiate from competitors
      - Build emotional connections
      - Create consistency

#### 2. Relationship Between Brand, Branding, dan Brand Identity

   Think of it as:

   BRAND (the entity):
   - "Who we are"
   - Core essence, values, purpose
   - Exists dalam minds dan hearts

   BRANDING (the process):
   - "What we do to build brand"
   - Strategy, planning, implementation
   - Ongoing activities dan management

   BRAND IDENTITY (the expression):
   - "How we look, sound, feel"
   - Tangible manifestations
   - Visual dan verbal systems

   Analogy:
   - Brand = person's character dan reputation
   - Branding = person's efforts to shape perception
   - Brand identity = person's appearance dan manner of speaking

#### 3. Why Brand Identity Matters

**a. RECOGNITION**

      - Consistent identity makes brand instantly recognizable
      - Familiar = trustworthy dalam consumer psychology
      - Shortcuts cognitive processing

**b. DIFFERENTIATION**

      - Stands out dalam crowded marketplace
      - Communicates unique value proposition
      - Prevents confusion dengan competitors

**c. TRUST & CREDIBILITY**

      - Professional identity signals professionalism
      - Consistency builds confidence
      - Visual quality indicates product/service quality

**d. EMOTIONAL CONNECTION**

      - Visual language evokes feelings
      - Personality expressed through design
      - Creates affinity dan loyalty

**e. UNITY & ALIGNMENT**

      - Internal: aligns team around shared identity
      - External: presents coherent face to world
      - All communications work together

**f. COMMERCIAL VALUE**

      - Strong brands command premium prices
      - Brand equity is significant asset
      - Facilitates business growth

### B. LOGO DESIGN

#### 1. Definisi dan Fungsi Logo

   Definisi:
   Logo adalah graphic mark, emblem, atau symbol used untuk aid dan promote
   public identification dan recognition.

   Primary functions:
   - IDENTIFICATION: "This is who we are"
   - DIFFERENTIATION: "We are not them"
   - COMMUNICATION: Convey personality, values, category
   - UNIFICATION: Umbrella untuk all brand expressions

   What makes good logo:
   - SIMPLE: Easy to recognize dan remember
   - MEMORABLE: Distinctive, unique
   - TIMELESS: Not overly trendy (should last 10+ years)
   - VERSATILE: Works dalam berbagai sizes dan contexts
   - APPROPRIATE: Suits brand personality dan industry
   - MEANINGFUL: Communicates something about brand (though can be subtle)

#### 2. Types of Logos

**a. WORDMARK (Logotype)**

      Definisi: Logo yang consists entirely dari brand name dalam styled
      typography.

      Characteristics:
      - Typography is the design
      - Custom atau heavily customized font
      - May have subtle graphic elements

      Examples:
      - Google, Coca-Cola, FedEx, Disney, IBM

      When to use:
      - Brand name is short, unique, memorable
      - Want strong name recognition
      - Typography can carry personality

      Advantages:
      - Clear name communication
      - No symbol to explain
      - Good untuk new atau unknown brands

      Disadvantages:
      - Harder to use without name visible
      - Less iconic recognition initially
      - Typography must be very distinctive

**b. LETTERMARK (Monogram)**

      Definisi: Logo dari initials atau abbreviated name.

      Characteristics:
      - Usually 2-4 letters
      - Stylized typography
      - Often geometric arrangement

      Examples:
      - HBO, CNN, NASA, HP, IBM

      When to use:
      - Brand name is long atau complicated
      - Initials are memorable
      - Need compact mark

      Advantages:
      - Simplicity
      - Easier to trademark
      - Compact for small applications

      Disadvantages:
      - Must build recognition of initials
      - Can be confused dengan other abbreviations
      - Less descriptive

**c. PICTORIAL MARK (Brand Mark/Logo Symbol)**

      Definisi: Iconic, graphic-based logo. Recognizable image/symbol.

      Characteristics:
      - Literal representation
      - Recognizable object atau concept
      - Works without words

      Examples:
      - Apple (apple), Twitter (bird), Target (bullseye), Shell (shell)

      When to use:
      - Want universal recognition
      - Cross language barriers
      - Strong, simple concept to visualize

      Advantages:
      - Universal, transcends language
      - Highly memorable when successful
      - Can convey meaning quickly

      Disadvantages:
      - Requires significant brand building
      - Can be limiting (tied to specific image)
      - Risk of misinterpretation

**d. ABSTRACT MARK**

      Definisi: Abstract geometric form that doesn't picture anything
      concrete.

      Characteristics:
      - Non-representational
      - Open to interpretation
      - Often conveys feeling over literal meaning

      Examples:
      - Nike (swoosh), Pepsi (circle), Adidas (three stripes)

      When to use:
      - Want unique, distinctive mark
      - Avoid being too literal
      - Convey feeling atau concept abstractly

      Advantages:
      - Completely unique
      - Flexible meaning
      - Can feel modern, sophisticated

      Disadvantages:
      - Requires brand building for meaning
      - No inherent meaning to communicate
      - Can feel generic if not well-designed

**e. MASCOT LOGO**

      Definisi: Logo featuring character atau mascot representing brand.

      Characteristics:
      - Illustrated character
      - Often friendly, approachable
      - Can be animated

      Examples:
      - KFC (Colonel Sanders), Pringles (Mr. Pringles), Michelin (Bibendum)

      When to use:
      - Want personable, friendly brand
      - Targeting families atau children
      - Need spokescharacter untuk marketing

      Advantages:
      - Engaging, memorable
      - Great untuk storytelling
      - Builds emotional connection

      Disadvantages:
      - Can be juvenile
      - Complex untuk small applications
      - May not age well

**f. EMBLEM LOGO**

      Definisi: Text inside symbol atau icon; badge, seal, atau crest style.

      Characteristics:
      - Enclosed design
      - Text dan symbol integrated
      - Often traditional, official feel

      Examples:
      - Starbucks, Harley-Davidson, NFL teams, universities

      When to use:
      - Want traditional, established feel
      - Schools, government, organizations
      - Heritage atau prestige important

      Advantages:
      - Distinctiveness
      - Traditional, trustworthy feel
      - Unified design

      Disadvantages:
      - Less versatile (complex untuk small sizes)
      - Can feel dated
      - Harder to separate elements

**g. COMBINATION MARK**

      Definisi: Combines wordmark dengan symbol (pictorial atau abstract).

      Characteristics:
      - Text + image together
      - Can be used together atau separately when established
      - Most flexible

      Examples:
      - Burger King, Doritos, Lacoste (alligator + name)

      When to use:
      - Want best of both worlds
      - Building new brand (name + symbol recognition)
      - Need versatility

      Advantages:
      - Clearest communication
      - Flexible usage
      - Builds recognition untuk both elements

      Disadvantages:
      - More complex
      - Requires careful integration
      - Multiple versions to manage

#### 3. Logo Design Principles

**a. SIMPLICITY**

      Why:
      - More memorable
      - Works dalam small sizes
      - Versatile across applications
      - Timeless (complex designs date quickly)

      How:
      - Remove unnecessary details
      - Limit colors (2-3 max often best)
      - Clean lines dan shapes
      - Avoid clutter

      Test: Can it be drawn dari memory after brief viewing?

**b. MEMORABLE/DISTINCTIVE**

      Why:
      - Stand out dalam cluttered marketplace
      - Aid recall
      - Build recognition

      How:
      - Unique visual approach
      - Clever concept atau twist
      - Distinctive shapes atau forms
      - Avoid clichés

      Test: Does it look like everything else, atau does it stand out?

**c. TIMELESS**

      Why:
      - Logo redesigns expensive
      - Consistency builds equity
      - Trends date quickly

      How:
      - Avoid trendy effects (gradients, shadows, 3D unless essential)
      - Classic, clean aesthetics
      - Focus on solid concept over style
      - Think 10+ years ahead

      Test: Will this still work dalam 10 years?

**d. VERSATILE/SCALABLE**

      Why:
      - Used across many contexts
      - Tiny (favicon, app icon) to huge (billboard)
      - Different media (screen, print, embroidery, engraving)
      - Various backgrounds

      How:
      - Works dalam one color (for special applications)
      - Works on light AND dark backgrounds
      - Detail visible at small sizes
      - Clear at any size

      Test: Print at 1 inch wide - still recognizable?

**e. APPROPRIATE**

      Why:
      - Must fit brand personality
      - Should suit industry context
      - Must resonate dengan target audience

      How:
      - Research brand thoroughly
      - Understand audience expectations
      - Match visual language to brand attributes
      - Balance uniqueness dengan appropriateness

      Test: Does this feel right untuk this brand?

**f. MEANINGFUL (but not required)**

      Why:
      - Can add depth
      - Makes brand storytelling easier
      - Creates talking points

      How:
      - Clever visual metaphors
      - Hidden meanings (FedEx arrow)
      - Cultural references
      - Historical connections

      Note: Meaning is nice-to-have, not necessity. Many great logos
      (Nike, Apple) have minimal inherent meaning but built associations
      over time.

#### 4. Logo Design Process

   STAGE 1: RESEARCH & DISCOVERY (20%)

**a. Understand the Brand**

      - What does company do?
      - What are core values?
      - What's brand personality?
      - What's mission dan vision?

**b. Understand the Audience**

      - Who are target customers?
      - What do they value?
      - What aesthetics appeal to them?

**c. Competitive Analysis**

      - What do competitors' logos look like?
      - What works, what doesn't?
      - How to differentiate?
      - What are category conventions?

**d. Technical Requirements**

      - Where will logo be used?
      - Size considerations?
      - Color limitations?
      - File format needs?

   STAGE 2: EXPLORATION & IDEATION (30%)

**a. Brainstorming**

      - List keywords associated dengan brand
      - Mind mapping concepts
      - Word associations
      - Visual metaphors

**b. Sketching**

      - Many rough sketches (50-100+)
      - Explore different directions
      - Don't judge yet, just generate
      - Thumbnail sketches, quick iterations

      Focus:
      - Different logo types
      - Various concepts
      - Multiple visual treatments

      Don't go digital too soon!

**c. Select Strongest Concepts**

      - Narrow to 3-5 strongest directions
      - Different approaches
      - Present variety to self/team/client

   STAGE 3: REFINEMENT (30%)

**a. Digital Execution**

      - Take selected sketches to Inkscape
      - Refine shapes
      - Experiment dengan proportions
      - Try different variations

**b. Test Versatility**

      - Try one-color versions
      - Test pada dark/light backgrounds
      - Scale up dan down
      - Try dalam different contexts

**c. Refine Details**

      - Perfect curves dan angles
      - Balance negative space
      - Adjust optical alignment
      - Fine-tune proportions

   STAGE 4: PRESENTATION & FINALIZATION (20%)

**a. Create Mockups**

      - Show logo dalam context
      - Business cards, signage, website, products
      - Help client visualize

**b. Prepare Variations**

      - Full color
      - One color (black)
      - Reverse (white)
      - Horizontal dan vertical layouts if applicable

**c. Documentation**

      - Design rationale
      - Usage guidelines (basics)
      - File deliverables

#### 5. Common Logo Design Mistakes

   - TOO COMPLEX: Will tidak work at small sizes
   - TOO TRENDY: Will date quickly
   - TOO MANY COLORS: Limits versatility
   - POOR SCALABILITY: Detail lost when small
   - CLICHÉ: Looks like everyone else
   - INAPPROPRIATE: Doesn't fit brand
   - ILLEGIBLE: Hard to read atau recognize
   - TOO LITERAL: No room untuk interpretation
   - COPYING: Similarity to existing logos (legal issues)
   - EFFECTS-DEPENDENT: Gradients, shadows that don't translate

### C. BRAND IDENTITY SYSTEMS

#### 1. Components Beyond Logo

**a. COLOR PALETTE**

      Elements:
      - Primary colors (1-2): main brand colors
      - Secondary colors (2-4): supporting colors
      - Accent colors (optional): highlight, special uses
      - Neutral colors: backgrounds, text

      Specifications:
      - Pantone (print standard)
      - CMYK (print process)
      - RGB (screen)
      - HEX (web)

      Guidelines:
      - When to use each color
      - Proportions dan dominance
      - Color combinations
      - Accessibility considerations

**b. TYPOGRAPHY**

      Elements:
      - Primary typeface (headlines, display)
      - Secondary typeface (body copy, supporting)
      - Sometimes tertiary (specific uses)

      Specifications:
      - Font families dan weights
      - Size hierarchy
      - Line spacing, letter spacing
      - Usage guidelines

      Considerations:
      - Pairing compatibility
      - Readability
      - Brand personality match
      - License untuk commercial use
      - Web font availability

**c. IMAGERY STYLE**

      Photography guidelines:
      - Subject matter
      - Composition style
      - Color treatment
      - Mood dan tone
      - What to avoid

      Illustration style:
      - Drawing style
      - Level of detail
      - Color approach
      - When to use

      Examples dan mood boards

**d. GRAPHIC ELEMENTS**

      Supporting visuals:
      - Patterns
      - Icons dan symbols
      - Shapes dan forms
      - Textures
      - Infographic styles

      Purpose:
      - Brand recognition
      - Visual interest
      - Consistency
      - Flexibility

**e. LAYOUT PRINCIPLES**

      Guidelines:
      - Grid systems
      - Spacing standards
      - Alignment principles
      - Composition approaches
      - Template structures

#### 2. Brand Consistency

   Why it matters:
   - Builds recognition
   - Strengthens brand memory
   - Projects professionalism
   - Increases trust
   - Improves marketing effectiveness

   How to maintain:

**a. BRAND STYLE GUIDE/GUIDELINES**

      - Document all standards
      - Visual examples
      - Do's dan don'ts
      - Accessible to all who need it

**b. TEMPLATES**

      - Pre-designed formats
      - Presentations, documents, social media
      - Ensure consistency without redesigning each time

**c. ASSET LIBRARY**

      - Centralized collection
      - All approved logos, images, graphics
      - Version control
      - Easy access untuk authorized users

**d. APPROVAL PROCESS**

      - Review before publishing
      - Brand guardian atau committee
      - Feedback dan corrections

**e. TRAINING**

      - Educate team on brand standards
      - Why guidelines exist
      - How to apply them

#### 3. Touchpoints untuk Brand Identity

   Where brand identity appears:

   PRINT:
   - Business cards, letterhead, envelopes
   - Brochures, flyers, posters
   - Packaging

   - Publications

   DIGITAL:
   - Website

   - Email signatures
   - Digital ads

   ENVIRONMENTAL:
   - Store design
   - Office space
   - Vehicle wraps
   - Trade show booths
   - Uniforms

   PRODUCT:
   - Product design itself
   - Packaging
   - Labels, tags

   - Follow brand guidelines
   - Adapt appropriately to context
   - Maintain recognition
   - Contribute to unified experience

## RINGKASAN

Pertemuan keenam ini covered comprehensive landscape dari brand identity
design:

#### 1. Foundations

   - Brand: perceptions dalam minds, reputation, promise
   - Branding: active process of building dan managing brand
   - Brand identity: visible expressions (visual, verbal, sensory)
   - Relationship: brand is essence, branding is process, identity is
     manifestation

#### 2. Logo Fundamentals

   - Central element of visual identity
   - Functions: identification, differentiation, communication, unification
   - Good logo is: simple, memorable, timeless, versatile, appropriate,
     (optionally) meaningful

#### 3. Logo Types

   - Wordmark: styled brand name
   - Lettermark: initials atau abbreviation
   - Pictorial mark: recognizable symbol
   - Abstract mark: non-representational form

## MATERI

### A. RASTER VS VECTOR: FUNDAMENTAL DIFFERENCES

#### 1. Raster Graphics (Bitmap)

   Definisi:
   Images composed dari grid of individual pixels, each dengan specific
   color value.

   Characteristics:

**a. PIXEL-BASED:**

      - Image adalah matrix of colored squares
      - Each pixel has fixed position dan color
      - Resolution-dependent (fixed dimensions)

**b. PHOTOGRAPHY & COMPLEX IMAGERY:**

      - Ideal untuk photorealistic images
      - Can capture subtle color gradations
      - Natural untuk photos, digital paintings

**c. FILE SIZE:**

      - Generally larger files
      - Size depends on dimensions + color depth
      - Compression dapat reduce (JPG, PNG)

**d. EDITING:**

      - Pixel-level manipulation
      - Color correction, filters, effects
      - Destructive editing (unless using layers)

**e. SCALING ISSUES:**

      - Upscaling = pixelation, loss of quality
      - Downscaling = loss of detail
      - Resolution must be determined at creation

   Formats:
   - JPEG (.jpg): compressed, photos, no transparency
   - PNG (.png): lossless compression, transparency support
   - GIF (.gif): limited colors, animation, transparency
   - TIFF (.tif): high quality, print, large files
   - PSD (.psd): Photoshop native
   - XCF (.xcf): GIMP native

   Best for:
   - Photographs
   - Complex color compositions
   - Photo editing dan manipulation
   - Digital painting
   - Textures
   - Detailed realistic images
   - Web graphics (certain types)

   NOT ideal for:
   - Logos (scaling issues)
   - Typography (aliasing when scaled)
   - Simple graphics (inefficient file size)
   - Print graphics requiring scaling

#### 2. Vector Graphics

   Definisi:
   Images composed dari mathematical paths, defined by points, lines,
   curves, dan formulas.

   Characteristics:

**a. PATH-BASED:**

      - Defined by mathematical equations
      - Points, curves, fills, strokes
      - Resolution-independent

**b. CRISP & CLEAN:**

      - Sharp edges at any size
      - Perfect untuk geometric shapes
      - Clean, scalable typography

**c. FILE SIZE:**

      - Generally smaller files
      - Complex paths can increase size
      - But still efficient untuk most graphics

**d. EDITING:**

      - Manipulate paths, nodes, shapes
      - Non-destructive by nature
      - Easy to revise dan modify

**e. INFINITE SCALABILITY:**

      - Scale to any size without quality loss
      - Tiny icon to billboard
      - Always crisp dan clean

   Formats:
   - SVG (.svg): web standard, scalable, editable
   - AI (.ai): Adobe Illustrator native
   - EPS (.eps): print standard, widely supported
   - PDF (.pdf): document standard, preserves vectors
   - CDR (.cdr): CorelDRAW native

   Best for:
   - Logos dan brand identity
   - Icons dan symbols
   - Typography dan type treatments
   - Illustrations (certain styles)
   - Infographics dan diagrams
   - Print graphics requiring scaling
   - Clean, geometric designs

   NOT ideal for:
   - Photographs
   - Complex color gradations
   - Highly detailed realistic imagery
   - Effects requiring pixel-level control

#### 3. Comparison Summary

   ASPECT           | RASTER                    | VECTOR
   -----------------|---------------------------|---------------------------
   Composition      | Pixels                    | Paths/mathematical
   Resolution       | Fixed                     | Independent
   Scaling          | Quality loss when scaled  | Infinite without loss
   File Size        | Larger (generally)        | Smaller (generally)
   Best For         | Photos, paintings         | Logos, icons, type
   Editing          | Pixel manipulation        | Path manipulation
   Complexity       | Great untuk complex images| Better untuk clean graphics
   Software         | GIMP, Photoshop, etc.     | Inkscape, Illustrator, etc.

#### 4. When to Use Which?

   Decision Framework:

   Q: Apakah project ini photography atau photo-based?
   → YES: Raster (GIMP)

   Q: Apakah project need to scale to different sizes?
   → YES: Vector (Inkscape)

   Q: Apakah ini logo atau brand identity element?
   → Vector (Inkscape)

   Q: Apakah design predominantly typography?
   → Vector (Inkscape)

   Q: Apakah project memerlukan realistic textures atau complex shading?
   → Raster (GIMP)

   Q: Apakah final output akan di-print dalam various sizes?
   → Vector (Inkscape) preferred

   HYBRID APPROACH:
   Many projects use BOTH:
   - Create vector elements dalam Inkscape
   - Import to GIMP untuk raster effects
   - Atau: Create raster textures dalam GIMP
   - Import to Inkscape as embedded images

   UNION (Ctrl+ +):

   DIFFERENCE (Ctrl+ -):
   - Subtract top path from bottom
   - Cut out shapes

   INTERSECTION (Ctrl+ *):
   - Keep only overlapping areas

   EXCLUSION (Ctrl+ ^):
   - Remove overlapping areas

   DIVISION (Ctrl+ /):
   - Slice path by path
   - Creates separate pieces

   These operations are ESSENTIAL untuk logo design

## RINGKASAN

Pertemuan ketujuh ini memberikan practical foundation dalam software desain
grafis:

#### 1. Raster vs Vector

   - Raster: pixels, resolution-dependent, photos/complex images, GIMP
   - Vector: paths, resolution-independent, logos/clean graphics, Inkscape
   - Each has strengths; choose based on project needs
   - Can be combined dalam single project

#### 2. GIMP Fundamentals

   - Raster image manipulation
   - Tools: selection, paint, transform, color
   - Layers untuk non-destructive editing
   - Color correction dan adjustments
   - Photo editing dan compositing
   - Export untuk various uses

#### 3. Inkscape Fundamentals

   - Vector graphics creation
   - Tools: selection, shapes, pen tool, text
   - Path operations: union, difference, intersection, dll
   - Fill and stroke management
   - Precision dengan alignment tools
   - Scalable output

#### 4. Practical Skills

   - Navigation dalam both interfaces
   - Essential tool usage
   - Basic workflows
   - File management
   - Format selection untuk delivery

#### 5. Integration

   - Using both tools dalam one project
   - Vector elements dalam raster compositions
   - Efficient workflow between aplikasi

Dengan hands-on practice hari ini dan praktikum exercises, Anda sekarang
equipped dengan technical skills untuk execute designs. Remember: these are
tools untuk realize creative vision - master them, tetapi don't let them
limit creativity.
