Raster vs Vector

AspekRaster (Bitmap)Vector
Terdiri dariPixels (titik warna)Paths matematis (titik, garis, kurva)
ScalingPecah/blur saat diperbesarTajam di ukuran apapun
File sizeBesar (tergantung resolusi)Relatif kecil
Cocok untukFoto, digital painting, photo editingLogo, ikon, ilustrasi, tipografi
SoftwareGIMP, PhotoshopInkscape, Illustrator
FormatJPG, PNG, GIF, TIFF, XCF, PSDSVG, AI, EPS, PDF
When to Use Which?

Gunakan raster (GIMP) untuk: edit foto, manipulasi gambar, digital painting, texture, web graphics dengan efek kompleks. Gunakan vector (Inkscape) untuk: logo, ikon, ilustrasi flat, infografis, desain yang harus discale (kartu nama hingga billboard). Banyak project menggunakan keduanya: logo vector + foto raster dalam satu poster.

GIMP — Raster Editing

Interface Overview ▼
  • Toolbox (kiri): selection, paint, transform tools
  • Tool Options: di bawah toolbox — setting tool aktif
  • Canvas (tengah): area kerja image
  • Layers/Channels/Paths (kanan): panel dockable
  • Single-Window Mode: Windows → Single-Window Mode (recommended)
Essential Tools ▼
  • Selection tools: Rectangle (R), Ellipse (E), Free Select/Lasso (F), Fuzzy Select/Magic Wand (U), Select by Color
  • Move (M) dan Crop (Shift+C)
  • Transform: Scale, Rotate, Flip, Perspective (Shift+T unified transform)
  • Paint: Pencil (N), Paintbrush (P), Eraser (Shift+E), Bucket Fill (Shift+B), Gradient (G)
  • Clone (C) dan Heal (H): retouching — hapus objek/noda
  • Text (T): tambah teks sebagai layer terpisah
  • Color Picker (O): ambil warna dari image
Working dengan Layers ▼
  • Layer = lembar transparan bertumpuk — edit non-destructive
  • Selalu duplicate layer sebelum edit besar (safety backup)
  • Layer modes: Multiply, Screen, Overlay untuk blending effects
  • Opacity: transparansi per layer
  • Layer mask: sembunyikan bagian layer tanpa menghapus
  • Flatten hanya saat export final
Adjustment Penting ▼
  • Colors → Levels: perbaiki exposure dengan histogram
  • Colors → Curves: kontrol tonal paling powerful
  • Colors → Hue-Saturation: ubah warna dan intensitas
  • Colors → Brightness-Contrast: penyesuaian cepat
  • Filters: Blur, Sharpen, Artistic, Light and Shadow

Inkscape — Vector Graphics

Essential Tools ▼
  • S Selection tool — pilih, pindah, scale, rotate objek
  • N Node tool — edit titik dan kurva path
  • R Rectangle, E Ellipse, * Star/Polygon
  • B Bezier/Pen tool — gambar path custom
  • P Pencil — freehand drawing
  • T Text tool
  • G Gradient tool
  • F7 Dropper — ambil warna
Path Operations (Boolean) ▼
  • Ctrl++ Union — gabungkan shapes
  • Ctrl+− Difference — potong shape atas dari bawah
  • Ctrl+* Intersection — hanya area overlap
  • Ctrl+^ Exclusion — semua kecuali overlap
  • Path → Object to Path (Shift+Ctrl+C): convert shape/text ke editable path
  • Kombinasi path operations = teknik utama desain logo
Fill, Stroke & Alignment ▼
  • Shift+Ctrl+F Fill and Stroke dialog — warna, gradient, pattern, stroke width
  • Klik palette = fill; Shift+klik = stroke
  • Shift+Ctrl+A Align and Distribute — rata kiri/tengah/kanan, distribusi merata
  • Object → Raise/Lower untuk z-order (Page Up/Down)
  • Group: Ctrl+G, Ungroup: Shift+Ctrl+G

Common Workflows

GIMP:

  1. Open image → Duplicate layer (backup)
  2. Adjustments: Colors → Levels, Curves, Hue-Saturation
  3. Retouching: Clone dan Heal tools
  4. Crop jika perlu
  5. Export: File → Export As (JPG/PNG)

GIMP:

  1. Open base image + open gambar tambahan
  2. Select, copy, paste as new layers
  3. Hapus background dengan selection + delete (atau layer mask)
  4. Blend dengan layer modes dan opacity
  5. Adjust colors untuk konsistensi antar elemen
  6. Export final

Inkscape:

  1. New document
  2. Gambar shapes dasar (circle, rectangle)
  3. Gunakan path operations (union, difference)
  4. Tambahkan teks, sesuaikan font
  5. Atur fill colors, hilangkan stroke (atau sebaliknya)
  6. Align elements dan group
  7. Save SVG (master) + export PNG
6

File Formats & Export

Format Raster
JPG: foto, lossy, file kecil — no transparency
PNG: lossless, support transparency — web graphics
GIF: animasi sederhana, 256 warna
TIFF: print quality, lossless
XCF: native GIMP — simpan layers
Format Vector
SVG: standar web, native Inkscape, editable
PDF: print dan dokumen, universal
EPS: legacy print format
AI: Adobe Illustrator (Inkscape bisa import)
Export Guidelines
  • Web: PNG/JPG, 72 DPI, RGB, optimasi ukuran file
  • Print: PDF/TIFF, 300 DPI, CMYK (perhatikan konversi warna)
  • Logo delivery: SVG + PNG (berbagai ukuran) + PDF
  • Selalu simpan file master (XCF/SVG) terpisah dari export

Praktikum Pertemuan 7

Latihan 1: GIMP Photo Enhancement
  • Pilih foto dengan exposure/warna kurang baik
  • Perbaiki dengan Levels, Curves, Hue-Saturation
  • Retouch noda/objek mengganggu dengan Clone/Heal
  • Export before-after comparison
Latihan 2: GIMP Composite
  • Gabungkan minimal 3 gambar menjadi satu komposisi
  • Gunakan layers, layer mask, dan blending modes
  • Pastikan pencahayaan dan warna konsisten
Latihan 3: Inkscape Vector Practice
  • Buat logo sederhana dengan path operations
  • Buat 4 ikon flat dengan style konsisten
  • Latih Bezier tool: trace bentuk sederhana
  • Export SVG + PNG
Latihan 4: Integrasi GIMP + Inkscape
  • Buat poster sederhana: foto (edit di GIMP) + logo dan teks vector (Inkscape)
  • Workflow: edit foto di GIMP → export PNG → import ke Inkscape → layout final
  • Export hasil akhir sebagai PNG dan PDF
