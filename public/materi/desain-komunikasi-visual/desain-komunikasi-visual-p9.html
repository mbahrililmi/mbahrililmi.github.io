Advanced Path Operations

Boolean Operations (Review+)
Union (Ctrl++): gabung shapes
Difference (Ctrl+−): potong
Intersection (Ctrl+*): area overlap
Exclusion (Ctrl+^): kebalikan intersection
Division (Ctrl+/): potong jadi bagian terpisah
Advanced Path Editing
Node tool (N): tambah/hapus/geser node
Node types: cusp, smooth, symmetric, auto
Simplify (Ctrl+L): kurangi jumlah node
Inset/Outset (Ctrl+( / Ctrl+)): perbesar/perkecil path
Clipping dan Masking ▼
  • Clip: potong objek dengan bentuk lain — Object → Clip → Set. Tepi tajam, on/off penuh
  • Mask: seperti clip tapi pakai grayscale — putih=terlihat, hitam=tersembunyi, abu-abu=transparan sebagian
  • Gunakan clip untuk crop foto dalam bentuk, mask untuk fade effects
  • Release: Object → Clip/Mask → Release

Gradients & Effects

Linear Gradient
Transisi warna sepanjang garis. Multiple color stops untuk transisi kompleks. Drag dengan Gradient tool (G).
Radial Gradient
Transisi dari pusat keluar. Cocok untuk efek cahaya, sphere, glow.
Mesh Gradient
Grid gradient untuk transisi kompleks multi-arah — realistic shading pada ilustrasi.
Filters dan Effects ▼
  • Blur: Fill & Stroke → Blur slider, atau Filters → Blurs
  • Drop Shadow: Filters → Shadows and Glows → Drop Shadow
  • Filters menu: Bevels, Distort, Morphology, Textures
  • Opacity per objek untuk layering effects
  • Hati-hati: filter berlebihan memperlambat render dan bisa bermasalah saat export print
4

Text Effects & Typography Artistic

Advanced Text Manipulation
  • Text to Path (Shift+Ctrl+C): convert teks ke path — edit per huruf, kustomisasi bentuk
  • Text on Path: Text → Put on Path — teks mengikuti kurva
  • Kerning manual: Alt+panah untuk geser huruf individual
  • Gradient fill pada teks + drop shadow untuk depth
  • Outline text: duplicate, beri stroke tebal di belakang, fill di depan
  • 3D effect: duplicate berlapis dengan offset dan warna lebih gelap
Typography Best Practices untuk Vector

Selalu convert teks ke path sebelum kirim file ke orang lain/percetakan (menghindari masalah missing font). Simpan versi dengan teks editable sebagai master. Maksimal 2-3 font, perhatikan hierarchy dan readability.

Ilustrasi Vector Kompleks

  1. Mulai dari sketch/referensi
  2. Break down objek kompleks menjadi shapes sederhana
  3. Bangun dari bentuk besar ke detail kecil
  4. Gunakan layers untuk organisasi (background, midground, foreground)
  5. Konsisten dengan style: flat, semi-flat, atau detailed
  6. Tambahkan shading dengan duplicate + darker color + clip
  7. Highlight dengan lighter shapes dan opacity
  1. Sketch proporsi dasar (kepala, badan, anggota tubuh)
  2. Buat basic shapes untuk setiap bagian tubuh
  3. Arrange dan resize untuk proporsi yang tepat
  4. Tambahkan fill colors (flat dulu)
  5. Detail: fitur wajah, outline, aksesoris
  6. Gradients untuk depth, shadow sederhana di bawah karakter
  • Gunakan grid (24×24 atau 48×48 unit) untuk konsistensi
  • Stroke width konsisten antar ikon dalam satu set
  • Corner radius seragam
  • Simplify — ikon harus terbaca di ukuran kecil
  • Style konsisten: outline, filled, atau duotone
  • Test di ukuran 16px, 24px, 48px

Infographic Design

Prinsip Infographic
Satu cerita utama yang jelas. Data akurat dan tervisualisasi jujur. Hierarchy: judul → poin utama → detail. Visual lebih banyak dari teks. Flow yang memandu pembaca.
Jenis Infographic
Statistical (data & charts), Informational (penjelasan konsep), Timeline (kronologi), Process (langkah-langkah), Comparison (perbandingan), Geographic (peta).
Workflow Infographic di Inkscape
  1. Research data dan tentukan key message (15 menit)
  2. Sketch struktur layout (10 menit)
  3. Setup dokumen dan guidelines (5 menit)
  4. Buat charts/visualizations — bar, pie, ikon proporsional (25 menit)
  5. Tambahkan ikon dan elemen dekoratif (15 menit)
  6. Typography dan konten teks (15 menit)
  7. Final refinement dan polish (5 menit)
7

Persiapan File Print Production

Color Modes
RGB untuk digital/layar. CMYK untuk print. Inkscape bekerja di RGB — konversi CMYK dilakukan saat export PDF atau oleh percetakan. Warna bisa bergeser, test print penting.
Resolution & Bleed
Vector tidak punya resolusi — tapi raster effects perlu 300 DPI. Bleed: 3-5mm di luar trim area untuk elemen sampai tepi. Safe zone: 5mm dari trim untuk konten penting.
Pre-Print Checklist ▼
  • ☐ Semua teks sudah convert to path
  • ☐ Bleed 3-5mm sudah ditambahkan
  • ☐ Konten penting dalam safe zone
  • ☐ Raster images minimal 300 DPI
  • ☐ Export PDF dengan setting print quality
  • ☐ Proof/preview sudah dicek (warna, typo, posisi)
  • ☐ File master SVG disimpan terpisah

Praktikum & Tugas

Latihan 1: Logo & Icon Set
  • Buat logo dengan kombinasi path operations
  • Buat icon set 4 ikon dengan style konsisten
Latihan 2: Gradient & Text Effects
  • Bandingkan linear, radial, dan mesh gradient pada shape yang sama
  • Buat teks "VECTOR ART" dengan gradient fill + drop shadow + variasi outline
Latihan 3: Character Illustration
  • Buat karakter sederhana dari basic shapes
  • Fill colors → detail wajah → gradients untuk depth → shadow
Latihan 4: Infographic A4
  • Ukuran A4 portrait (210×297mm), topik bebas dengan data nyata
  • Include: judul, minimal 2 visualisasi data, ikon, sumber data
  • Ikuti workflow 90 menit yang dijelaskan di atas
TUGAS: Vector Illustration Project

Buat satu karya vector illustration kompleks (poster ilustrasi, infografis lengkap, atau ilustrasi karakter + environment). Penilaian: Concept & Creativity (25%), Technical Execution (30%), Visual Design (25%), Functionality (15%), Documentation (5%). Deliverables: SVG master + PNG export + dokumentasi proses.
