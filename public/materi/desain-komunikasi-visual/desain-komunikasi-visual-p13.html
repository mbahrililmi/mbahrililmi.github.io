Digital vs Print Brosur

Karakteristik Digital Brosur
RGB color, 72-150 DPI, dilihat di layar (sering mobile), bisa interaktif (link, navigasi, video), distribusi instant dan gratis, mudah diupdate, ukuran file penting.
Design Considerations
Landscape orientation lebih cocok untuk layar. Font lebih besar (min 12-14pt). Kontras kuat untuk berbagai layar. Navigasi jelas antar halaman. Single-page scroll juga opsi.
Interactive PDF Features ▼
  • Hyperlinks: ke website, email, telepon
  • Internal navigation: tombol ke halaman/section tertentu, table of contents clickable
  • Bookmarks: struktur navigasi di sidebar PDF reader
  • Embedded media: (terbatas dukungan) — lebih aman link ke video online
  • Tips: file di bawah 10MB, test di beberapa PDF reader, sediakan versi print-friendly, alt text untuk accessibility

Email Marketing Fundamentals

Mengapa Email Marketing?

Email marketing memiliki ROI tertinggi di antara channel digital (~$36-42 per $1). Audience yang owned (bukan pinjam platform), targeting personal, dan measurable penuh. EDM (Email Direct Marketing) = email promosi yang dirancang untuk action spesifik.

Jenis Email
Newsletter (rutin, informatif), Promotional (penawaran), Transactional (konfirmasi), Welcome series, Re-engagement.
Anatomy Email
Subject line → preheader → header/hero → body → CTA → footer (unsubscribe, kontak, social).
Legal & Etika
Wajib: unsubscribe link, identitas pengirim jelas, izin (opt-in). Jangan beli email list!
4

EDM Design Principles

  • 60%+ email dibuka di mobile — design mobile-first
  • Single column layout untuk mobile
  • Lebar email 600-640px standar
  • Font minimal 14-16px untuk body
  • Tombol CTA minimal 44×44px (thumb-friendly)
  • Compress gambar — total email di bawah 102KB ideal
  • Test di email client nyata (Gmail, Outlook, mobile), bukan hanya preview
  • Informasi terpenting above the fold (terlihat tanpa scroll)
  • Satu pesan utama per email
  • Flow: hero visual → headline → benefit singkat → CTA
  • Design untuk scanning, bukan reading — paragraf pendek, bullet
  • Jangan image-only — banyak user mematikan gambar (plus alt text wajib)
  • Satu primary CTA — terlalu banyak menciptakan kebingungan
  • Button, bukan text link, untuk primary action
  • Warna kontras dengan keseluruhan email
  • Copy action-oriented: "Klaim Diskon", "Daftar Sekarang", "Lihat Koleksi"
  • Ulangi CTA di akhir email untuk pembaca yang scroll penuh

Tools: Canva & GIMP untuk EDM

Canva untuk Email Design
Template email/newsletter siap pakai. Resize untuk format berbeda. Export PNG per section untuk email builder. Brand kit untuk konsistensi warna/font. Drag-and-drop cepat untuk non-coder.
GIMP untuk Image Processing
Kompres dan resize gambar (Export As → kualitas JPG 70-85%). Crop hero image ke rasio tepat. Optimasi file size tanpa kehilangan kualitas visual. Batch processing untuk banyak gambar.
Workflow Interactive PDF
  1. Define purpose dan audience
  2. Outline struktur konten (4-6 halaman)
  3. Daftar elemen interaktif yang dibutuhkan
  4. Sketch layout per halaman
  5. Desain di Canva → export PDF
  6. Tambahkan link/navigasi (Canva mendukung link saat export PDF)
  7. Test interactivity di beberapa PDF reader

A/B Testing & Metrics

A/B Testing Fundamentals

  • Test satu variabel dalam satu waktu: subject line, hero image, warna CTA, copy
  • Sample size memadai dan durasi cukup untuk signifikansi
  • Dokumentasikan semua hasil untuk pembelajaran kampanye berikutnya
  • Insight per segmen — yang works untuk satu audiens belum tentu untuk yang lain
Open Rate
% penerima yang membuka. Dipengaruhi subject line, sender name, timing. Benchmark ~20%.
Click-Through Rate
% yang klik link/CTA. Dipengaruhi desain, hierarchy, CTA. Benchmark ~2-3%.
Conversion Rate
% yang menyelesaikan action (beli, daftar). Metrik bisnis sesungguhnya.
Common Mistakes EDM ▼
  • Image-only emails (accessibility & deliverability buruk)
  • Teks terlalu kecil di mobile
  • Terlalu banyak CTA
  • Tidak test mobile sebelum kirim
  • Broken links dan missing unsubscribe
  • File terlalu besar — loading lambat
  • Branding tidak konsisten dengan touchpoint lain
7

Praktikum & Tugas

Praktikum: Digital Brosur + EDM Campaign
  • Interactive digital brosur (PDF): 4-6 halaman, navigasi, hyperlinks, mobile-readable
  • EDM campaign series: minimal 2 email dengan desain konsisten
  • Evaluasi: first impression, mobile-responsiveness, visual hierarchy, CTA effectiveness
  • Sertakan A/B test proposal: elemen yang ditest, versi A/B, hipotesis, success metric
TUGAS INDIVIDUAL: Personal Brand EDM Campaign

Buat kampanye email untuk personal brand Anda: (1) definisi audience, (2) seri 3 email — welcome, value/content, promotional — dengan desain konsisten, (3) campaign strategy document 2-3 halaman, (4) A/B test proposal, (5) mobile preview screenshots + source files.
