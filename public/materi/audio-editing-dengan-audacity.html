Importing & Managing Audio Files

Audacity mendukung berbagai format audio. Pahami perbedaan antara menyimpan project dan melakukan export untuk delivery.

Format Import (Input)

WAV AIFF FLAC MP3 OGG M4A / AAC WMA + FFmpeg
3 Cara Import:
1. Ctrl+Shift+I → Browse file
2. Drag & Drop dari Explorer
3. File → Open (window baru)

Format Export (Output)

Format Kegunaan
WAV 48kHz/24bit Master / Archiving
MP3 192kbps Podcast delivery
MP3 128kbps Speech-only podcast
FLAC Lossless distribution
OGG Web audio

Project vs Export

💾
Save Project (.aup3)
Menyimpan semua edits, tracks, undo history. Hanya bisa dibuka di Audacity. BUKAN file untuk dikirim ke orang lain. Gunakan: Ctrl+S
📤
Export Audio
Membuat file deliverable (WAV, MP3, dll). Ini yang dikirim, diupload, atau diserahkan. Gunakan: Ctrl+Shift+E
⚠ Penting: Jangan kirim file .aup3 kepada orang lain. Selalu Export ke format standar (WAV/MP3) untuk delivery.

Cut vs Delete vs Silence

✂️
Cut Ctrl+X
Menghapus selection & menyimpannya ke clipboard. Gap tertutup (audio setelah selection bergeser ke kiri). Dapat di-paste ulang.
🗑️
Delete Del
Menghapus selection tanpa menyimpan ke clipboard. Gap tertutup. Lebih cepat dari Cut jika tidak perlu paste.
🔇
Silence Ctrl+L
Mengganti selection dengan diam (zeros). Durasi TETAP tidak berubah. Cocok untuk menghapus bunyi batuk tanpa mengubah timing.

🎚️ Noise Reduction
Two-step process: (1) Get Noise Profile → (2) Apply Reduction
Get Noise Profile

Select 1–2 detik audio yang HANYA berisi noise (tidak ada suara/musik). Biasanya di awal atau akhir rekaman. Lalu: Effect → Noise Reduction → Get Noise Profile

Apply Noise Reduction

Pilih seluruh audio (Ctrl+A), lalu kembali ke Effect → Noise Reduction. Atur parameter dan klik OK.

Noise Reduction (dB)12 dB
Sensitivity6.0
Frequency Smoothing3 bands
Tips: Less is more! Nilai terlalu tinggi → suara "underwater"/artifact. Mulai dari 12 dB, naikkan hanya jika perlu. Tidak bisa menghapus noise yang bervariasi (suara orang, traffic).
📊 Normalization

Meninggikan volume seluruh audio secara proporsional sehingga sample terkeras mencapai level yang ditentukan.

Normalize peak to-1.0 dB
Selalu centang:
✓ Remove DC offset
✓ Normalize peak amplitude
-1.0 dB memberikan sedikit headroom agar tidak terjadi clipping di tahap selanjutnya.

Mixing Tips untuk Podcast

🎵 Balancing Voice + Background Music

▼

Voice harus selalu menjadi elemen paling jelas terdengar. Background music berfungsi sebagai pendukung, bukan pesaing.

  • Voice track: Gain sekitar 0 dB (reference level)
  • Background music: -18 dB s/d -24 dB saat voice berbicara
  • Gunakan Envelope Tool (E) untuk volume automation
  • Fade musik ke bawah saat voice mulai, naik kembali saat pause
  • High-pass filter musik di 200–300 Hz untuk memberikan ruang pada voice

🎼 Generate Audio Elements

▼

Generate Menu

  • Tone — Sine, square, sawtooth waves
  • Silence — Insert pause yang presisi
  • White/Pink Noise — Ambience, testing
  • Chirp — Frequency sweep
  • DTMF Tones — Suara telepon
  • Pluck — Petikan string sintetis

Use Cases

  • Music bed untuk podcast
  • Sound effects sederhana
  • Test signal (sine wave 1kHz)
  • Insert pause antara segmen
Materi 06

Podcast Production Workflow

Langkah-langkah lengkap dari raw recording hingga file siap publish. Ikuti urutan ini setiap kali memproduksi episode.

Import Assets

Import semua file: rekaman voice, intro music, outro music, jingles. Drag & drop ke Audacity atau gunakan Ctrl+Shift+I.

Arrangement

Susun struktur: intro music → main content → mid-roll → outro music. Gunakan Time Shift Tool (T) untuk memindahkan clips.

Content Editing

Hapus kesalahan, jeda panjang, um/ah yang berlebihan. Silence napas yang terdengar. Rapikan cuts agar natural.

Noise Reduction (Voice Tracks)

Get noise profile dari section noise-only → Apply ke seluruh voice track. Settings: 12 dB, Sensitivity 6.0, Smoothing 3 bands.

EQ

High-pass di 80 Hz (remove rumble). Presence boost 2–4 kHz (clarity). Slight cut 6–8 kHz jika sibilance bermasalah.

Compression

Threshold -20 dB · Ratio 3:1 · Attack 5ms · Release 200ms · Make-up gain ON. Ratakan dynamics voice.

Mixing

Balance levels voice dan music. Fade automation (music turun saat voice berbicara). Transitions antar segmen.

Loudness Normalization

Effect → Loudness Normalization → Target: -16 LUFS untuk podcast streaming (Spotify, Apple Podcasts).

Final Limiter

Limit to -1.0 dB sebagai safety. Pastikan tidak ada clipping yang lolos.

Export

WAV 48kHz/24bit (master archive) + MP3 192kbps (delivery). Isi metadata: Title, Artist, Album Art.

Loudness Standards:
Podcast/Streaming: -16 LUFS · YouTube: -14 LUFS · Broadcast TV: -23 LUFS
Praktikum 01

Lab 1 — Interface Mastery & Basic Editing

Familiar dengan Audacity interface dan operasi editing dasar. Centang setiap langkah setelah selesai dilakukan di Audacity.

✅ Checklist Praktikum 1

0 / 10 selesai
  • Buka Audacity dan kenali semua area interface (Menu, Transport, Tools, Track Panel, Timeline)
  • Import audio file (recording dari Pertemuan 3 atau sample file)
  • Play audio menggunakan Spacebar, pause, dan stop
  • Buat selection 5 detik dengan click & drag di waveform
  • Extend selection menggunakan Shift+Click
  • Cut (Ctrl+X) bagian yang tidak diinginkan (silence atau mistake)
  • Trim audio menggunakan Ctrl+T (select apa yang ingin DIPERTAHANKAN dulu)
  • Split clip menggunakan Ctrl+I dan pindahkan dengan Time Shift Tool (T)
  • Silence region menggunakan Ctrl+L (bandingkan dengan Delete)
  • Save Project (.aup3) dan Export Audio (WAV)
Praktikum 02

Lab 2 — Noise Reduction & Audio Cleanup

Master noise reduction dan produksi clean audio. Gunakan rekaman dengan background noise (kipas, hum, room tone).

📋 Prosedur Praktikum 2

0 / 8 selesai
  • Analisis: Dengarkan seluruh rekaman. Screenshot Analyze → Plot Spectrum. Catat frekuensi noise yang terlihat.
  • Cari noise-only region: Identifikasi 1–2 detik audio yang HANYA berisi noise tanpa suara.
  • Get Noise Profile: Select noise-only region → Effect → Noise Reduction → Get Noise Profile.
  • Apply Reduction: Ctrl+A → Effect → Noise Reduction → Noise: 12 dB, Sensitivity: 6.0, Smoothing: 3 → OK.
  • Compare: Ctrl+Z untuk undo, Ctrl+Y untuk redo. Dengarkan perbedaan before/after.
  • Cleanup tambahan: Silence napas panjang atau pause (select + Ctrl+L). Fix pop/click: Effect → Repair.
  • High-pass filter: Ctrl+A → Effect → Filter Curve EQ → Turunkan frekuensi 80 Hz ke bawah.
  • Export: Simpan versi cleaned sebagai WAV. Screenshot spectrum before/after untuk laporan (400 kata).
📝 Laporan Lab 2 (400 kata) harus mencakup:
Jenis noise yang teridentifikasi · Settings yang digunakan · Assessment hasil · Perbandingan before/after (dengan screenshot spectrum)

Lab 3 — Effects Processing Chain

Apply professional processing chain untuk broadcast-quality audio. Gunakan audio bersih dari Lab 2.

⛓️ Processing Chain Tracker

0 / 5 steps
1

Normalization 

Effect → Normalize
☑ Remove DC offset
☑ Normalize peak amplitude: -1.0 dB
2

Equalization 

Effect → Filter Curve EQ
• High-pass: 80 Hz dan di bawahnya → DOWN (remove rumble)
• Presence:  2–4 kHz → sedikit UP (clarity)
• De-ess:    6–8 kHz → sedikit DOWN (harsh S)
3

Compression 

Effect → Compressor
• Threshold:  -20 dB
• Ratio:       3:1
• Attack:      5 ms
• Release:   200 ms
• Make-up gain: ☑ ON
4

Limiter 

Effect → Limiter
• Limit to:   -1.0 dB
• Hold:        10 ms
• Make-up gain: ☑ ON
5

Export & Compare 

1. Save project: Ctrl+S
2. Export processed: File → Export → WAV 48kHz/24bit
3. Dengarkan dan bandingkan dengan original
4. Screenshot waveform before/after untuk laporan

Ringkasan Pertemuan 4

🖥️
Interface
Transport controls, Tools toolbar, Timeline, Track Panel, Selection Toolbar, Waveform Display. Keyboard shortcuts untuk efisiensi.
📁
Import & Project
.aup3 untuk save project, Export untuk delivery. Drag & drop atau Ctrl+Shift+I. Mendukung WAV, MP3, FLAC, OGG, M4A.
✂️
Basic Editing
Cut (gap tertutup), Delete (tanpa clipboard), Silence (durasi tetap). Split, trim, arrange clips. Undo infinite!
🎚️
Noise Reduction
Two-step: Get Profile → Apply. 12 dB typical. Hanya efektif untuk constant noise (fan, hum, hiss).
🎛️
EQ
High-pass 80 Hz (remove rumble), presence boost 2–5 kHz, de-ess 6–8 kHz. Filter Curve EQ di Audacity.
📉
Compression
Ratakan dynamic range. Threshold -20 dB, Ratio 3:1 untuk voice. Jangan over-compress (sounds lifeless).
🛑
Limiter
Brick wall ceiling di -1.0 dB. Safety final. Always last step before export.
🎙️
Multi-track & Podcast
Mute/Solo/Gain/Pan per track. Podcast export: MP3 192kbps, Loudness -16 LUFS untuk streaming.
